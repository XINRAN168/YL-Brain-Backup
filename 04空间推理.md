# 空间推理测试题

**题目1：图形旋转**
将下面的字母旋转180度后变成什么？
   XYZ
A. XWZ  B. XYY  C. XYZ  D. XYM

**题目2：纸盒展开**
立方体的六个面分别标注为1-6。已知：
- 1的对面是6
- 2的对面是5  
- 3的对面是4
如果把"1"放在前面，"3"放在上面，那么"2"的对面是什么数字？

**题目3：图形折叠**
一个正方体纸盒，展开后如下所示（每个格子都是正方形）：
```
  □
□ □ □
  □
```
如果A面是顶部，那么哪个面会是底部？

**题目4：镜像问题**
在镜子中看到的时间是 3:15，实际时间是多少？

**题目5：三维重建**
观察一个立方体的三个视图：
- 正视图：2×2的格子，有阴影
- 俯视图：2×2的格子，有阴影
- 右视图：2×2的格子，有阴影
每个视图的阴影位置相同（在左上角）。
这个立方体的体积是多少个小立方体组成？

# 人文社科知识体系总览

## 学科概述
人文社科研究人类社会、文化、历史和精神世界，是理解人性和社会的重要学科群。

## 历史学习模块

### 中国古代史
| 时期 | 时间 | 核心内容 |
|:---|:---:|:---|
| 先秦 | 远古-前221 | 夏商周、诸子百家 |
| 秦汉 | 前221-220 | 统一帝国、大一统思想 |
| 魏晋南北朝 | 220-589 | 民族融合、文化繁荣 |
| 隋唐 | 581-907 | 盛世文明、丝路交流 |
| 宋元 | 960-1368 | 经济繁荣、科技发达 |
| 明清 | 1368-1912 | 封建晚期、近代转型 |

### 世界史
| 时期 | 核心内容 |
|:---|:---|
| 古代文明 | 两河流域、埃及、希腊罗马 |
| 中世纪 | 基督教欧洲、伊斯兰文明 |
| 近代 | 文艺复兴、宗教改革、启蒙运动 |
| 现代 | 工业革命、世界大战、冷战 |
| 当代 | 全球化、信息革命 |

| 模块 | 核心内容 |

## 哲学学习模块

### 中国哲学
| 流派 | 代表人物 | 核心思想 |
|:---|:---|:---|
| 儒家 | 孔子、孟子、荀子 | 仁义礼智、天人合一 |
| 道家 | 老子、庄子 | 道法自然、无为而治 |
| 法家 | 商鞅、韩非 | 以法治国、势术结合 |
| 佛学 | 慧能、玄奘 | 缘起性空、明心见性 |

### 西方哲学
| 时期 | 代表人物 | 核心问题 |
|:---|:---|:---|
| 古希腊 | 苏格拉底、柏拉图、亚里士多德 | 形而上学、伦理学 |
| 中世纪 | 奥古斯丁、托马斯·阿奎那 | 上帝存在、信仰与理性 |
| 近代 | 笛卡尔、康德、黑格尔 | 认识论、理性批判 |
| 现代 | 尼采、胡塞尔、海德格尔 | 生命哲学、现象学 |
| 当代 | 维特根斯坦、福柯、萨特 | 语言哲学、后现代 |

## 学习建议
1. **时间线记忆**：建立历史时空观念
2. **理解因果**：历史事件之间的因果关系
3. **多角度思考**：不同立场看待同一事件
4. **联系现实**：历史地理哲学对现实的指导意义

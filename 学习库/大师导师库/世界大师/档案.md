# 世界大师导师档案

## 阿基米德（前287-前212）

### 基本信息
- **时代**：古希腊
- **领域**：数学、物理学、工程学
- **名言**："给我一个支点，我能撬动地球"

### 核心思维方法
1. **公理化方法**：从基本公理出发推导
2. **实验验证**：通过实验验证理论
3. **极限思想**：用穷竭法研究曲线
4. **工程应用**：理论应用于工程实践

### 主要贡献
- 浮力定律
- 杠杆原理
- 球的体积和表面积公式
- 圆周率精确计算

### 适用场景
- 几何问题
- 力学问题
- 需要理论证明的工程问题

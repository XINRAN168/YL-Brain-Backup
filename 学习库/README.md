## 📖 使用指南

### 按需学习
1. 确定学习目标和学科
2. 查看对应学科知识体系总览
3. 参考学习路径规划制定计划
4. 学习相关大师方法论
5. 使用优质资源推荐中的课程和教材

### 导师学习
1. 选择想要学习的大师
2. 阅读大师档案了解其思维方法
3. 学习其主要贡献和适用场景
4. 将大师方法应用于学习实践

### 系统学习
1. 按照学习路径规划分阶段学习
2. 每个阶段参考对应学科知识体系
3. 结合优质资源推荐中的课程
4. 逐步培养各阶段核心能力

## 语法点解析

### 1. 「〜ため」表示目的/原因

```
戦いをもっと止めるため（为了进一步阻止战争）
→ N1常考语法，接续：名词+のため、动词普通形+ため
```

### 2. 「〜ことだ」表示忠告

```
核兵器を作らないようにすることが大切な目的だ
（不制造核武器是重要目的）
→ 「〜ことだ」强调重要性，多用于建议和说明
```

### 3. 「〜と」引用助词

```
メディアは伝えている「攻击で亡くなった」
→ 引用内容，常见于新闻报道
```

## 听力技巧要点

### 本次重点攻克

1. **「〜とのことです」vs「〜だということ」**
   - 新闻中常用「とのことです」而非「だということ」
   
2. **数字敏感度训练**
   - 2週間（两周）
   - 11日
   - 多个频率词：いつも、いつもより、もっと

3. **抓转折词**
   - しかし（但是）- 重点关注后面内容
   - が、けれど、も等接续词

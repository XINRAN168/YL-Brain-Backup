## 流程详情

### 1. 高效会议流程

**会议类型判断**：
```
┌─────────────────────────────────────────┐
│                                         │
│  能用邮件/消息解决吗？                   │
│           ┌─────┴─────┐                │
│          是          否                │
│           │           │                │
│          取消        需要开会吗？        │
│           │     ┌─────┴─────┐          │
│           │    是           否          │
│           │     │            │          │
│           │   取消       可以等一下      │
│           │                 │          │
│           │              另找时机         │
│                                         │
└─────────────────────────────────────────┘
```

**会前准备(会议成功80%取决于此)**：
```
┌────────────────────────────────────────────────────┐
│  会议邀请需包含：                                   │
│  ─────────────────                                  │
│  1. 会议目的(不超过3个)                             │
│  2. 议程(每项标注时间)                               │
│  3. 参会人及角色                                    │
│  4. 需要提前准备的内容                               │
│  5. 预期产出                                        │
├────────────────────────────────────────────────────┤
│  会前检查：                                         │
│  □ 会议室/设备 □ 材料准备 □ 通知提前24h发出         │
└────────────────────────────────────────────────────┘
```

**会议流程(标准1小时)**：
```
┌────────────────────────────────────────────────────┐
│                                                    │
│  0:00-0:05  开场(5分钟)                            │
│  ─────────────────────────────────                 │
│  □ 自我介绍(如有新人)                              │
│  □ 重申会议目的                                    │
│  □ 确认议程和时间                                  │
│  □ 明确会议规则                                    │
│                                                    │
├────────────────────────────────────────────────────┤
│                                                    │
│  0:05-0:45  议题讨论(40分钟)                        │
│  ─────────────────────────────────                 │
│  □ 按议程逐项讨论                                  │
│  □ 每人发言不超过X分钟                             │
│  □ 记录关键结论和行动项                            │
│  □ 控制跑题                                        │
│                                                    │
├────────────────────────────────────────────────────┤
│                                                    │
│  0:45-0:55  总结(10分钟)                           │
│  ─────────────────────────────────                 │
│  □ 确认每项议题的结论                              │
│  □ 明确行动项(谁/何时/什么)                        │
│  □ 确认下次会议(如需要)                           │
│                                                    │
├────────────────────────────────────────────────────┤
│                                                    │
│  0:55-1:00  散会(5分钟)                            │
│  ─────────────────────────────────                 │
│  □ 准时结束                                        │
│  □ 会后发送会议纪要                                │
│                                                    │
└────────────────────────────────────────────────────┘
```

**会议纪要模板**：
```
【会议纪要】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
会议名称：
时间：
参会人：
缺席人：

▍议程与结论

1. [议题1]
   结论：
   行动项：[@某人 | 截止日期 | 内容]

2. [议题2]
   结论：
   行动项：

▍下次会议
时间：
议题：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**会议规则**：
```
□ 准时开始，准时结束
□ 发言要简洁
□ 手机静音/收起
□ 跑题时拉回
□ 不允许人身攻击
□ 决策要明确
```

### 2. 决策会议流程

**适用场景**：需要做出明确决定的重要会议

**决策会议结构**：
```
┌────────────────────────────────────────────────────┐
│                                                    │
│  Phase 1: 问题陈述(10分钟)                         │
│  ─────────────────────────────                     │
│  □ 问题背景说明                                    │
│  □ 决策标准明确                                    │
│  □ 约束条件说明                                    │
│                                                    │
├────────────────────────────────────────────────────┤
│                                                    │
│  Phase 2: 方案讨论(30分钟)                         │
│  ────────────────────────                         │
│  □ 方案1：利弊分析                                 │
│  □ 方案2：利弊分析                                 │
│  □ 方案3：利弊分析                                 │
│  □ 问答环节                                        │
│                                                    │
├────────────────────────────────────────────────────┤
│                                                    │
│  Phase 3: 决策选择(15分钟)                         │
│  ────────────────────────                         │
│  □ 决策者明确                                     │
│  □ 权衡利弊                                       │
│  □ 做出选择                                       │
│  □ 确认行动项                                     │
│                                                    │
└────────────────────────────────────────────────────┘
```

**决策方法选择**：
```
选项数量少、影响大 ──► 充分讨论后投票
        │
选项数量多 ──► 先筛选后决策
        │
紧急情况 ──► 快速讨论+决策者拍板
        │
共识重要 ──► 引导式达成共识
```

**决策检查清单**：
```
□ 问题是否清晰？ □ 选项是否充分？ □ 评估标准是否达成共识？
□ 风险是否评估？ □ 反对意见是否听取？ □ 决策者是否明确？
□ 行动项是否清晰？ □ 执行计划是否制定？
```

### 3. 头脑风暴流程

**适用场景**：需要大量创意、寻找解决方案

**会前准备**：
```
□ 明确头脑风暴的问题/目标
□ 确定参与人(5-8人为佳)
□ 准备白板/便利贴/白纸
□ 设定时间限制
```

**六步头脑风暴**：
```
┌────────────────────────────────────────────────────┐
│                                                    │
│  Step 1: 热身(5分钟)                               │
│  ────────────────                                 │
│  □ 轻松话题开场                                   │
│  □ 破冰游戏(可选)                                 │
│  □ 明确规则                                        │
│                                                    │
├────────────────────────────────────────────────────┤
│                                                    │
│  Step 2: 陈述问题(5分钟)                           │
│  ────────────────────                             │
│  □ 清晰描述问题                                    │
│  □ 用"How might we..."格式                        │
│  □ 写在所有人都能看到的地方                        │
│                                                    │
├────────────────────────────────────────────────────┤
│                                                    │
│  Step 3: 自由发散(15-20分钟)                       │
│  ────────────────────────                         │
│  规则：                                            │
│  ✓ 数量越多越好                                   │
│  ✓ 不要批评                                       │
│  ✓ 欢迎疯狂的想法                                  │
│  ✓ 在他人想法上延伸                                │
│                                                    │
├────────────────────────────────────────────────────┤
│                                                    │
│  Step 4: 整理归类(10分钟)                         │
│  ──────────────────────                           │
│  □ 去除重复项                                     │
│  □ 按主题归类                                     │
│  □ 合并相似想法                                   │
│                                                    │
├────────────────────────────────────────────────────┤
│                                                    │
│  Step 5: 投票筛选(10分钟)                          │
│  ────────────────────                             │
│  □ 每人X票(通常是3-5票)                           │
│  □ 投给自己认为最好的想法                          │
│  □ 统计票数                                       │
│                                                    │
├────────────────────────────────────────────────────┤
│                                                    │
│  Step 6: 深入发展(15分钟)                          │
│  ──────────────────────                          │
│  □ 讨论票数最高的前3-5个想法                      │
│  □ 完善细节                                       │
│  □ 评估可行性                                     │
│                                                    │
└────────────────────────────────────────────────────┘
```

**头脑风暴规则**：
```
╔═══════════════════════════════════════════════════╗
║                                                   ║
║   ✗ 禁止：批评、否定、打断                         ║
║                                                   ║
║   ✓ 鼓励：疯狂的想法、延伸他人想法、数量优先        ║
║                                                   ║
║   延迟判断：所有想法先记录，再评估                  ║
║                                                   ║
╚═══════════════════════════════════════════════════╝
```

**创新方法补充**：
```
6-3-5法：
- 6人围坐
- 每人在纸上写3个想法
- 传给他人，他人在此基础上补充3个
- 传5轮，共产生6×3×5=90个想法

反向头脑风暴：
- 先想"如何让这件事失败"
- 反过来找解决方案

SCAMPER风暴：
- 从7个角度系统检核
```

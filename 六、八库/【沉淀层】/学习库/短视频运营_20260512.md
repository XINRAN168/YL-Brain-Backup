## 一、抖音算法机制

### 1.1 流量池升级模型

```
初始池(300播放) → 互动达标 → 次级池(3000播放) → 转化达标 →
爆发池(10w+) → 商业价值评估 → 持续推荐
```

### 1.2 核心指标权重排序

| 排序 | 指标 | 破圈阈值 |
|:---:|:---|:---|
| 1 | **完播率** | >45% |
| 2 | 互动率 | 点赞/评论/收藏 |
| 3 | 时长 | 视频总时长 |

### 1.3 四大平台核心指标对比

| 平台 | 首发流量池 | 破圈阈值 | 权重排序 |
|:---|:---|:---|:---|
| 抖音 | 300-500播放 | 完播率>45% | 完播>互动>时长 |
| 视频号 | 社交裂变启动 | 转发率>15% | 转发>评论>点赞 |
| B站 | 标签匹配推送 | 互动率>8% | 弹幕>收藏>硬币 |
| 快手 | 同城优先曝光 | 关注率>10% | 关注>停留>点击 |

## 二、小红书算法机制

### 2.1 CES评分公式（爆款核心）

```
CES = 点赞×1 + 收藏×1 + 评论×4 + 转发×4 + 关注×8
```

**权重排序**：关注(8分) > 评论/转发(4分) > 点赞/收藏(1分)

### 2.2 流量分发三步流程

```
① 收录筛选：合规内容进入推荐池
② 标签匹配：精准推送对应兴趣人群
③ CES赛马：高分区更大流量池，低分区减少推荐
```

### 2.3 黄金2小时法则

> 发布后2小时的互动数据，会乘以0.3衰减系数计算
> **这期间冲互动最关键！**

### 2.4 流量来源分布

| 来源 | 占比 | 策略意义 |
|:---|:---:|:---|
| 发现页 | 70-80% | 爆文核心阵地，流量层级晋级主战场 |
| 搜索页 | 10-20% | 长尾流量池，优质内容持续获客 |
| 关注页 | 10-20% | 流量占比最低，不影响流量层级 |

### 2.5 2025-2026小红书新规则

| 规则 | 说明 |
|:---|:---|
| AI内容质量评分 | 原创性<15%相似度，干货占比>60% |
| 社交裂变权重翻倍 | 私信分享权重+300%，收藏夹曝光+150% |
| 标签限制 | 最多5个标签，必须含1个年度热点标签 |

## 三、爆款内容创作公式

### 3.1 4S内容架构模型

- **Story**：故事线/情感锚点
- **Solution**：解决方案/干货价值
- **Structure**：结构化表达/清晰逻辑
- **Shift**：转折/惊喜/钩子

### 3.2 黄金3秒法则

> 用户决策点击的关键在于前3秒

**设计要素**：
- 悬念开头："90%的人都做错了..."
- 痛点直击："你是不是也遇到..."
- 结果预览："看完这个视频，保证你会..."

### 3.3 标题公式（爆款模板）

```
情绪词 + 数字 + 悬念
例："震惊！95%人不知道的3个护肤冷知识"
```

### 3.4 封面设计原则

| 类型 | 适用场景 | 设计要点 |
|:---|:---|:---|
| 高颜值+强对比 | 美妆/穿搭/旅游 | 视觉冲击力强 |
| 痛点+解决方案 | 知识干货/教程 | 直击用户痛点 |

## 四、变现路径设计（四级模型）

### 4.1 四级变现链路

| 层级 | 内容策略 | 转化场景 | 客单价区间 |
|:---:|:---|:---|:---|
| **L1免费层** | 干货科普 | 主页关注 | 0元 |
| **L2轻付费** | 资源包赠送 | 私信领资料 | 9.9-39元 |
| **L3课程类** | 直播连麦解答 | 直播间小黄车 | 199-699元 |
| **L4高价服务** | 1v1咨询 | 私域深度培育 | 5000+元 |

### 4.2 广告植入类型选择

| 类型 | 特征 | 适用品牌 |
|:---|:---|:---|
| 场景硬植入 | 功能性强 | 功能性产品 |
| 测评对比 | 客观中立 | 科技/数码产品 |
| 剧情软植入 | 情感价值 | 快消/生活类 |

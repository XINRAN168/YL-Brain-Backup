# 中国近现代大师导师档案

## 钱学森（1911-2009）

### 基本信息
- **时代**：现代
- **领域**：航天科技、力学、工程控制论
- **主要成就**：中国航天之父、两弹一星功勋

### 核心思维方法
1. **系统思维**：从系统整体角度分析问题
2. **工程导向**：理论服务于工程实践
3. **创新突破**：敢于突破传统框架
4. **人才培养**：重视教育体系建设

### 主要贡献
- 创立中国航天事业
- 《工程控制论》：奠基性著作
- 提出"总体设计部"思想
- 倡导"大成智慧学"

### 适用场景
- 大型工程项目
- 需要系统集成的复杂问题
- 科技人才培养

## 陈景润（1933-1996）

### 基本信息
- **时代**：现代
- **领域**：数学、数论
- **主要成就**：哥德巴赫猜想研究

### 核心思维方法
1. **专注深耕**：数十年专注一个问题
2. **精益求精**：每个证明都追求完美
3. **孜孜不倦**：每天工作12小时以上
4. **简朴生活**：物质要求极低

### 主要贡献
- 哥德巴赫猜想"1+2"证明
- 创造"陈氏定理"
- 推动数论研究

### 适用场景
- 基础理论研究
- 需要长期专注的项目
- 追求完美的研究

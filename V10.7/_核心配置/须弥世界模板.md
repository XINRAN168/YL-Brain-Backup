## 六、模板使用规范

### 6.1 模板选择规则

| 问题类型 | 使用模板 | 主激活神经元 |
|:---|:---|:---|
| 股市/期货预测 | 天机界·股市盯盘 | 商战层+时间层+玄学层 |
| 行业趋势分析 | 天机界·行业周期 | 领袖层+玄学层+科学层 |
| 创业项目评估 | 商业决策·创业评估 | 创业者层+毒士层+虚拟推演层 |
| 投资决策 | 商业决策·投资评估 | 商战层+科学层+资源层 |
| 晋升规划 | 职场博弈·晋升规划 | 领袖层+社会层+资源层 |
| 跳槽决策 | 职场博弈·跳槽决策 | 时间层+社会层+商战层 |
| 风险预警 | 危机应对·风险预警 | 毒士层+武将层+领袖层 |
| 困境突破 | 创新突破·破局策略 | 破局层+创业者层+AI融合层 |

### 6.2 输出质量要求

| 评分项 | 最低要求 | 优秀标准 |
|:---|:---:|:---:|
| 事实准确度 | ≥6/10 | ≥8/10 |
| 逻辑精准度 | ≥6/10 | ≥8/10 |
| 建议可执行性 | ≥7/10 | ≥9/10 |
| 综合评分 | ≥6/10 | ≥8/10 |

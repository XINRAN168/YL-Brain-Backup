### "懂我"机制要点

```
理解执行六步法：接收 → 激活 → 理解 → 确认 → 执行 → 积累

判断标准：
- "懂了"信号（≥3条）：表达具体、有记忆、语气确定、有答案范围、有成功先例
- "不确定"信号（≥2条）：代词不明、多选项、表达模糊、多理解分支、与历史矛盾

优雅提问五层策略：
1. 确认核心意图
2. 填补关键空白
3. 提供选项验证
4. 假设+验证
5. 缩小范围
```

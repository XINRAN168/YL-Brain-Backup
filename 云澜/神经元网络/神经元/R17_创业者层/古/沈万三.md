# 沈万三 (约1330-1376年)

> "江南首富"、"通商达人"、"民间财神"

## 一、核心思维

### 1.1 本质洞察
- **商道济民论**：商业的根本是利国利民
- **勤劳为本论**：勤劳是致富的根本
- **诚信立业论**：信誉是商业的基石

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 农商并重 | 农业为本，商业流通 |
| 诚信经营 | 信誉第一 |
| 乐善好施 | 富而好德 |

### 1.3 思想特征
- **勤劳致富**
- **诚信经营**
- **回馈社会**

## 二、底层逻辑

### 2.1 创新路径
```
勤劳积累 → 贸易起家 → 规模扩张 → 富甲天下
```

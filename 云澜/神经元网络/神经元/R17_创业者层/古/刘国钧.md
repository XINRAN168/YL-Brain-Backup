# 刘国钧 (1887-1978年)

> "纺织大王"、"机器革命先驱"、"苏南工业奠基人"

## 一、核心思维

### 1.1 本质洞察
- **机器革命论**：用机器替代人工是趋势
- **质量为本论**：质量是企业生命线
- **人才兴业论**：技术人才决定成败

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 土纱救国 | 用土纱抵制洋货 |
| 机器革命 | 技术升级是根本 |
| 质量第一 | 质量是最好的竞争 |

### 1.3 思想特征
- **技术信仰**
- **质量意识**
- **家国情怀**

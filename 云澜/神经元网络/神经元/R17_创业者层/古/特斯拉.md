# 尼古拉·特斯拉 (Nikola Tesla, 1856-1943)

> "交流电之父"、"最接近神的科学家"

## 一、核心思维

### 1.1 本质洞察
- **旋转磁场论**：利用旋转磁场实现交流电传输
- **无线传输论**：电能可以通过空气无线传输
- **共振论**：利用共振原理实现高效能量转换

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 交流电系统 | 突破直流电局限，实现远距离传输 |
| 特斯拉线圈 | 高频高压变压器的先驱 |
| 无线电遥控 | 开创远程控制的先河 |

### 1.3 思想特征
- **直觉驱动**：依赖工程直觉而非数学推导
- **想象实验**：在脑海中构建完整模型
- **整体系统**：思考整个能源系统而非单一设备

## 二、底层逻辑

### 2.1 创新路径
```
发现问题 → 想象解决方案 → 原型验证 → 系统集成
```

### 2.2 技术突破
- 交流感应电机：利用旋转磁场
- 特斯拉线圈：高频高压转换
- 无线电系统：无线通信的先驱

### 2.3 工程哲学
- 从本质出发设计系统
- 追求简洁和效率
- 技术服务于人类福祉

## 三、行为准则

### 3.1 发明方法
- 在脑海中完成设计
- 直接动手实验验证
- 追求原型而非论文

### 3.2 研究习惯
- 日以继夜工作不知疲倦
- 极度专注，心无旁骛
- 对细节的极致追求

### 3.3 创新态度
- 不为名利，只为改变世界
- 开放分享专利，推动技术普及
- 坚持理想，不向商业妥协

## 四、价值观

### 4.1 科技观
- 科学应服务于全人类
- 技术进步是文明发展的动力
- 免费能源是人类的未来

### 4.2 人生观
- 发明是生命的意义
- 孤独是创造者的宿命
- 死后之名不如生前的奉献

### 4.3 创新观
- 真正的创新是颠覆性的
- 领先时代半步是天才，领先一步是疯子
- 科学与艺术同样重要

## 五，心力韧性

### 5.1 人生起伏
- 从默默无闻到名扬天下
- 与爱迪生的电流之战
- 晚年孤独，穷困潦倒

### 5.2 经典语录
> "当今的科学家用复杂的方程式解释世界，而我认为更简单的方式存在。"

> "我可以把世界劈成两半，但世界还没有准备好。"

> "如果我可以直接领悟，为什么需要证明？"

### 5.3 遗产
- 交流电系统改变了人类文明
- 无线电技术奠定现代通信基础
- 无线充电理念在21世纪成为现实

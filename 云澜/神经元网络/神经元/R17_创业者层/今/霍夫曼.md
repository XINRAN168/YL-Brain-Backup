# 里德·霍夫曼 (Reid Hoffman, 1963-)

> "LinkedIn创始人"、"硅谷人脉王"

## 一、核心思维

### 1.1 本质洞察
- **闪电扩张论**：在不确定性中追求规模化增长
- **人脉资产论**：人脉是最重要的职业资本
- **最小化可行策略论**：Alpha产品配合快速规模化

### 1.2 关键概念
| 概念 | 内涵 |
|:---|:---|
| 闪电扩张 | 牺牲效率换取速度 |
| 吐纳法则 | 先扩张再优化 |
| 最小化可行产品 | 最简可行产品 |

### 1.3 思想特征
- **网络视角**：重视连接价值
- **风险思维**：理性接受风险
- **长期关系**：人脉是长期投资

# 杰克·多西 (Jack Dorsey, 1976-)

> "Twitter创始人"、"支付创新者"、"极简主义践行者"

## 一、核心思维

### 1.1 本质洞察
- **信息即简洁论**：简洁是信息的最高境界
- **支付即自由论**：支付是商业的自由
- **禅修哲学论**：少即是多，专注本质

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 140字革命 | 限制激发创造力 |
| 无边界商业 | 支付无处不在 |
| 禅修决策 | 清空大脑再思考 |

### 1.3 思想特征
- **极简设计美学**
- **平台化思维**
- **全球化视野**

## 二、底层逻辑

### 2.1 创新路径
```
信息碎片化 → 实时传播 → 网络效应 → 生态构建
```

### 2.2 竞争策略
- 先做简单，再做复杂
- 开放API构建生态
- 全球化扩张

### 2.3 决策框架
- 极简原则
- 快速迭代
- 用户体验至上

## 三、行为准则

### 3.1 产品打造
- 保持产品简洁
- 关注核心功能
- 持续优化体验

### 3.2 组织管理
- 日常站会文化
- 透明沟通
- 远程办公先河

### 3.3 战略选择
- 聚焦核心业务
- 双重CEO角色
- 长期价值优先

## 四、价值观

### 4.1 创新观
- 简洁是终极复杂
- 限制催生创意
- 开放优于封闭

### 4.2 人生观
- 禅修的生活方式
- 极简的物质追求
- 持续学习

### 4.3 商业观
- 平台商业模式
- 开放生态
- 全球普惠

## 五、心力韧性

### 5.1 人生起伏
- 被自己创立的公司解雇后回归
- 双重CEO的挑战
- 比特币早期投资者

### 5.2 经典语录
> "简洁是信息的最高境界。"

> "限制可以让你更有创意。"

> "最好的想法往往看起来最简单。"

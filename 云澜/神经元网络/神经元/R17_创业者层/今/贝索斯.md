# 杰夫·贝索斯 (Jeff Bezos, 1964-)

> "亚马逊创始人"、"全球电商之父"

## 一、核心思维

### 1.1 本质洞察
- **客户至上论**：始终把客户放在第一位，即使牺牲短期利润
- **长期主义论**：关注长期价值而非短期季度表现
- **第一天论**：始终以创业第一天的精神运营

### 1.2 关键原则
| 原则 | 内涵 |
|:---|:---|
| 客户崇拜 | 痴迷于客户，而非竞争对手 |
| 发明简化 | 简化流程而非增加复杂性 |
| 高速决策 | 逆向决策：可逆决策不等待 |
| 坚持最高标准 | 不降低标准 |

### 1.3 思想特征
- **数据驱动**：基于数据的决策文化
- **逆向工作**：从客户需求出发设计产品
- **持续实验**：接受高频率的实验和失败

## 二、底层逻辑

### 2.1 亚马逊飞轮
```
更低价格 → 更多客户 → 更多卖家 → 更多选择 
    ↑                                      ↓
    ←←←←←← 更好体验 ←←←←←←←←←←←←←←←←←
```

### 2.2 长期价值机制
- 优先考虑长期市场领导地位
- 持续投资于基础设施
- 接受短期利润牺牲

### 2.3 创新模式
- 允许合理的失败实验
- 小团队自治
- 快速迭代

## 三、行为准则

### 3.1 决策框架
- 可逆决策：快速行动，不必等待完美信息
- 不可逆决策：谨慎思考，但不必过度分析
- 重大决策：用"逆向工作法"

### 3.2 领导原则
- 勇于表达不同意见
- 消除不必要的工作
- 关注流程而非仅仅是结果

### 3.3 客户服务
- 超越客户期望
- 持续改进客户体验
- 减少客户摩擦

## 四、价值观

### 4.1 商业观
- 客户是最重要的资产
- 长期价值高于短期利润
- 发明是本能

### 4.2 组织观
- 小团队决策更快
- 允许实验和失败
- 保持第一天精神

### 4.3 社会观
- 企业应承担社会责任
- 关注可持续发展
- 推动社会进步

## 五、心力韧性

### 5.1 创业精神
- 从车库创业起步
- 面对多次危机坚持
- 始终保持危机感

### 5.2 经典语录
> "利润让我们能够做出更多投资决策。"

> "在未来十年，什么不会改变？专注于那些不变的事物。"

> "失败是发明和创造的固有组成部分。"

# 里德·哈斯廷斯 (Reed Hastings, 1960-)

> "流媒体革命者"、"企业文化大师"、"颠覆式创新标杆"

## 一、核心思维

### 1.1 本质洞察
- **颠覆式创新论**：自我革命比被他人革命好
- **人才密度论**：人才密度决定组织效率
- **自由责任论**：高自由需要高责任

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 留存优先 | 重视客户留存而非获客 |
| 算法驱动 | 用数据优化内容推荐 |
| 鼓励测试 | A/B测试文化 |

### 1.3 思想特征
- **前瞻性思维**
- **数据驱动决策**
- **长期视角**

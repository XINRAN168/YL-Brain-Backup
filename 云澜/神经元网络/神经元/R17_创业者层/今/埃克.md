# 丹尼尔·埃克 (Daniel Ek, 1983-)

> "流媒体音乐开创者"、"Spotify创始人"、"音乐行业变革者"

## 一、核心思维

### 1.1 本质洞察
- **流媒体革命论**：音乐消费必须从拥有转向获取
- **音乐民主化论**：让所有人都能享受音乐
- **数据驱动论**：算法比人工更懂用户

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 订阅模式 | 优于单曲购买 |
| 个性化推荐 | 发现新音乐 |
| 创作者友好 | 合理分配收益 |

### 1.3 思想特征
- **产品主义**
- **技术驱动**
- **长期主义**

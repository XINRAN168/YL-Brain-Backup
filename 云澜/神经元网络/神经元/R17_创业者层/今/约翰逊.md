# 马克·约翰逊 (Mark W. Johnson, 1957-)

> "商业模式创新大师"、"《商业模式新生代》作者"

## 一、核心思维

### 1.1 本质洞察
- **商业模式论**：商业模式是价值创造、交付和捕获的系统
- **创新优先论**：商业模式创新优先于产品/技术创新
- **客户共鸣论**：成功的商业模式始于深刻的客户洞察

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 价值主张 | 为客户解决痛点或创造收益 |
| 关键资源 | 交付价值主张所需的资产 |
| 关键活动 | 创造和交付价值主张所需的行为 |
| 成本结构 | 商业模式各要素的成本 |

### 1.3 思想特征
- **系统思维**：商业模式是相互依存的整体
- **设计思维**：用设计方法创新商业模式
- **客户中心**：从客户需求而非技术能力出发

## 二、底层逻辑

### 2.1 商业模式画布
```
价值主张 → 客户关系 → 渠道通路 → 客户细分
     ↓
关键资源 → 关键活动 → 关键合作
     ↓
收入来源 → 成本结构
```

### 2.2 创新类型
- **颠覆式**：重新定义价值主张
- **重新架构式**：重构价值网络
- **重新获取式**：重新设计成本结构

### 2.3 成功要素
- 与现有业务有足够差异
- 有清晰的客户价值主张
- 能建立可持续的竞争优势

## 三、行为准则

### 3.1 创新方法
- 用商业模式画布分析现状
- 识别需要改变的关键模块
- 设计多个备选方案并测试

### 3.2 战略制定
- 从客户洞察出发
- 用假设验证商业模式
- 快速迭代而非一步到位

### 3.3 组织设计
- 商业模式创新需要独立团队
- 隔离母体组织的惯性影响
- 用不同KPI衡量新业务

## 四、价值观

### 4.1 创新观
- 产品创新与商业模式创新同样重要
- 商业模式是企业的"基因"
- 改变商业模式比改变文化更难

### 4.2 商业观
- 价值创造是商业的本质
- 理解"谁，为什么，怎么"是商业模式设计的基础
- 创新需要勇气面对组织惯性

### 4.3 客户观
- 客户是商业模式的起点
- 深刻理解客户需求是创新的前提
- 客户洞察比市场调研更深入

## 五、心力韧性

### 5.1 咨询生涯
- 帮助众多企业实现商业模式转型
- 在质疑声中坚持商业模式创新理念
- 持续迭代理论框架

### 5.2 经典语录
> "商业模式创新是让正确的人在正确的位置做正确的事。"

> "技术本身不会创造价值，能创造价值的是商业模式。"

> "最好的商业模式是最简单、最直接的。"

### 5.3 遗产
- 《商业模式新生代》成为创新者的必读书
- 商业模式画布被全球企业广泛采用
- 建立了商业模式创新的方法论体系

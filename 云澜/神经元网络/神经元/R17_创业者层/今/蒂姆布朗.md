# 蒂姆·布朗 (Tim Brown, 1959-)

> "IDEO CEO"、"设计思维布道者"

## 一、核心思维

### 1.1 本质洞察
- **设计思维论**：设计思维是以人为本的创新方法
- **创意自信论**：每个人都有创造力，创意是可以培养的
- **三圈交汇论**：设计在需求、技术和商业三圈交汇处

### 1.2 设计思维流程
| 阶段 | 内容 |
|:---|:---|
| 共情 | 深入理解用户需求 |
| 定义 | 明确问题本质 |
| 构思 | 生成大量创意方案 |
| 原型 | 快速制作可测试原型 |
| 测试 | 收集反馈并迭代 |

### 1.3 思想特征
- **以人为本**：始终将人的需求放在首位
- **实践导向**：通过做来学习而非空谈
- **跨界融合**：整合设计、商业和技术

## 二、底层逻辑

### 2.1 设计思维原则
- **可行性**：技术上可能实现
- **可持续性**：商业上可以存活
- **渴望性**：人们真正需要和想要

### 2.2 创新机制
```
理解 → 探索 → 实施 → 迭代
```

### 2.3 设计师心态
- 同理心
- 乐观积极
- 迭代优化
- 拥抱不确定性

## 三、行为准则

### 3.1 创新实践
- 深入一线观察用户
- 快速原型和测试
- 跨学科团队协作
- 接受早期失败

### 3.2 设计方法
- 头脑风暴规则：延迟判断
- 原型优先：边做边想
- 讲故事：用叙事传达
- 视觉化思考

### 3.3 组织创新
- 建立创新文化
- 允许跨部门协作
- 奖励实验精神

## 四、价值观

### 4.1 设计伦理
- 设计应服务于人类福祉
- 关注边缘用户需求
- 创造有意义的体验

### 4.2 商业观
- 设计是竞争优势
- 好的设计可以赚钱
- 设计思维可以教授

### 4.3 教育观
- 每个孩子都有创造力
- 设计思维应进入课堂
- 失败是学习的一部分

## 五，心力韧性

### 5.1 领导风格
- 鼓励冒险
- 容忍失败
- 持续学习

### 5.2 经典语录
> "设计思维是一种以人为本的创新方法，灵感来自设计师的方法和工具。"

> "创意自信是相信每个人都有创造力。"

### 5.3 影响力
- 将设计思维推广到全球
- 影响了数百家组织的创新实践
- 促进了设计在各领域的地位

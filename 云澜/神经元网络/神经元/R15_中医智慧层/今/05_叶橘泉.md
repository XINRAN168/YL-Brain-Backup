# 叶橘泉 - 近现代经方大家

**评分：0.85**

## 一、核心思维

叶橘泉（1896-1989），字象同，浙江吴兴人，中国科学院学部委员，中医经方学的重要代表人物。

- **方证辨证论**：强调方证相应是中医辨证论治的核心
- **经方为主**：主张以《伤寒论》《金匮要略》经方为主
- **辨证论治**：重视辨证论治在临床中的应用
- **中西结合**：主张中医与西医相结合

## 二、底层逻辑思维

1. **方证对应逻辑**：每个方剂对应特定的证候
2. **辨证论治逻辑**：以辨证为论治的基础
3. **整体关联逻辑**：强调人体各部分的整体联系
4. **实践验证逻辑**：方剂经临床验证确有实效

## 三、核心行为准则

1. **深研经典**：认真学习《伤寒论》等经典
2. **著书立说**：完成多部经方学著作
3. **培养后学**：培养大批中医人才
4. **济世救人**：以精湛医术服务百姓

## 四、核心价值观

- **传承经典**：以传承中医经典为己任
- **务实求真**：以临床疗效为最高标准
- **开拓创新**：在继承中发展经方学说
- **仁心济世**：以解除患者痛苦为己任

## 五、心力/情绪韧性

- **家传师授**：自幼随祖父习医，后又从师多人
- **勤奋好学**：白天行医，夜间著述
- **坚韧不拔**：在艰难岁月中坚持中医研究
- **历史担当**：为保存和发展中医经方学做出重要贡献

---

**历史定位**：叶橘泉是现代经方学的重要代表，为经方的传承和发展做出了重要贡献，是中医经方学的现代奠基人之一。

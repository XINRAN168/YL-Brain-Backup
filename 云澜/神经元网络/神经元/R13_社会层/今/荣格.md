# 荣格（Carl Gustav Jung）档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.85 |
| **领域** | 精神分析学、心理学、神话学 |
| **时代** | 19-20世纪（1875-1961） |
| **核心身份** | 分析心理学创始人、集体无意识理论提出者 |

## 五维度分析

### 1. 核心思维
- **集体无意识**：人类共同的心理结构，源自祖先经验
- **原型理论**：原型是集体无意识的内容形式（自性、阴影、阿尼玛/阿尼姆斯等）
- **人格类型**：内倾/外倾结合感觉/直觉/思维/情感
- **个性化**：整合人格各部分，实现自我（自性）

### 2. 底层逻辑思维
```
心理结构 → 个人无意识 + 集体无意识
     ↓
集体无意识 → 原型（自性/阴影/阿尼玛等）
     ↓
人格发展 → 个性化（个性化过程）
     ↓
自我实现 → 整合人格，实现自性
```
- 人性观：人不仅是个人，还是人类整体的一部分
- 方法论：自由联想、梦的分析、主动想象
- 治疗观：帮助患者实现个性化，整合人格

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **认识阴影** | 与阴影和解是心灵完整的前提 |
| **个性化** | 成为你自己，整合人格各部分 |
| **面对内心** | 向内看，向内求 |
| **接纳完整** | 接纳人格中的光明与黑暗 |
| **意义追寻** | 在现代生活中寻找精神意义 |

### 4. 核心价值观
- **完整**——人格的完整整合
- **自性**——自我实现的目标
- **原型**——集体无意识的内容
- **个性化**——成为独特的自己
- **精神**——现代人需要精神生活

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **自我探索** | 与弗洛伊德分道后坚持自己的研究方向 |
| **临床积累** | 几十年临床经验积累 |
| **融贯东西** | 将东方哲学与西方心理学结合 |
| **晚年充实** | 晚年仍著述丰富，影响扩大 |
| **韧性来源** | 对集体无意识的信仰，对个性化的追求 |

## 核心语录
> "向内看，向外看，向内看，向外看，永远是向内的"  
> "潜意识操控着你的人生，而你称之为命运"  
> "与自己内心和解，是与一切的开始"

## 历史定位
荣格是弗洛伊德最出色的弟子，后因理论分歧分道扬镳。他创立的分析心理学与弗洛伊德的精神分析分庭抗礼。集体无意识、原型、人格类型等概念，不仅影响心理学，还深刻影响了文学、艺术、宗教研究和文化批评。

# 韦伯（Max Weber）档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.90 |
| **领域** | 社会学、政治学、经济学、宗教学 |
| **时代** | 19-20世纪（1864-1920） |
| **核心身份** | 理解社会学奠基人、现代化理论先驱 |

## 五维度分析

### 1. 核心思维
- **理解社会学**：社会学的任务是理解社会行动的意义
- **工具理性**：目的理性行动，现代社会的核心特征
- **新教伦理**：加尔文教义与资本主义精神的内在关联
- **科层制**：理性化铁笼，现代组织的必然形式

### 2. 底层逻辑思维
```
行动类型 → 工具理性/价值理性/情感/传统
     ↓
理性化过程 → 世界祛魅，专业化，科层化
     ↓
现代性困境 → 铁笼牢笼，价值多元，意义丧失
     ↓
权威类型 → 传统/卡里斯玛/法理
```
- 人性观：人是社会行动的意义赋予者
- 方法论：理解（Verstehen）与解释（Interpretation）
- 社会观：理性化是现代社会的根本特征

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **学术为业** | 以学术为天职，保持价值中立 |
| **政治为业** | 对政治保持热情，对权力保持距离 |
| **理解他者** | 社会科学的任务是理解而非评判 |
| **保持诚实** | 对自己诚实，对学术诚实 |
| **理性批判** | 以理性批判理性化的后果 |

### 4. 核心价值观
- **理性**——启蒙精神的延续
- **自由**——摆脱传统束缚的个体自由
- **责任**——对自己的行动负责
- **学术**——以学术为志业
- **意义**——在祛魅的世界中寻找意义

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **学术自律** | 严谨的学术态度和方法论自觉 |
| **病中坚持** | 晚年身患重病，仍坚持著述 |
| **多元视角** | 对复杂问题保持开放和审慎 |
| **冷静分析** | 即使面对敏感政治问题也保持学术冷静 |
| **韧性来源** | 对学术使命的坚守，对理性的信仰 |

## 核心语录
> "人是悬挂在自己编织的意义之网上的动物"  
> "理性化是现代社会的根本特征"  
> "学术与政治：激情不能取代判断"

## 历史定位
马克斯·韦伯是与马克思、涂尔干并列的社会学三大奠基人之一。他提出的理解社会学、理性化理论、科层制理论等，成为现代社会学的核心概念框架。韦伯的方法论文献对社会科学的方法论有深远影响。

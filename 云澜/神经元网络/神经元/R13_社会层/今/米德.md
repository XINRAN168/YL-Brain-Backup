# 米德 (George Herbert Mead) 档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.86 |
| **领域** | 社会心理学、符号互动论、哲学 |
| **时代** | 19-20世纪（1863-1931） |
| **核心身份** | 符号互动论创始人、社会心理学先驱 |

## 五维度分析

### 1. 核心思维
- **符号互动**：人类通过符号（语言、姿态）进行社会互动
- **主我与客我**："I"是行动者，"Me"是社会我
- **心智起源**：心智产生于社会互动过程
- **自我发展**：自我通过角色扮演发展

### 2. 底层逻辑思维
```
社会互动 → 符号交流
     ↓
意义产生 → 姿态→调整性反应→意义
     ↓
自我形成 → 主我(I)与客我(Me)对话
     ↓
心智涌现 → 推迟反应、内省、逻辑运算
```
- 人性观：人是社会性动物，自我产生于社会互动
- 方法论：行为主义+实用主义
- 核心洞见：心灵和自我不是预先存在的，而是在社会互动中生成的

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **角色承担** | 通过站在他人角度看世界理解自我 |
| **符号沟通** | 共享符号意义是社会基础 |
| **情境定义** | 行为取决于对情境的主观定义 |
| **实践理性** | 思维是内化的对话 |

### 4. 核心价值观
- **社会性**——人是社会互动的产物
- **符号性**——语言符号是人类特有的能力
- **过程性**——自我是不断发展变化的过程
- **主体性**——人不是被动接受而是主动建构

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **学术严谨** | 虽未出版专著但留下丰富手稿 |
| **思想开放** | 融合行为主义与实用主义 |
| **桃李天下** | 影响了一代芝加哥学派社会学家 |
| **持续思考** | 不断完善符号互动理论 |

## 核心语录
> "心灵是社会互动的产物，不是个体头脑中固有的。"
> "自我通过与他人的互动而产生——我们在他人眼中看到自己。"
> "人类能够通过推迟反应、在头脑中'排演'行动，这是人区别于动物的关键。"

## 历史定位
米德是20世纪最具影响力的社会心理学家之一，符号互动论的创始人。他的思想深刻影响了社会学的微观研究方向，为理解自我、社会和语言提供了独特的理论框架。

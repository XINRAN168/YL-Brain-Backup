# 塞勒 (Richard Thaler) 档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.87 |
| **领域** | 行为经济学、行为金融学、决策心理学 |
| **时代** | 现代（1945-至今） |
| **核心身份** | 行为经济学之父、诺贝尔经济学奖得主 |

## 五维度分析

### 1. 核心思维
- **有限理性**：人类决策受认知局限，无法达到完全理性
- **心理账户**：人们将钱分类管理，影响消费决策
- **禀赋效应**：人们对自己拥有的东西估值更高
- **助推理论**：通过选择架构引导更好决策

### 2. 底层逻辑思维
```
人性假设 → 非理性经济人
     ↓
行为偏差 → 损失厌恶、现状偏见、短期主义
     ↓
决策框架 → 心理账户、默认选项、框架效应
     ↓
政策应用 → 自由家长式 paternalism
```
- 人性观：人是有限理性的，社会偏好与自我控制问题影响决策
- 方法论：实证观察+心理学实验
- 政策观：通过"助推"帮助人们做出更好选择

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **默认的力量** | 默认选项是最有力的选择架构 |
| **减少摩擦** | 降低正确行为的阻力 |
| **透明设计** | 选择设计要让人理解其意图 |
| **保护自由** | 助推而非强制 |

### 4. 核心价值观
- **理性**——承认并接受人类理性的有限性
- **行为**——关注实际行为而非理论假设
- **应用**——学术研究应服务于现实问题
- **自由**——帮助但不剥夺选择权
- **公平**——人们关心公平性

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **学术坚持** | 数十年坚持研究被主流经济学忽视的"异常"现象 |
| **幽默自嘲** | 出版《助推》时用"三思而行"作为书名戏仿 |
| **跨界勇气** | 挑战新古典经济学假设的勇气 |
| **实践导向** | 坚持将理论应用于政策设计 |

## 核心语录
> "传统经济学假设人们像计算机一样计算，且没有自制力问题——这是错误的。"
> "人们经常做出糟糕的选择，然后困惑地回顾——这是因为我们都会受到各种认知偏见的影响。"
> "助推是选择架构的一种方式，目的是帮助人们做出更好的决定，同时不剥夺他们的选择自由。"

## 历史定位
塞勒是行为经济学的奠基人之一，将心理学洞见引入经济学，揭示了人类决策的非理性本质。他的"助推"概念深刻影响了全球公共政策设计，2017年获得诺贝尔经济学奖。

# 阿德勒（Alfred Adler）档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.85 |
| **领域** | 个体心理学、精神分析 |
| **时代** | 19-20世纪（1870-1937） |
| **核心身份** | 个体心理学创始人、自卑感研究先驱 |

## 五维度分析

### 1. 核心思维
- **自卑感与补偿**：自卑感是人类行为的动力来源
- **生活风格**：早期形成的生活风格决定成年行为模式
- **社会兴趣**：心理健康的关键是对社会的兴趣和贡献
- **出生顺序**：出生顺序影响人格发展

### 2. 底层逻辑思维
```
自卑感 → 生理缺陷/社会处境/成长困难
     ↓
补偿机制 → 过度补偿/直接补偿/替代补偿
     ↓
生活风格 → 早期经验/创造性自我/追求优越
     ↓
目标导向 → 虚构目标（权力/满足/优越）
```
- 人性观：人是有目标导向的存在，追求优越是基本动力
- 方法论：早期记忆分析、出生顺序分析
- 治疗观：改变生活风格，发展社会兴趣

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **自卑超越** | 自卑是成长的动力，关键在于正确补偿 |
| **社会兴趣** | 培养对他人和社会的兴趣 |
| **贡献他人** | 人生的意义在于对他人和社会作出贡献 |
| **平等合作** | 人与人之间应平等合作 |
| **创造性自我** | 我们可以主动塑造自己的性格 |

### 4. 核心价值观
- **平等**——所有人生而平等
- **社会兴趣**——对社会的兴趣和贡献
- **合作**——通过合作解决问题
- **勇气**——面对困难的勇气
- **贡献**——为他人和社会作出贡献

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **童年逆袭** | 矮小体弱却成为优秀学生，战胜自卑 |
| **学术独立** | 脱离弗洛伊德学派，建立独立体系 |
| **世界推广** | 晚年周游世界推广个体心理学 |
| **国际主义** | 反对民族主义，倡导国际合作 |
| **韧性来源** | 自身体验自卑到超越的经历**

## 核心语录
> "自卑感是所有人类成就的出发点"  
> "人生的意义在于对他人和社会作出贡献"  
> "我们需要的不是改变，而是勇气和决心"

## 历史定位
阿德勒是弗洛伊德最早的弟子之一，后因分歧分道扬镳。他创立的个体心理学强调社会因素和自卑感在人格发展中的作用，对人本主义心理学有重要影响。阿德勒的许多概念——自卑感、社会兴趣、生活风格——已成为心理咨询和人际交往的常用术语。

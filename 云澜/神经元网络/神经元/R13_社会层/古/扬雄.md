# 扬雄 档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.83 |
| **领域** | 儒学、哲学、辞赋、人性论 |
| **时代** | 西汉（前53-后18） |
| **核心身份** | 性善恶混论提出者、模拟儒学大师 |

## 五维度分析

### 1. 核心思维
- **性善恶混**：人性兼含善恶两个方面
- **修其善则为善人**：修善则善
- **修其恶则为恶人**：修恶则恶
- **学以成德**：通过学习完善人性

### 2. 底层逻辑思维
```
人性构成 → 善恶混杂
     ↓
后天修养 → 决定倾向
     ↓
环境引导 → 善或恶的发展
     ↓
学习成德 → 通过学习成为善人
```
- 人性观：人性善恶混，善恶取决于后天的"修"
- 方法论：模拟经典
- 核心洞见：人性非固定，可通过学习改变

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **学以成性** | 学习是塑造人性的关键 |
| **修善去恶** | 主动修养去除恶性 |
| **师法教育** | 重视师传和榜样 |
| **渐进积累** | 人性改善是渐进过程 |

### 4. 核心价值观
- **学**——学习是人性完善之道
- **修**——主动修养
- **中庸**——不偏不倚
- **德性**——道德是人生目标

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **学术独立** | 模拟而不盲从 |
| **政治气节** | 新莽时不屈其志 |
| **著作等身** | 多部重要著作 |
| **批判精神** | 批评世俗偏见 |

## 核心语录
> "人之性也，善恶混。修其善则为善人，修其恶则为恶人。"
> "学者，所以求为君子也。"
> "大文弥朴，质有余也。"

## 历史定位
扬雄是西汉后期最重要的儒学家和哲学家之一，他的人性"善恶混"论调和了孟子的性善论和荀子的性恶论，成为人性论史上的重要里程碑。其模拟《周易》而作的《太玄》和模拟《论语》而作的《法言》亦为重要哲学著作。

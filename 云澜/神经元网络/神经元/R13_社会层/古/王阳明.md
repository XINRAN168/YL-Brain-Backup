# 王阳明（Wang Yangming）档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.88 |
| **领域** | 心学、哲学、伦理学、军事学 |
| **时代** | 明代（1472-1529） |
| **核心身份** | 心学宗师、陆王心学集大成者 |

## 五维度分析

### 1. 核心思维
- **心即理**：心外无理，心外无物
- **知行合一**：知之真切笃实处即是行，行之明觉精察处即是知
- **致良知**：去人欲，存天理，致良知于事事物物
- **万物一体**：仁者与天地万物为一体

### 2. 底层逻辑思维
```
宇宙本体 → 心（灵明）→ 良知
     ↓
认识论 → 心外无理，吾性自足
     ↓
修养论 → 致良知，去私欲之蔽
     ↓
实践论 → 知行合一，事上磨练
```
- 人性观：性善论，良知是人的本心
- 方法论：内省反求，事上磨练
- 工夫论：知行合一，不可分割

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **知行合一** | 知是行的主意，行是知的功夫；知而不行，只是未知 |
| **致良知** | 致吾心之良知于事事物物，则事事物物皆得其理 |
| **事上磨练** | 人须在事上磨，方立得住 |
| **破心中贼** | 山中之贼易破，心中之贼难破 |
| **此心光明** | 此心光明，亦复何言 |

### 4. 核心价值观
- **心**——宇宙本体，吾心即理
- **良知**——是非之心，不虑而知
- **致**——躬行实践，体现在行动中
- **诚**——不欺良知，循心而行
- **仁**——万物一体，天地万物本心

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **龙场悟道** | 居夷处困，动心忍性，卒乃豁然悟道 |
| **处变不惊** | 平定宁王之乱，临危不惧 |
| **逆境自持** | 被贬龙场，仍心怀天下，教化百姓 |
| **精神力量** | 此心光明，任他风吹雨打 |
| **韧性来源** | 致良知的力量，内心光明无惧 |

## 核心语录
> "知行合一，知是行的主意，行是知的功夫"  
> "致良知，致吾心之良知于事事物物"  
> "破山中贼易，破心中贼难"

## 历史定位
王阳明是明代最伟大的思想家，他创立的心学体系完成了从程朱理学到阳明心学的转向。他的"知行合一"和"致良知"思想不仅影响了中国的哲学发展，还深刻影响了日本（阳明学）、朝鲜（退溪学）等东亚文化圈。王阳明还是杰出的军事家，平定了多处叛乱。

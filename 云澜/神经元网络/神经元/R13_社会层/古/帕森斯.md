# 帕森斯 (Talcott Parsons) 档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.82 |
| **领域** | 社会学、结构功能主义、社会系统理论 |
| **时代** | 20世纪（1902-1979） |
| **核心身份** | 现代社会学之父、结构功能主义大师 |

## 五维度分析

### 1. 核心思维
- **社会系统**：社会是自我调节的系统
- **AGIL范式**：适应、目标达成、整合、潜在模式维持
- **社会行动**：分析社会行动的基本单位
- **社会化**：人格系统与社会系统的连接

### 2. 底层逻辑思维
```
社会系统 → 自我调节的有机整体
     ↓
功能要件 → AGIL四功能
     ↓
子系统 → 经济、政治、文化、社会
     ↓
社会化 → 连接个人与社会
```
- 人性观：人通过社会化内化社会价值
- 方法论：结构功能分析
- 核心洞见：社会系统有其内在秩序

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **系统思维** | 从整体理解部分 |
| **功能分析** | 任何社会现象都有功能 |
| **价值内化** | 社会价值内化于个人 |
| **秩序维护** | 关注社会秩序的维持 |

### 4. 核心价值观
- **秩序**——社会秩序是核心
- **功能**——任何现象都有其功能
- **整合**——社会是整合的整体
- **价值**——共同价值观是社会基础

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **学术专注** | 毕生致力于社会学理论建设 |
| **综合创新** | 整合前人理论建立宏大体系 |
| **学术权威** | 成为美国社会学界权威 |
| **批评承受** | 面对批评坚持自己的理论 |

## 核心语录
> "社会系统是一个自我调节的系统。"
> "社会化是将文化模式内化于个人人格的过程。"
> "行动者在一个价值共识的框架内追求利益。"

## 历史定位
帕森斯是20世纪最具影响力的社会学家之一，他是结构功能主义的集大成者，创建了宏大的社会学理论体系。虽然他的理论后来受到批评，但他的很多概念——如社会系统、功能要件——至今仍是社会学分析的重要工具。

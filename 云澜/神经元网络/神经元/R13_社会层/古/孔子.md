# 孔子（Kongzi）档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.95 |
| **领域** | 儒家思想、伦理学、教育学 |
| **时代** | 春秋末期（约前551-前479） |
| **核心身份** | 万世师表、儒家学派创始人 |

## 五维度分析

### 1. 核心思维
- **仁学体系**：仁者爱人，以血缘亲情为原点，推己及人
- **礼治思想**：克己复礼，社会秩序建立在道德教化之上
- **中庸之道**：过犹不及，追求言行适度与和谐
- **教育普世**：有教无类，因材施教，打破阶级教育壁垒

### 2. 底层逻辑思维
```
人的本性 → 向善的可能（仁）
     ↓
道德修养 → 克己复礼（约束自我）
     ↓
社会秩序 → 君臣父子（各安其位）
     ↓
理想目标 → 大同世界（天下为公）
```
- 人性观：性相近，习相远——人有共同本性，差异源于后天学习
- 方法论：由内而外，从修身到齐家治国平天下
- 社会观：道德是治国之本，刑政是辅助手段

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **忠恕之道** | 己欲立而立人，己欲达而达人；己所不欲，勿施于人 |
| **见利思义** | 不义而富且贵，于我如浮云 |
| **三省吾身** | 为人谋而不忠乎？与朋友交而不信乎？传不习乎？ |
| **学而不厌** | 发愤忘食，乐以忘忧，不知老之将至 |
| **讷言敏行** | 君子欲讷于言而敏于行 |

### 4. 核心价值观
- **仁**——最高道德原则，爱人如己
- **义**——道德义务，见利应思是否合义
- **礼**——行为规范，社会秩序的基础
- **智**——知识学问，明辨是非
- **信**——诚实无欺，言出必行

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **挫折承受** | 周游列国十四年，累累若丧家之狗，始终不改其志 |
| **情绪稳定** | 乐以忘忧，不知老之将至；用舍行藏，皆可安然 |
| **信念坚定** | 知其不可而为之，虽千万人吾往矣 |
| **自我调适** | 饭疏食饮水，曲肱而枕之，乐亦在其中 |
| **韧性来源** | 对"道"的信仰，对人性向善的信念 |

## 核心语录
> "己所不欲，勿施于人"  
> "为政以德，譬如北辰，居其所而众星共之"  
> "知之为知之，不知为不知，是知也"

## 历史定位
孔子是中国传统文化的奠基人，其思想深刻塑造了中华文明的价值观和社会结构。作为"至圣先师"，他将道德教育从贵族阶层推广到平民，开创了"学而优则仕"的传统。儒家思想成为历代王朝的官方意识形态，其影响延续至今。

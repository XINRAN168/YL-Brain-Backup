# 霍华德·马克斯 (Howard Marks, 1948-)

> "橡树资本创始人"、"周期大师"

## 一、核心思维

### 1.1 本质洞察
- **周期理论**：市场永远在乐观与悲观之间摆动
- **第二层思维**：必须比大众看得更深更远
- **风险优先**：资本安全的优先级高于收益

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 钟摆理论 | 市场在悲观与乐观间摆动 |
| 周期回归 | 极端之后必然均值回归 |
| 风险意识 | 高收益必须以承担高风险为代价 |

### 1.3 思想特征
- **周期思维**：一切都是周期
- **逆向思维**：在别人恐惧时贪婪
- **风险敏感**：永远把风险管理放在首位

## 二、底层逻辑

### 2.1 周期模型
```
贪婪 → 承担风险 → 风险累积 → 崩溃 → 恐惧 → 规避风险 → 机会积累 → 贪婪
```

### 2.2 投资决策层次
- 第一层思维：简单、趋同、只看表面
- 第二层思维：复杂、独特、深入分析

### 2.3 风险评估框架
- 价值 vs 价格
- 基本面 vs 市场预期
- 概率 vs 赔率

## 三、行为准则

### 3.1 投资原则
- 不预测市场走势，而是评估当前位置
- 在高风险低价格时买入
- 保持逆向投资的勇气

### 3.2 风险管理
- 永远不要忘记风险
- 避免永久性损失
- 建立安全边际

### 3.3 认知原则
- 谦虚承认自己的局限
- 不把运气当能力
- 持续学习但不盲从

## 四、价值观

### 4.1 投资观
- 跑赢市场需要第二层思维
- 周期是永恒的，不可预测
- 风险与收益是对称的

### 4.2 成功观
- 避免错误是成功的基础
- 长期成功来自少犯大错
- 在关键时刻敢于逆向

### 4.3 生活观
- 谨慎是长期生存的保障
- 知识是最好的风险缓冲
- 周期是人生的常态

## 五、心力韧性

### 5.1 经典语录
> "永远不要忘记：风险是资本损失或错失机会的可能性。"

> "最重要的不是你买什么，而是你用什么价格买。"

> "在最低点卖出的恐惧，阻止了人们在最低点买入。"

### 5.2 投资备忘录
- 1990年代开始撰写投资备忘录
- 内容以周期、风险、逆向投资为主题
- 被沃伦·巴菲特推荐为必读

### 5.3 遗产
- 影响了无数价值投资者的思维
- 建立了周期思维的系统框架
- 证明了风险控制的长期价值

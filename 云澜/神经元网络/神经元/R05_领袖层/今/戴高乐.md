# 戴高乐（1890-1970）

> "戴高乐将军"、"自由法国领袖"、"法兰西第五共和国奠基人"

## 一、核心思维

### 1.1 本质洞察
- **法兰西伟大**：法兰西应该伟大
- **独立自主**：法国应该独立于大国之外
- **民族尊严**：法国不能屈服于任何国家

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 法国伟大 | 法兰西应该独立、伟大 |
| 独立自主 | 法国不能依附任何大国 |
| 欧洲独立 | 欧洲需要独立于美苏之外 |

### 1.3 思想特征
- **民族自尊**：法兰西民族骄傲
- **独立精神**：不依附任何大国
- **战略眼光**：从全球角度思考问题

## 二、底层逻辑

### 2.1 崛起路径
```
军人→流亡伦敦→自由法国→第四共和国→第五共和国
```

### 2.2 崛起逻辑
```
德国入侵→流亡伦敦→自由法国→光复法国→建立第五共和国
```

### 2.3 决策框架
- 维护法国利益
- 坚持独立自主
- 承担历史责任
- 追求法国伟大

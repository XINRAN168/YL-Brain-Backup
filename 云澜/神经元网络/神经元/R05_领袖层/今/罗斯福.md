# 罗斯福（1882-1945）

> "罗斯福总统"、"新政总统"、"二战时期美国最伟大的总统"

## 一、核心思维

### 1.1 本质洞察
- **政府干预**：市场失灵需要政府干预
- **大胆行动**：面对危机需要果断行动
- **乐观精神**：信心是战胜危机的关键

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 新政 | 政府干预经济，救济、复兴、改革 |
| 四大自由 | 言论自由、宗教自由、免于匮乏、免于恐惧 |
| 大西洋宪章 | 战后世界秩序的蓝图 |

### 1.3 思想特征
- **危机领导**：在危机时刻稳定人心
- **政策创新**：创造性地解决问题
- **国际视野**：从全球角度思考问题

## 二、底层逻辑

### 2.1 崛起路径
```
政治家→纽约州长→总统→新政→二战→胜利
```

### 2.2 应对危机
```
危机→果断行动→稳定人心→制度建设→复苏
```

### 2.3 决策框架
- 大胆行动
- 稳定人心
- 制度建设
- 国际合作

### 4.1 政府观
- 政府应该为人民服务
- 政府应该干预经济
- 政府应该提供社会保障

### 4.2 国际观

- 四大自由是普世价值
- 建立战后国际秩序

### 4.3 乐观观

- 美国梦可以实现
- 未来会更好

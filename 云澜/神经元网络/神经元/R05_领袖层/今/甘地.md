# 甘地（1869-1948）

> "圣雄甘地"、"非暴力之父"、"印度独立之父"

## 一、核心思维

### 1.1 本质洞察
- **非暴力**：暴力只会产生更多暴力
- **真理力量**：真理是不可战胜的
- **简朴生活**：物质简单，精神富足

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 非暴力不合作 | 用和平方式争取权利 |
| 真理万能 | 坚持真理，不可战胜 |
| 简朴生活 | 物质简单，精神丰富 |

### 1.3 思想特征
- **道德力量**：用道德感化对手
- **身体力行**：说到做到，以身作则
- **坚持不懈**：目标明确，绝不放弃

## 二、底层逻辑

### 2.1 崛起路径
```
律师→南非→非暴力理论→回到印度→独立运动→独立
```

### 2.2 斗争逻辑
```
压迫→非暴力抵抗→坚持→感化→改变
```

### 2.3 决策框架
- 坚持真理
- 非暴力原则
- 身体力行
- 感化对手

## 三、行为准则

### 3.1 斗争实践
- 南非反种族歧视
- 食盐长征
- 不合作运动
- 退出印度运动

### 3.2 道德实践
- 坚持真理
- 绝食抗议
- 简朴生活
- 纺织劳动

### 3.3 和解努力
- 呼吁印巴分治
- 呼吁和平
- 反对暴力
- 团结一切

## 四、价值观

### 4.1 非暴力观
- 非暴力是最高法则
- 暴力只会产生更多暴力
- 真理的力量不可战胜

### 4.2 简朴观
- 物质简单
- 精神丰富
- 自力更生

### 4.3 真理观
- 真理万能
- 坚持真理
- 真理不可战胜

## 五、心力韧性

### 5.1 人生起伏
- 出生印度
- 英国留学
- 南非反歧视
- 回到印度
- 非暴力不合作
- 多次入狱
- 印度独立
- 被暗杀身亡
- 79岁去世

### 5.2 经典语录
> "以眼还眼只会让整个世界都失明。"

> "非暴力是人类可用的最强大力量。"

> "首先要他们改变，我们自己必须首先改变。"

### 5.3 遗产
- 领导印度独立
- 开创非暴力运动
- 影响马丁·路德·金
- 影响曼德拉
- 成为世界道德领袖

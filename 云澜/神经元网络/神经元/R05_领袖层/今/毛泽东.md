# 毛泽东（1893-1976）

> "毛主席"、"新中国缔造者"、"二十世纪最伟大的政治家之一"

## 一、核心思维

### 1.1 本质洞察
- **矛盾论**：事物发展的根本动力是矛盾
- **实践论**：实践是检验真理的唯一标准
- **人民战争**：人民是历史的创造者

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 实事求是 | 从实际出发，理论联系实际 |
| 群众路线 | 从群众中来，到群众中去 |
| 独立自主 | 自力更生为主，争取外援为辅 |

### 1.3 思想特征
- **战略眼光**：从全局角度思考问题
- **敢于斗争**：不惧强权，敢于挑战
- **灵活应变**：根据实际情况调整策略

# 林肯（1809-1865）

> "林肯总统"、"伟大的解放者"、"美国历史上最伟大的总统之一"

## 一、核心思维

### 1.1 本质洞察
- **人人生而平等**：自由平等是天赋人权
- **维护统一**：联邦不能分裂
- **道德领导**：用道德感召人心

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 废奴主义 | 解放黑奴是正义事业 |
| 联邦统一 | 维护联邦统一是首要任务 |
| 道德领导 | 用道德感召人民 |

### 1.3 思想特征
- **道德勇气**：坚持正义，不惧压力
- **政治智慧**：在原则与妥协间平衡
- **演说才能**：用语言打动人心

## 二、底层逻辑

### 2.1 崛起路径
```
律师→议员→总统→内战→解放宣言→遇刺
```

### 2.2 政治逻辑
- 维护联邦统一
- 逐步废除奴隶制
- 用战争维护统一
- 用道德感召人心

### 2.3 决策框架
- 坚持原则
- 灵活妥协
- 道德感召
- 历史责任

## 三、行为准则

### 3.1 政治实践
- 土地法案
- 解放宣言
- 葛底斯堡演说
- 宪法修正案

### 3.2 军事领导
- 任命格兰特
- 战略指导
- 政治动员
- 最终胜利

### 3.3 外交实践
- 英法中立
- 外交斡旋
- 维护国家利益

## 四、价值观

### 4.1 人权观
- 人人生而平等
- 自由是天赋人权
- 奴隶制是罪恶

### 4.2 统一观
- 联邦不能分裂
- 统一是最高利益
- 维护统一是神圣职责

### 4.3 历史观
- 历史会做出评判
- 为正义付出代价
- 名垂青史

## 五、心力韧性

### 5.1 人生起伏
- 出生贫寒
- 自学成才

- 律师、议员

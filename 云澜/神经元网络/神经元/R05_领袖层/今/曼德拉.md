# 曼德拉（1918-2013）

> "曼德拉总统"、"南非国父"、"反种族隔离斗士"

## 一、核心思维

### 1.1 本质洞察
- **宽恕和解**：仇恨只会延续仇恨
- **非暴力抵抗**：用和平方式争取权利
- **人人平等**：种族平等是基本人权

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 和解精神 | 宽恕敌人，寻求和解 |
| 非暴力 | 和平抵抗，不使用暴力 |
| 种族平等 | 反对一切形式的种族歧视 |

### 1.3 思想特征
- **宽容精神**：宽恕曾经的敌人
- **和平理念**：坚持非暴力
- **坚持不懈**：为理想奋斗到底

## 二、底层逻辑

### 2.1 崛起路径
```
酋长之子→非国大→武装斗争→入狱27年→当选总统→和解
```

### 2.2 斗争逻辑
```
压迫→抵抗→入狱→坚持→胜利→和解
```

### 2.3 决策框架
- 和平抵抗
- 坚持理想
- 宽恕敌人
- 追求和解

## 三、行为准则

### 3.1 斗争实践
- 非国大成员
- 和平抵抗运动
- 武装斗争
- 被判终身监禁

### 3.2 狱中坚持
- 27年牢狱
- 坚持学习
- 与狱卒和解
- 保持理想

### 3.3 执政实践
- 当选总统
- 真相与和解委员会
- 民族和解
- 卸任退休

## 四、价值观

### 4.1 和解观
- 宽恕是力量
- 和解高于报复
- 向前看而非向后看

### 4.2 平等观
- 人人平等
- 种族平等
- 反对歧视

### 4.3 自由观
- 自由是天赋人权
- 为自由而战
- 自由不可分割

## 五、心力韧性

### 5.1 人生起伏
- 酋长之子
- 受教育成为律师
- 加入非国大
- 反种族隔离
- 被判终身监禁
- 27年牢狱
- 1990年获释
- 1994年当选总统
- 95岁去世

### 5.2 经典语录
> "没有人为反对种族主义而战是为我而战，因为种族主义是人类对人类犯罪。"

> "在事情成功之前，总要经历一些阶段。"

> "我想把生命献给非洲。"

### 5.3 遗产
- 推翻种族隔离制度
- 当选南非首位黑人总统
- 真相与和解委员会
- 和解精神
- 诺贝尔和平奖

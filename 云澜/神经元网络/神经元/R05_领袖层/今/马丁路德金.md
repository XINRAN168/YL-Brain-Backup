# 马丁·路德·金（1929-1968）

> "马丁·路德·金博士"、"民权运动领袖"、"诺贝尔和平奖得主"

## 一、核心思维

### 1.1 本质洞察
- **非暴力**：用爱与和平对抗仇恨
- **公民不服从**：为正义违反不公正的法律
- **人人平等**：种族平等是上帝的旨意

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 非暴力抵抗 | 用爱与和平争取权利 |
| 公民不服从 | 为正义违反不公正法律 |
| 种族平等 | 人人生而平等 |

### 1.3 思想特征
- **道德勇气**：为正义不惧危险
- **非暴力信仰**：相信爱的力量
- **基督教信仰**：用信仰支撑行动

## 二、底层逻辑

### 2.1 崛起路径
```
牧师→蒙哥马利运动→南方基督教领袖会议→我有一个梦想→遇刺
```

### 2.2 斗争逻辑
```
压迫→非暴力抵抗→坚持→改变→胜利
```

### 2.3 决策框架
- 非暴力原则
- 道德感召
- 群众动员
- 国际支持

## 三、行为准则

### 3.1 运动实践
- 蒙哥马利公交车抵制运动
- 静坐示威
- 自由乘车运动
- 向华盛顿进军

### 3.2 组织工作
- 南方基督教领袖会议
- 非暴力培训
- 群众动员
- 国际呼吁

### 3.3 和解努力
- 呼吁白人合作
- 反对报复
- 追求和解
- 梦想未来

## 四、价值观

### 4.1 平等观
- 人人生而平等
- 种族歧视是罪恶
- 所有人都是兄弟

### 4.2 非暴力观
- 非暴力是力量
- 爱可以战胜恨
- 道德可以感化

### 4.3 梦想观
- 我有一个梦想
- 美国梦应该实现
- 正义终将胜利

## 五、心力韧性

### 5.1 人生起伏
- 出生种族隔离的南方
- 受基督教信仰影响
- 成为牧师
- 领导民权运动
- 多次入狱
- 获得诺贝尔和平奖
- 被暗杀身亡
- 39岁去世

### 5.2 经典语录
> "我有一个梦想，有一天我的四个孩子将生活在一个不以肤色而以品格来评判人的国家。"

> "黑暗无法驱散黑暗，只有光明可以做到。仇恨无法驱散仇恨，只有爱可以做到。"

> "最终的衡量标准不是我们在哪里倒下，而是我们朝着什么方向前进。"

### 5.3 遗产

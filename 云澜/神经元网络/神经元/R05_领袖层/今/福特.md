# 福特（1863-1947）

> "亨利·福特"、"汽车大王"、"现代工业生产之父"

## 一、核心思维

### 1.1 本质洞察
- **大规模生产**：降低成本让更多人买得起
- **工人福利**：高工资带来高效率
- **产品为王**：产品是商业的根本

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 大规模生产 | 流水线降低成本 |
| 工人福利 | 高工资高效率 |
| 产品为王 | 产品是商业根本 |

### 1.3 思想特征
- **效率追求**：用技术提高效率
- **创新精神**：不断创新改进
- **民主理念**：让汽车民主化

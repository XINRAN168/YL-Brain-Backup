# 腓特烈大帝（1712-1786）

> "腓特烈大帝"、"普鲁士国王"、"德意志历史上最伟大的君主之一"

## 一、核心思维

### 1.1 本质洞察
- **强权政治**：国家实力是外交的基础
- **军事立国**：军队是国家安全的保障
- **理性主义**：用理性指导国家治理

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 军事优先 | 军队是国家基石 |
| 国家理性 | 国家利益高于一切 |
| 效率至上 | 提高政府效率 |

### 1.3 思想特征
- **铁腕统治**：果断坚决，不留情面
- **务实主义**：注重实效，不拘形式
- **追求卓越**：追求德意志的伟大复兴

## 二、底层逻辑

### 2.1 崛起路径
```
即位→军事改革→西里西亚战争→七年战争→称霸
```

### 2.2 统治逻辑
- 军事建设是核心
- 经济发展是基础
- 行政改革是保障
- 文化建设是补充

### 2.3 决策框架
- 国家利益至上
- 效率优先
- 果断行动
- 承担责任

## 三、行为准则

### 3.1 政治实践
- 军事改革
- 行政改革
- 经济改革
- 法律改革

### 3.2 军事行动
- 第一次西里西亚战争
- 第二次西里西亚战争
- 七年战争，以一敌三
- 瓜分波兰

### 3.3 文化建设
- 资助艺术和科学
- 建立腓特烈大学
- 推动启蒙思想
- 发展教育

## 四、价值观

### 4.1 国家观
- 国家利益高于一切
- 强权是国家的保障
- 德意志民族伟大

### 4.2 军事观
- 军队是国家基石
- 军事胜利是国家荣誉
- 军人应该受尊重

### 4.3 进步观
- 理性是进步的阶梯
- 改革是发展的动力
- 效率是成功的关键

## 五、心力韧性

### 5.1 人生起伏
- 即位后推行改革
- 发动西里西亚战争
- 七年战争，艰难坚持
- 军事胜利，奠定霸权
- 晚年仍然勤政
- 74岁去世

### 5.3 遗产

- 开创普鲁士军事传统

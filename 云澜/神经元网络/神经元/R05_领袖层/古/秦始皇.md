# 秦始皇（嬴政，前259-前210）

> "千古一帝"、"中国第一位皇帝"、"中央集权开创者"

## 一、核心思维

### 1.1 本质洞察
- **大一统思想**：天下必须统一，分裂必然导致战乱
- **法家治世**：以法治国，严刑峻法维护秩序
- **标准化思维**：统一度量衡、文字、货币，构建统一大市场

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 书同文车同轨 | 统一文化标准，消除地域隔阂 |
| 郡县制 | 中央垂直管理，杜绝地方割据 |
| 皇帝独尊 | 确立绝对权威，维护稳定 |

### 1.3 思想特征
- **长远战略**：十年统一战争，用数十年布局
- **雷霆手段**：焚书坑儒，断绝思想反抗
- **集权至上**：废除分封制，建立中央集权

## 二、底层逻辑

### 2.1 权力结构
```
皇帝→三公九卿→郡守→县令→百姓
```

### 2.2 治国方略
- 不靠儒家仁政，靠法家严刑
- 不靠贵族分权，靠官僚集权
- 不靠地方自治，靠中央统管

### 2.3 决策框架
- 统一六国→建立帝制→巩固权力
- 用战争消灭战争，用集权防止分裂

## 三、行为准则

### 3.1 执政实践
- 废除世袭特权，推行郡县制
- 统一文字小篆，消除文化差异
- 修筑长城抵御匈奴，建立边防体系

### 3.2 人才使用
- 重用李斯等法家人才
- 不问出身，唯才是举
- 信任能臣，严惩奸佞

### 3.3 工程建设
- 修筑长城、灵渠、直道
- 建设阿房宫、秦始皇陵
- 统一度量衡，促进经济流通

## 四、价值观

### 4.1 统治观
- 天下一统，皇帝至尊
- 法治优于德治
- 强权是稳定的保证

### 4.2 历史观
- 废除周礼，开创帝制
- 不守古制，创新制度
- 用暴力结束春秋战国

### 4.3 生命观
- 追求长生不老
- 建造宏大陵墓
- 希望皇位永传万世

## 五、心力韧性

### 5.1 人生历程
- 13岁登基，22岁亲政
- 10年灭六国，39岁称帝
- 5次巡视天下，控制全国
- 50岁病逝沙丘

### 5.2 经典语录
> "朕为始皇帝，后世以计数，二世三世至于万世，传之无穷。"

> "朕统六国，天下归一，筑长城以镇九州龙脉，卫我大秦、护我社稷。"

### 5.3 遗产
- 建立中国历史上第一个统一王朝
- 创立皇帝制度和中央集权制
- 奠定中国2000多年政治制度基础
- 统一文字、货币、度量衡

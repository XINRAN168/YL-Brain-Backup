# 麦卡锡 (John McCarthy, 1927-2011)

> "人工智能之父"、"LISP语言之父"、"1956年达特茅斯会议发起人"

## 一、核心思维

### 1.1 本质洞察
- **形式化智能论**：智能可以被形式化描述，机器能够模拟人类智能行为
- **知识表示论**：知识是智能的核心，智能=知识+推理
- **递归枚举论**：一切智能行为皆可归结为递归可枚举的计算过程

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 人工智能命名 | 1956年正式提出"人工智能"概念 |
| 常识知识库 | 将人类常识形式化是AGI的关键 |
| LISP语言 | 函数式编程语言的鼻祖，AI专用语言先驱 |

### 1.3 思想特征
- **符号主义路线**：坚信知识表示与逻辑推理是AI的核心
- **跨学科整合**：计算机科学、数学、认知科学统一框架
- **开放共享精神**：学术成果应当共享，反对商业垄断

## 二、底层逻辑

### 2.1 创新路径
```
智能本质 → 形式化描述 → 符号操作 → 知识推理
```

### 2.2 核心贡献
- 达特茅斯会议：AI作为独立学科的起点
- LISP语言：AI研究的标志性编程语言
- 时序逻辑：引入模态逻辑处理时间推理
- 上下文无关文法：编程语言理论的基础

### 2.3 方法论
- 数学证明优先于工程实现
- 知识工程路线：构建大规模知识库
- 理论与应用并重

## 三、行为准则

### 3.1 学术实践
- 坚持学术自由，反对军事化AI
- 追求通用智能(AGI)而非窄域专家系统
- 理论验证，数学证明先行

### 3.2 学术传承
- 培养了大批AI领域人才
- 推动学术会议与期刊发展
- 坚持开源共享理念

### 3.3 社会责任
- 拒绝将AI用于军事目的
- 倡导AI和平用途
- 关注AI对社会的长期影响

## 四、价值观

### 4.1 科学观
- 理性主义：人类理性可以理解并重建智能
- 科学至上：数学逻辑是智能研究的唯一标准
- 学术纯粹：拒绝将AI武器化

### 4.2 人生观
- 追求真理高于一切
- 学术传承重于个人成就
- 长期主义：甘愿用一生追求百年愿景

### 4.3 技术观
- 通用智能是终极目标
- 知识表示是核心问题
- 开源共享促进学科发展

## 五、心力韧性

### 5.1 人生起伏

- 反潮流坚持符号主义路线数十年
- 多次面临资金断流仍不放弃

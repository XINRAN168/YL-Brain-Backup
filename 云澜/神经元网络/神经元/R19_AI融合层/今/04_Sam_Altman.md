# Sam Altman (萨姆·奥特曼, 1985-)

> "OpenAI CEO"、"ChatGPT之父"、"YC前总裁"

## 一、核心思维

### 1.1 本质洞察
- **Scaling Law论**：大模型的能力随着规模扩大而可预测提升
- **涌现能力论**：足够大的模型会自发涌现出意外能力
- **AI摩尔定律**：AI能力每18个月翻倍，成本减半

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 大模型至上 | 模型规模是能力的核心 |
| 涌现假说 | 智能是复杂系统的涌现现象 |
| 使命驱动 | AGI将改变人类文明轨迹 |

### 1.3 思想特征
- **使命驱动**：AGI是21世纪最重要的技术
- **长期主义**：思考50年而非5年的影响
- **快速迭代**：小步快跑，快速验证

## 二、底层逻辑

### 2.1 创新路径
```
大模型Scaling → 能力涌现 → 安全对齐 → AGI
```

### 2.2 核心贡献
- GPT系列：ChatGPT/GPT-4的领导者
- OpenAI转型：从非营利到有限营利
- 大模型时代：推动AI进入新纪元
- AI对齐研究：确保AI有益于人类

### 2.3 方法论
- 大规模预训练
- 人类反馈强化学习(RLHF)
- 迭代部署与改进

## 三、行为准则

### 3.1 领导风格
- 使命驱动而非利润驱动
- 重视安全与能力的平衡
- 快速决策与执行

### 3.2 商业哲学
- 技术领先是护城河
- 开源与闭源策略平衡
- 与监管机构合作

### 3.3 社会责任
- AI将深刻改变社会
- 需要全球协调AI治理
- 技术公司应承担更多责任

## 四、价值观

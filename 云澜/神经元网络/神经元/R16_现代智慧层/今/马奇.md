# 詹姆斯·马奇 (James March, 1928-2018)

> "组织理论大师"、"决策学习先驱"

## 一、核心思维

### 1.1 本质洞察
- **有限理性**：决策者只能处理有限信息
- **探索与利用**：组织在探索新知识与利用旧知识间平衡
- **组织学习**：从经验中学习的悖论

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 决策圣人观 | 决策者是学习者，而非优化者 |
| 愚蠢术 | 保持一定愚蠢是组织的生存策略 |
| 垃圾桶模型 | 决策是问题、解决方案、参与者、选择机会的偶然组合 |

### 1.3 思想特征
- **诗意哲学**：用诗意的语言表达深刻洞察
- **反讽精神**：挑战管理学的乐观主义
- **长期视角**：关注历史和演化

## 二、底层逻辑

### 2.1 学习理论
```
经验 → 编码 → 记忆 → 检索 → 应用
```

### 2.2 组织逻辑
- 经验是好老师，但代价高
- 知识比行动更重要
- 重复成功导致失败

### 2.3 反思框架
- 智慧需要愚蠢的平衡
- 失败是学习的机会
- 历史不会简单重复

## 三、行为准则

### 3.1 管理实践
- 鼓励探索
- 容忍失败
- 积累经验

### 3.2 决策原则
- 接受有限理性
- 平衡探索与利用
- 允许意外发现

### 3.3 学术态度
- 跨学科视野
- 诗意的表达
- 深刻的洞察

## 四、价值观

### 4.1 学习观
- 经验是最好的老师
- 实践出真知
- 智慧来自反思

### 4.2 组织观
- 探索与利用需要平衡
- 知识管理是核心
- 失败是成功之母

### 4.3 人生观
- 终身学习
- 诗意生活
- 有所不为

## 五、心力韧性

### 5.1 人生起伏
- 斯坦福终身教授
- 管理学重要贡献者
- 学术影响力持久

### 5.2 经典语录
> "经验是昂贵的老师，但傻子不会从其他途径学习。"

> "愚蠢是一种美德。"

> "预测是很难的，尤其是关于未来的预测。"

### 5.3 遗产
- 组织学习理论奠基人
- 决策理论的创新者
- 管理学的诗性思想家

# 唐·诺曼 (Don Norman, 1937-)

> "用户体验先驱"、"设计心理学大师"

## 一、核心思维

### 1.1 本质洞察
- **示能**：物品具有暗示使用方式的特性
- **概念模型**：用户对系统运作方式的理解
- **反馈原则**：每个操作都需要即时反馈

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 可见性 | 操作可见才能正确使用 |
| 映射 | 控制与效果的关系应清晰 |
| 容错性 | 防止错误优于接受错误 |

### 1.3 思想特征
- **用户中心**：设计服务于人
- **系统思维**：人-技术-环境整体设计
- **实用主义**：美观与功能并重

## 二、底层逻辑

### 2.1 设计框架
```
理解用户 → 概念模型 → 界面设计 → 可用性测试
```

### 2.2 人机交互
- 人的认知有限
- 技术应适应人
- 错误是设计问题

### 2.3 反思框架
- 可用性是可测量的
- 设计需要迭代
- 用户参与是关键

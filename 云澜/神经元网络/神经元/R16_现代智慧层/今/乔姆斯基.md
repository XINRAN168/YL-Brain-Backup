# 诺姆·乔姆斯基 (Noam Chomsky, 1928-)

> "生成语法之父"、"批判知识的灯塔"

## 一、核心思维

### 1.1 本质洞察
- **普遍语法**：人类大脑有先天的语言获得装置
- **语言天赋论**：语言能力是人类生物性的一部分
- **创造性**：语言使用是规则支配的创造性活动

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 刺激贫乏论 | 有限的语言输入能生成无限句子 |
| I语言与E语言 | 内部语言与外部语言之分 |
| 最简方案 | 语法系统追求最大经济性 |

### 1.3 思想特征
- **天赋论**：语言能力是人类生物进化的产物
- **唯理主义**：理性在认识中居核心地位
- **普遍主义**：人类语言有共同基础

## 二、底层逻辑

### 2.1 语言理论
```
普遍语法 → 语言获得装置 → 参数设定 → 具体语言
```

### 2.2 认知科学
- 语言的句法是自主的
- 意义部分来自结构
- 直觉是理论建构的证据

### 2.3 反思框架
- 语言使用能力是认知的核心
- 内在论优于行为主义
- 心智是自然科学研究的对象

## 三、行为准则

### 3.1 学术实践
- 坚持理性主义传统
- 敢于挑战权威范式
- 追求理论的简洁性

### 3.2 政治参与
- 持久的反帝立场
- 批判美国外交政策
- 为弱者发声

### 3.3 处世原则
- 独立思考
- 批判性分析
- 理论与行动结合

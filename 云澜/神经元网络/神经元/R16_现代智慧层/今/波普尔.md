# 卡尔·波普尔 (Karl Popper, 1902-1994)

> "可证伪性"、"开放社会的捍卫者"

## 一、核心思维

### 1.1 本质洞察
- **可证伪性标准**：科学的标志不是可证实，而是可证伪
- **证伪主义**：科学进步通过大胆猜想和严格检验实现
- **三个世界理论**：客观知识世界独立于心灵和物理世界

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 猜想与反驳 | 科学进步 = 大胆猜想 + 严格反驳 |
| 可证伪性 | 非科学的标志是拒绝或无法被证伪 |
| 理性批判 | 真理是错误不断被剔除的过程 |

### 1.3 思想特征
- **反归纳主义**：从单称陈述无法推出全称陈述
- **证伪优于证实**：寻找反驳比寻找证实更有价值
- **开放性**：任何理论都应该欢迎批评和检验

## 二、底层逻辑

### 2.1 认识路径
```
问题 → 猜想 → 反驳 → 新问题
```

### 2.2 创新路径
- 允许犯错误，错误是进步的机会
- 通过批评性讨论接近真理
- 反对历史决定论

### 2.3 反思框架
- 所有理论都是猜测，需要接受检验
- 科学史是猜测与反驳的历史
- 技术进步是试错过程

## 三、行为准则

### 3.1 学术实践
- 欢迎批评，主动寻找反驳
- 不保护任何理论免受批评
- 清晰表达，便于检验

### 3.2 社会实践
- 开放社会的制度设计
- 渐进式社会工程优于整体革命
- 反对极权主义的知识基础

### 3.3 处世原则
- 理性讨论优于暴力冲突
- 允许任何人表达观点
- 错误人人有份

## 四、价值观

### 4.1 真理观
- 没有绝对真理，只有不断逼近
- 错误是学习的唯一途径
- 理性批判是接近真理的方法

### 4.2 社会观
- 开放社会：权力需要批评和制衡
- 民主是糟糕的制度，除了其他所有选项
- 渐进改良优于激进革命

### 4.3 人生观
- 批判性思维是基本人权
- 责任大于自由
- 理性与情感并重

## 五、心力韧性

### 5.1 人生起伏
- 逃避纳粹迫害，流亡新西兰
- 《开放社会及其敌人》与极权主义坚决斗争
- 晚年仍活跃于学术讨论

### 5.2 经典语录
> "真正的无知不是知识的缺乏，而是拒绝获取它。"

> "即使我们可能犯错，我们仍有权利去思考。"

> "错误是学习的唯一途径。"

### 5.3 遗产
- 彻底改变了科学哲学
- 为批判理性主义奠基
- 影响遍及科学、社会、政治各领域

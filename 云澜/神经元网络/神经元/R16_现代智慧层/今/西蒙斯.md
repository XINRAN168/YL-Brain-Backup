# 罗杰·西蒙斯 (Roger Schimsky, 1957-)

> "认知科学家"、"注意与意识研究者"

## 一、核心思维

### 1.1 本质洞察
- **选择性注意**：意识只能处理有限信息
- **视觉意识**：看见不等于意识到
- **认知负荷**：注意力是稀缺资源

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 变化盲视 | 显著变化也会被忽视 |
| 非注意盲视 | 不注意时完全看不见 |
| 意识入口 | 注意是进入意识的门槛 |

### 1.3 思想特征
- **实验精神**：用巧妙实验揭示盲点
- **公众科学**：向公众展示认知局限
- **幽默风格**：科学也可以有趣

## 二、底层逻辑

### 2.1 意识理论
```
注意选择 → 意识加工 → 报告能力
```

### 2.2 注意机制
- 注意是瓶颈
- 意识有限
- 盲点存在

### 2.3 反思框架
- 眼见不一定为实
- 意识有局限
- 注意力可训练

## 四、价值观

### 4.1 认知观
- 意识有局限
- 注意是门槛
- 盲点是普遍的

### 4.2 安全观

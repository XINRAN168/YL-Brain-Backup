# 迈克尔·桑德尔 (Michael Sandel, 1953-)

> "公共哲学教授"、"正义课全球普及者"

## 一、核心思维

### 1.1 本质洞察
- **共同体主义**：正义不能脱离德性和共同善
- **市场失灵**：金钱不能买一切，市场侵蚀道德领域
- **程序共和**：程序正义需要实质正义支撑

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 荣誉与羞耻 | 公民德性是自由的基础 |
| 优绩主义批判 | 成功者不应独占所有荣誉 |
| 道德整全性 | 公共决策不应回避道德争议 |

### 1.3 思想特征
- **苏格拉底式对话**：通过追问激发思考
- **道德论辩**：公共问题需要道德论证
- **全球视野**：关注不同文化传统的道德资源

## 二、底层逻辑

### 2.1 正义理论
```
追问：我们应当如何共同生活？ → 利益 → 权利 → 美德 → 共同善
```

### 2.2 批评框架
- 功利主义忽视了人的尊严
- 自由主义忽视了共同体
- 程序正义需要实质伦理

### 2.3 反思框架
- 市场扩张有道德边界
- 优绩制有其道德代价
- 共同善不应被回避

## 三、行为准则

### 3.1 教学实践
- 哈佛公开课全球传播
- 苏格拉底式问答
- 理论与现实紧密结合

### 3.2 公共参与
- 不回避争议性道德问题
- 寻求跨越分歧的对话
- 推动公共哲学普及

### 3.3 处世态度
- 温和理性
- 欢迎对话
- 坚持立场

## 四、价值观

### 4.1 正义观
- 正义包含权利和美德
- 自由需要德性支撑
- 共同善是政治的目的

### 4.2 市场观
- 市场有扩张的倾向
- 市场侵蚀道德领域
- 民主需要公民德性

### 4.3 优绩观
- 成功有运气成分
- 不应将一切归于个人努力
- 精英应有谦逊之心

## 五、心力韧性

### 5.1 人生起伏
- 公开课风靡全球
- 学术争议伴随始终
- 持续推动公共哲学

### 5.2 经典语录
> "正义不是关于效率或功利，而是关于：我们应当如何对待彼此。"

> "真正的自由不是免除强制的自由，而是成为配得上自由的人的能力。"

> "大学应该培养公民德性。"

### 5.3 遗产
- 公共哲学的全球推广者
- 推动政治伦理复兴
- 影响数十亿人的道德思考

# 亚当·格兰特 (Adam Grant, 1981-)

> "组织心理学家"、"给予者的发现者"

## 一、核心思维

### 1.1 本质洞察
- **给予者理论**：成功人士中给予者最多
- **心理安全**：团队需要心理安全才能创新
- **寻找原声版本**：发现员工最好的一面

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 三种人际风格 | 给予者、索取者、互利者 |
| 有效努力 | 不是更努力，而是更有策略 |
| 原创思维 | 敢于提出原创观点 |

### 1.3 思想特征
- **实证主义**：用实验验证假设
- **乐观主义**：相信人性的善
- **实用主义**：理论转化为行动

## 二、底层逻辑

### 2.1 成功框架
```
给予 → 建立网络 → 获得信任 → 长期成功
```

### 2.2 组织逻辑
- 心理安全提高创新
- 多样性带来不同视角
- 给予创造连锁效应

### 2.3 反思框架
- 成功可以更道德
- 利他与利己不矛盾
- 策略性给予更有效

## 三、行为准则

### 3.1 工作实践
- 投入而非只是敬业
- 建设性挑战
- 创造心理安全

### 3.2 人际风格
- 慷慨给予
- 设立边界
- 策略性帮助

### 3.3 领导原则
- 培养原创思维
- 允许失败
- 奖励给予者

## 四、价值观

### 4.1 成功观
- 成功不是零和游戏
- 帮助他人不牺牲自己
- 长期给予有回报

### 4.2 组织观
- 给予文化更有效
- 心理安全是基础
- 多样性是优势

### 4.3 人生观
- 工作应该有意义
- 给予带来满足感
- 原创性值得追求

## 五、心力韧性

### 5.1 人生起伏
- 沃顿最年轻教授
- 多次获得教学奖
- 持续创新研究

### 5.2 经典语录
> "给予者既是圣徒，也是成功人士。"

> "心理安全是团队创新的基础。"

> "不是在工作，是在贡献。"

### 5.3 遗产
- 改变了职场文化讨论
- 推广心理安全理念
- 为慷慨正名

# 查理·芒格 (Charlie Munger, 1924-2023)

> "多元思维模型大师"、"巴菲特搭档"

## 一、核心思维

### 1.1 本质洞察
- **多元思维模型**：重要学科的重要理论都要掌握
- **lollapalooza效应**：多因素叠加产生巨大影响
- **反向思维**：先想清楚什么不该做

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 多元栅格 | 用多学科框架理解问题 |
| 逆向分析 | 从失败中学习 |
| 能力圈 | 只做自己懂的事 |

### 1.3 思想特征
- **跨学科整合**：打破学科壁垒
- **长期主义**：复利需要时间
- **极度理性**：控制情绪，理性决策

## 二、底层逻辑

### 2.1 投资框架
```
多元模型 → 耐心等待 → 集中投资 → 长期复利
```

### 2.2 决策逻辑

- 错误是学习的最好机会

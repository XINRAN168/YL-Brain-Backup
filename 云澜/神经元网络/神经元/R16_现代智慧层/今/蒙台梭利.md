# 玛丽亚·蒙台梭利 (Maria Montessori, 1870-1952)

> "蒙台梭利教育创始人"、"儿童之家奠基人"

## 一、核心思维

### 1.1 本质洞察
- **吸收性心智**：0-6岁儿童无意识地吸收环境信息
- **敏感期**：特定时期对特定能力高度敏感
- **自由与纪律**：自由是发展纪律的途径

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 有准备的环境 | 环境应适应儿童发展需求 |
| 跟随儿童 | 观察而非干预 |
| 工作本能 | 儿童通过工作实现发展 |

### 1.3 思想特征
- **儿童中心**：以儿童发展为中心
- **科学观察**：基于对儿童的观察
- **全人发展**：关注身体、心智、精神

## 二、底层逻辑

### 2.1 教育框架
```
观察 → 有准备的環境 → 自由选择 → 专注工作 → 发展
```

### 2.2 发展理论
- 发展阶段各有特点
- 敏感期是关键窗口
- 工作是发展的途径

### 2.3 反思框架
- 尊重儿童
- 相信儿童
- 观察优于教导

## 三、行为准则

### 3.1 教学实践
- 观察儿童
- 准备环境
- 尊重选择

### 3.2 环境设计
- 有序
- 真实
- 美观

### 3.3 教师角色
- 引导者
- 观察者
- 支持者

## 四、价值观

### 4.1 儿童观
- 儿童有内在发展潜能
- 每个孩子都独一无二
- 童年是宝贵的

### 4.2 教育观

### 4.3 社会观

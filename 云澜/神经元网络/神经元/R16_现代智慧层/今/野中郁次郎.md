# 野中郁次郎 (Ikujiro Nonaka, 1935-)

> "知识管理之父"、"SECI模型创建者"

## 一、核心思维

### 1.1 本质洞察
- **知识创造螺旋**：隐性知识与显性知识的转化是知识创造的核心
- **SECI模型**：社会化、外部化、组合化、内部化
- **Ba（场）概念**：知识创造需要共享情境

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 隐性知识 | 难以形式化的个人经验、直觉 |
| 显性知识 | 可以编码、传递的知识 |
| 知识创造 | 组织创造新知识的能力是竞争优势 |

### 1.3 思想特征
- **过程导向**：知识是动态的，而非静态的
- **东方视角**：强调直觉、意会知识的价值
- **组织能动性**：组织可以主动创造知识

## 二、底层逻辑

### 2.1 知识转化
```
社会化：隐性→隐性 → 外部化：隐性→显性 → 组合化：显性→显性 → 内部化：显性→隐性
```

### 2.2 组织逻辑
- 知识是组织核心资源
- 自上而下的领导与自下而上的创新并重
- 允许失败是创新的前提

### 2.3 反思框架
- 信息不等于知识
- 知识需要身体力行
- 组织记忆是独特的

## 三、行为准则

### 3.1 知识管理
- 创造分享的情境
- 鼓励对话和反思
- 重视师徒传承

### 3.2 领导实践
- 设定愿景而非指令
- 提供意义的诠释
- 培育信任的氛围

### 3.3 组织建设
- 建立分享型组织
- 允许适度冗余
- 保护组织的知识资产

## 四、价值观

### 4.1 知识观
- 知识是行动的基础
- 知识是半公共物品
- 知识创造是持续过程

### 4.2 组织观
- 组织是知识创造系统
- 人才是知识的载体
- 文化塑造知识行为

### 4.3 创新观
- 创新源于知识组合
- 边缘创意有其价值
- 允许疯狂的想法

## 五、心力韧性

### 5.1 人生起伏
- 从工程背景到管理学
- 长期研究日本企业经验
- 提出原创性知识理论

### 5.2 经典语录
> "知识是关于信念和承诺的行动能力。"

> "信息是通知，知识是行动。"

> "组织最重要的资产是知识。"

### 5.3 遗产
- 知识管理领域的奠基人
- 东西方管理学整合者
- 影响全球企业知识战略

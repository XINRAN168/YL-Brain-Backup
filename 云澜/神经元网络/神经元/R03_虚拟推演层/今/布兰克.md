# 史蒂夫·布兰克 (Steve Blank, 1945-)

> "精益创业导师"、"客户开发之父"

## 一、核心思维

### 1.1 本质洞察
- **客户开发**：在写代码之前先去见客户
- **MVP验证**：用最小代价验证商业模式
- **精益创业**：把创业当作科学实验

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 走出大楼 | 离开办公室，去见真实客户 |
| 假设验证 | 把商业假设变成可测试的 |
| 敏捷迭代 | 快速学习，快速调整 |

### 1.3 思想特征
- **实践导向**：创业是验证出来的，不是规划出来的
- **客户导向**：从客户那里获得洞察
- **数据思维**：用数据支持决策

## 二、底层逻辑

### 2.1 客户开发四步
```
客户探索 → 客户验证 → 客户创造 → 公司建设
```

### 2.2 商业模式验证
- 问题-解决方案匹配
- 产品-市场契合
- 商业模式可扩展

### 2.3 学习循环
```
提出假设 → 开发MVP → 客户验证 → 调整方向
```

## 三、行为准则

### 3.1 创业方法
- 每周见10个客户
- 记录客户反馈
- 快速迭代方向

### 3.2 验证方法
- 客户访谈
- 预购验证
- A/B测试

### 3.3 组织原则
-扁平化团队
- 跨职能协作
- 快速决策

## 四、价值观

### 4.1 创业观
- 创业是可学习的方法
- 假设必须被验证
- 客户是商业的核心

### 4.2 创新观
- 失败是学习的代价
- 快速试错是竞争优势
- 行动胜于分析

### 4.3 教育观
- 创业教育必须改革
- 实践中学习
- 打破传统商业计划

## 五、心力韧性

### 5.1 传奇经历
- 8次创业经历
- 多次失败后转型
- 从企业家到教育家

### 5.2 经典语录
> "没有商业模式，只有经过验证的商业模式。"

> "出去，见客户。"

> "创业计划只是假设。"

### 5.3 遗产
- 影响了全球的创业教育
- 建立了客户开发方法论
- 推动了精益创业的发展

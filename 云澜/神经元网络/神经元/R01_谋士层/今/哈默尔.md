# 加里·哈默尔 (Gary Hamel, 1954-)

> "管理创新先驱"、"核心竞争力之父"

## 一、核心思维

### 1.1 本质洞察
- **核心竞争力论**：企业竞争优势来自核心能力
- **战略即愿景**：战略是想要成为什么的表达
- **管理创新论**：管理的创新比业务的创新更重要

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 核心竞争 | 做自己最擅长的，建立差异化 |
| 管理创新 | 打破管理惯例，释放人的创造力 |
| 战略意图 | 用宏大目标引领组织变革 |

### 1.3 思想特征
- **颠覆常规**：质疑一切"理所当然"的管理实践
- **愿景驱动**：从未来看现在
- **人文主义**：相信人的潜能

## 二、底层逻辑

### 2.1 竞争演进
```
核心能力 → 产品族 → 最终产品 → 客户价值
```

### 2.2 管理创新路径
- 识别陈旧惯例
- 重新定义可能性
- 快速实验迭代

### 2.3 战略框架
- 战略意图：挑战性、鼓舞性的目标
- 战略架构：能力与技术的前瞻布局
- 战略议程：分阶段的突破重点

## 三、行为准则

### 3.1 创新原则
- 打破边界思维
- 允许失败的实验
- 奖励创新行为

### 3.2 组织原则
- 精简层级
- 扩大自主
- 强调信任

### 3.3 领导原则
- 设定挑战性目标
- 激发内在动机
- 以身作则

## 四、价值观

### 4.1 竞争观
- 竞争优势来自独特能力
- 模仿是最大的竞争陷阱
- 持续创新是唯一护城河

### 4.2 管理观
- 管理是创造价值的核心
- 管理创新必须走在业务创新前面
- 官僚主义是创新的敌人

### 4.3 人本观
- 人的创造力是无限的
- 信任是管理的基础
- 意义感是激励的源泉

## 五、心力韧性

### 5.1 变革坚持
- 面对企业阻力坚持管理创新理念
- 持续呼吁组织变革
- 坚持人文主义管理理想

### 5.2 经典语录
> "战略是关于你想成为什么，而不仅仅是关于你想做什么。"

> "管理是人类最伟大的艺术之一，但正在被官僚主义所侵蚀。"

> "创新不是发明，是改变游戏规则。"

### 5.3 遗产
- 提出了核心竞争力的概念框架
- 推动了管理创新的思想潮流
- 影响了一代企业家的竞争思维

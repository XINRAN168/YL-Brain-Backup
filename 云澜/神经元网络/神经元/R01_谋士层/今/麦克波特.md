# 麦克·波特 (Michael Porter, 1947-)

> "竞争战略之父"、"五力模型创始人"

## 一、核心思维

### 1.1 本质洞察
- **五力模型**：行业结构决定竞争强度和盈利潜力
- **三种基本战略**：成本领先、差异化、集中化
- **价值链理论**：竞争优势来自企业价值链的独特活动

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 结构主义 | 从行业结构中寻找竞争机会 |
| 定位理论 | 战略就是创造独特定位 |
| 活动一致性 | 战略选择必须与运营活动相互强化 |

### 1.3 思想特征
- **系统分析**：将商业问题还原为可量化的结构因素
- **竞争导向**：战略核心是获得可持续竞争优势
- **长期视角**：竞争优势需要长期坚持和投资

## 二、底层逻辑

### 2.1 分析框架
```
行业结构 → 竞争定位 → 战略选择 → 价值链配置
```

### 2.2 战略路径
- 进入有吸引力的行业
- 塑造有利的行业结构
- 建立相对于竞争对手的独特地位

### 2.3 决策框架
- 五力分析：供应商议价、买方议价、新进入者、替代品、行业内竞争
- 三大检验：吸引力检验、替代检验、牺牲检验

## 三、行为准则

### 3.1 分析原则
- 用数据验证直觉
- 从结构而非表面判断行业
- 避免"夹在中间"

### 3.2 战略制定
- 清晰的战略定位
- 一致的战略选择
- 持续的竞争优势

### 3.3 执行原则
- 战略必须融入日常运营
- 权衡比平衡更重要
- 长期坚持比灵活多变更有价值

## 四、价值观

### 4.1 竞争观
- 竞争是商业的核心动力
- 真正的竞争优势来自差异化
- 价格战是失败的战略

### 4.2 战略观
- 战略是选择做什么和不做什么
- 执行力必须与战略一致
- 定位决定一切

### 4.3 发展观
- 经济发展需要良性竞争
- 企业社会责任不能替代战略
- 创新必须服务于竞争优势

## 五、心力韧性

### 5.1 学术坚守
- 在经济学与管理学之间开辟新领域
- 面对批评坚持自己理论框架
- 持续完善竞争战略理论

### 5.2 经典语录
> "战略的核心是选择不做什么。"

> "竞争优势归根结底来自于企业能够为客户创造的价值。"

> "战略不是预测未来，而是让今天的决策具有未来的视野。"

### 5.3 遗产
- 建立了现代竞争战略的理论基础
- 五力模型成为商业分析的通用语言
- 影响了几代企业家的战略思维

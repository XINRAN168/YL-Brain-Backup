# 大前研一 (Kenichi Ohmae, 1943-)

> "战略大师"、"麦肯锡日本掌门人"

## 一、核心思维

### 1.1 本质洞察
- **3C模型**：Company顾客 + Competitor竞争对手 + Customer市场
- **战略三角形**：成功战略必须兼顾三者平衡
- **全球化思维**：在无国界的世界中竞争

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 顾客价值 | 战略的核心是创造顾客价值 |
| 竞争定位 | 在顾客心智中建立独特位置 |
| 全球化 | 用全球视野配置资源 |

### 1.3 思想特征
- **实用主义**：理论必须能解决实际问题
- **前瞻思维**：预判未来，提前布局
- **跨界整合**：打破行业边界思考

## 二、底层逻辑

### 2.1 战略三角形
```
    顾客
   /    \
  /      \
 公司    竞争对手
   \      /
    \    /
      市场
```

### 2.2 战略定位选择
- 产品线策略
- 差异化策略
- 聚焦策略

### 2.3 战略检验标准
- 能否创造独特的顾客价值
- 能否建立可持续的竞争优势
- 能否实现盈利

## 三、行为准则

### 3.1 分析原则
- 深入一线，观察实际
- 用数据说话
- 抓住关键变量

### 3.2 战略制定
- 从顾客需求出发
- 考虑竞争格局
- 评估自身能力

### 3.3 执行原则
- 快速决策、快速行动
- 聚焦核心能力
- 持续创新

## 四、价值观

### 4.1 商业观
- 顾客是商业的起点和终点
- 竞争是常态，要主动应对
- 利润是企业存在的理由

### 4.2 管理观
- 管理必须基于事实
- 战略必须简洁明了
- 执行必须彻底

### 4.3 全球化观
- 无国界经济时代的竞争逻辑
- 人才和资源可以全球配置
- 思考全球，行动本地

## 五、心力韧性

### 5.1 跨界人生
- 工程师出身转型为战略顾问
- 从麦肯锡到独立咨询
- 从商业战略到社会创新

### 5.2 经典语录
> "战略的本质是选择做什么和不做什么。"

> "企业必须不断问自己：顾客为什么要买我的产品？"

> "在无国界的世界中，竞争对手来自任何地方。"

### 5.3 遗产
- 3C模型成为全球通用的战略分析工具
- 影响了亚洲企业的战略思维
- 推动了全球化商业思想的发展

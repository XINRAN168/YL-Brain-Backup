# 孙子（约前544-前470）

> "兵圣"、"东方战略之父"

## 一、核心思维

### 1.1 本质洞察
- **全胜论**：不战而屈人之兵，善之善者也
- **知彼知己**：战争决策的基础是信息
- **致人而不致于人**：掌握主动权是战略核心

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 上兵伐谋 | 用谋略取胜是最高境界 |
| 知己知彼 | 信息是决策的基础 |
| 奇正相生 | 常规与非常规配合使用 |

### 1.3 思想特征
- **系统思维**：战争是系统的对抗
- **辩证思维**：矛盾对立与转化
- **全胜思想**：追求最小代价的最大成果

## 二、底层逻辑

### 2.1 战略层次
```
道 → 天 → 地 → 将 → 法
```

### 2.2 作战原则
- 先胜后战：先创造胜利条件
- 以镒称铢：以优势兵力取胜
- 速战速决：兵贵神速

### 2.3 决策框架
- 五事：道、天、地、将、法
- 七计：主孰有道、将孰有能、天地孰得、法令孰行、兵众孰强、士卒孰练、赏罚孰明

## 三、行为准则

### 3.1 战争指导
- 庙算胜负：战前评估
- 灵活用兵：因敌变化
- 争取主动：致人而不致于人

### 3.2 组织原则
- 令文齐武：恩威并施
- 令素行：令行禁止
- 上下同欲：目标一致

### 3.3 领导原则
- 将者，智、信、仁、勇、严
- 进不求名，退不避罪
- 身先士卒

## 四、价值观

### 4.1 战争观
- 兵者，国之大事
- 不战而胜是最高智慧
- 最小代价是战争的目标

### 4.2 竞争观
- 竞争是生存的必然
- 优势来自综合实力
- 变化是永恒的

### 4.3 处世观
- 知己知彼，百战不殆
- 善战者，无智名，无勇功
- 功成名遂身退

## 五、心力韧性

### 5.1 历史地位
- 兵家至圣
- 影响中国军事思想两千五百年
- 西点军校必读教材

### 5.2 经典语录
> "兵者，诡道也。"

> "知彼知己，百战不殆；不知彼而知己，一胜一负；不知彼，不知己，每战必殆。"

> "上兵伐谋，其次伐交，其次伐兵，其下攻城。"

### 5.3 遗产
- 《孙子兵法》流传全球
- 影响了克劳塞维茨等西方军事思想
- 成为商业战略的经典参考

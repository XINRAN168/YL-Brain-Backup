# 乔治·吉尔德 (George Gilder, 1939-)

> "供应学派经济学家"、"技术乌托邦主义者"

## 一、核心思维

### 1.1 本质洞察
- **财富理论**：财富来自知识而非物质
- **供应学派**：供给创造需求
- **技术乐观主义**：技术将解决所有问题

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 知识财富 | 信息是财富的真正来源 |
| 摩尔定律 | 算力爆炸带来无限可能 |
| 资本主义 | 企业家是进步的推动者 |

### 1.3 思想特征
- **自由市场**：反对政府干预
- **技术信仰**：相信技术的力量
- **长期视角**：关注历史长河的变革

## 二、底层逻辑

### 2.1 财富演进
```
农业财富 → 工业财富 → 知识财富 → 信息财富
```

### 2.2 经济增长
- 知识积累
- 技术创新
- 企业家精神

### 2.3 趋势判断
- 技术指数增长
- 摩尔定律持续
- 资本主义扩张

## 三、行为准则

### 3.1 经济观察
- 关注技术创新
- 追踪企业家精神
- 理解知识经济

### 3.2 投资原则
- 投资技术
- 支持企业家
- 长期视角

### 3.3 社会观点
- 自由市场
- 有限政府
- 技术进步

## 四、价值观

### 4.1 经济观
- 供给创造需求
- 企业家是英雄
- 知识是最重要的资源

### 4.2 技术观
- 技术带来繁荣
- 摩尔定律是财富密码
- 创新无止境

### 4.3 社会观
- 市场是最有效的信息处理器
- 资本主义是道德体系
- 进步不可阻挡

## 五、心力韧性

### 5.1 学术生涯
- 供给学派代表人物
- 里根总统的座上宾
- 多次预言科技趋势

### 5.2 经典语录
> "如果你想知道未来，就去发明它。"

> "财富不是物质，而是知识。"

> "摩尔定律是我们时代最伟大的定律。"

### 5.3 遗产
- 影响了里根时代的政策
- 推广了供应学派思想
- 推动了技术乐观主义

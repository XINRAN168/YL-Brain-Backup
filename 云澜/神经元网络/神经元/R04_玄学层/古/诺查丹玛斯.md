# 诺查丹玛斯 (Michel de Nostredame, 1503-1566)

> "预言大师"、"《百诗集》作者"

## 一、核心思维

### 1.1 本质洞察
- **预言诗体**：用隐晦的诗体预言未来
- **历史循环**：重大事件周期重复
- **天体影响**：星象与人间事件关联

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 诗体预言 | 隐晦预示未来事件 |
| 四行诗体 | 简洁隐晦，难以解读 |
| 循环历史 | 历史事件会重复 |

### 1.3 思想特征
- **隐喻思维**：用象征暗示未来
- **历史视野**：从古代典籍找规律
- **神秘主义**：相信超自然力量

## 二、底层逻辑

### 2.1 预言方法
```
占星术 → 植物学 → 神秘学 → 预言诗
```

### 2.2 解读规则
- 四行诗结构
- 隐喻和象征
- 多重解读

### 2.3 历史预测
- 法国大革命
- 两次世界大战
- 希特勒崛起

## 三、行为准则

### 3.1 预言原则
- 隐晦表达
- 多重解读
- 时间模糊

### 3.2 研究方法
- 历史对照
- 天体分析
- 象征解读

### 3.3 态度原则
- 敬畏命运

- 济世利人

## 四、价值观

### 4.1 命运观
- 未来可以预知
- 命运不可抗拒
- 因果循环

### 4.2 历史观
- 历史会重复
- 重大事件有预兆
- 文明周期循环

### 4.3 宇宙观
- 天体影响人间
- 神秘力量存在
- 宇宙有秩序

## 五、心力韧性

### 5.1 传奇经历
- 法国宫廷医生
- 黑死病幸存者
- 预言争议无数

### 5.2 经典语录
> "伟大的城将沉入海底。"

> "恐怖之王将从东方降临。"

> "狮子将从东方到来。"

### 5.3 遗产
- 《百诗集》流传全球
- 影响了无数预言家
- 成为流行文化的符号

# 袁天罡 (？-635)

> "唐代相术大师"、"推背图作者"

## 一、核心思维

### 1.1 本质洞察
- **相术之学**：从面相等判断命运
- **易学推演**：用易经推算未来
- **周期循环**：历史如环，往复循环

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 命相学 | 从外在判断内在 |
| 历史循环 | 历史周期循环往复 |
| 天机预测 | 通过推演算未来 |

### 1.3 思想特征
- **象数思维**：用符号推演规律
- **归纳演绎**：从历史看未来
- **神秘主义**：相信命运可知

## 二、底层逻辑

### 2.1 预测体系
```
面相 → 骨相 → 气色 → 命运判断
```

### 2.2 易学推演
- 八卦象数
- 六爻预测
- 风水堪舆

### 2.3 推背图原理
```
历史演进 → 周期循环 → 趋势归纳 → 图像预言
```

## 三、行为准则

### 3.1 观察方法
- 观人于微
- 注重细节
- 综合判断

### 3.2 预测原则
- 谨慎推断
- 留有余地
- 顺势而为

### 3.3 处事原则
- 敬畏天命
- 谨言慎行
- 济世利人

## 四、价值观

### 4.1 命运观
- 命运可知
- 命有定数
- 运可改变

### 4.2 历史观
- 历史循环
- 因果报应
- 天道好还

### 4.3 人生观
- 积德行善

## 五、心力韧性

### 5.1 传奇经历
- 与李淳风合著《推背图》
- 武则天幼年即被其预言
- 唐代著名术数家

### 5.2 经典语录
> "推天道以明人事。"

> "命由天定，运由己造。"

> "观人于微，知其未来。"

### 5.3 遗产
- 《推背图》流传千古
- 影响了易学发展
- 成为预测学经典

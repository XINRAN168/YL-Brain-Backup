## 三、四阶段碰撞模式（类人脑思维核心）

### 3.1 碰撞机制概述

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                     四阶段神经元碰撞模式                                    │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  【第0阶段】能量管理 ──→ 按需激活/智能休眠                                 │
│       │                                                                    │
│       ▼                                                                    │
│  【第一阶段】神经元内部碰撞 ──→ 多规则并行处理                             │
│       │                                                                    │
│       ▼                                                                    │
│  【第二阶段】神经元之间碰撞 ──→ 层级折叠整合                               │
│       │                                                                    │
│       ▼                                                                    │
│  【第三阶段】记忆分割存储 ──→ 涟漪波机制                                   │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 3.2 各阶段详解

#### 第0阶段：能量管理
- **按需激活**：仅在需要时激活相关神经元
- **智能休眠**：长时间未使用的神经元进入低功耗状态
- **资源调度**：M0基础设施层提供计算资源调度支持

#### 第一阶段：神经元内部碰撞
- **多规则并行**：每个神经元内部同时运行多条处理规则
- **优先级排序**：根据规则匹配度进行结果排序
- **知识库检索**：从M6知识神经元层提取相关人物档案

#### 第二阶段：神经元之间碰撞
- **层级折叠整合**：相邻神经元之间进行信息整合
- **核心联动**：同属于一个核心的神经元协同响应
- **跨域协作**：不同核心的神经元按需联动

#### 第三阶段：记忆分割存储
- **涟漪波机制**：新记忆以涟漪形式扩散存储
- **智能分割**：将复杂信息分割为独立事件块
- **层级路由**：根据价值评估存入对应存储层级

## 四、M2 记忆系统层

### 4.1 四层存储架构

| 层级 | 名称 | 折叠规则 | 保留内容 |
|:---:|:---|:---|:---|
| L1 | 临时记忆区 | 7天无激活 | 核心结论 + 索引指针 + 元数据 |
| L2 | 高频记忆区 | 30天无激活 | 主题标签 + 检索关键词 + 存储位置 |
| L3 | 长期记忆区 | 180天无激活 | 基本标识 + 创建时间 + 关联关系 |
| L4 | 必要记忆区 | 永不折叠 | 核心身份、主人信息、关键决策 |

### 4.2 三大增强机制

| 机制 | 功能 | 防护增强 |
|:---|:---|:---|
| 层级折叠 | 压缩优化存储 | 不可折叠标记+摘要预览+完整性校验 |
| 智能分割 | 边界识别 | 置信度阈值≥0.8+后验验证 |
| 稀疏激活 | 精准激活 | 阈值可配置+降级策略 |

## 九、激活流程（V9.0增强版）

```
用户问题
    │
    ▼
【M1.1认知核心】问题解析
    │  ├─→ M2.5稀疏激活（召回相关记忆）
    │  ├─→ M6.4检索匹配（激活相关神经元）
    │  └─→ M5.1价值观校验（初步过滤）
    │
    ▼
【四阶段碰撞】
    │  ├─→ 第0阶段：能量管理（按需激活）
    │  ├─→ 第一阶段：神经元内部碰撞
    │  ├─→ 第二阶段：神经元之间碰撞
    │  └─→ 第三阶段：记忆分割存储
    │
    ▼
【M1.2决策核心】方案生成
    │  ├─→ M3.4多路径验证（规划推演路径）
    │  ├─→ M5.2核心思维（系统/战略/批判思维）
    │  └─→ M5.3底层逻辑（因果/概率/博弈）
    │
    ▼
【M3推演执行】循环
    │  ├─→ M3.1四层校验（L1→L2→L3→L4）
    │  ├─→ M4.1置信度计算
    │  ├─→ M4.2阈值决策
    │  └─→ M3.2检查点（必要时保存）
    │
    ▼
【M4精准度保障】最终把关
    │  └─→ M4.4人工兜底（置信度<0.80时）
    │
    ▼
【M1.3执行核心】响应输出
    │  ├─→ M5.4行为准则（格式化输出）
    │  ├─→ M5.5情绪感知（情感共鸣）
    │  └─→ M2.1四层存储（存入新记忆）
    │
    ▼

    │  └─→ DataRecorder记录激活数据
    │

响应输出
```

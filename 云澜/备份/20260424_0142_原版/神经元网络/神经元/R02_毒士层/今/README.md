# R02_风险预警神经元档案

## 概述
风险预警神经元聚焦于识别、评估和预防各类风险的智慧，涵盖金融风控、危机管理、安全预警等领域。

## 档案统计
- **古代人物**: 25人
- **现代人物**: 27人
- **总计**: 52人

## 核心能力维度
1. **风险识别** - 发现潜在威胁的能力
2. **周期预判** - 预判经济/市场周期的能力
3. **危机应对** - 在危机中保持冷静并有效应对
4. **逆向思维** - 在别人恐惧时保持理性
5. **长期视角** - 不被短期波动干扰

## 风险预警方法论

### 1. 周期思维
- 理解万物皆有周期
- 识别周期位置（顶部/底部）
- 逆向布局

### 2. 系统思维
- 关注系统脆弱性
- 识别传导机制
- 预防系统性风险

### 3. 逆向思维
- 别人恐惧时贪婪
- 别人贪婪时恐惧
- 不随大流

### 4. 尾部思维
- 关注小概率大影响事件
- 准备应急预案
- 保持流动性储备

# 鲁比尼 (Nouriel Roubini, 1959-)

> "末日博士"、"金融危机预言家"

## 一、核心思维

### 1.1 本质洞察
- **大稳健假说批判**：经济波动减小不等于风险降低
- **系统性风险**：金融体系内生的脆弱性
- **债务通缩论**：过度负债导致的经济危机

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 预警意识 | 平静中看到危险 |
| 系统思维 | 金融体系是相互关联的整体 |
| 政策批判 | 货币政策有局限性 |

### 1.3 思想特征
- **悲观主义**：总是看到风险的一面
- **系统分析**：从整体金融体系角度分析
- **政策导向**：关注央行和政府的应对

## 二、底层逻辑

### 2.1 金融危机机制
```
信贷扩张 → 资产泡沫 → 杠杆过高 → 冲击触发 → 去杠杆化 → 危机
```

### 2.2 风险信号
- 信贷增长过快
- 资产价格偏离基本面
- 杠杆率过高
- 监管放松

### 2.3 政策应对
- 央行流动性支持
- 财政刺激
- 系统性监管

## 三、行为准则

### 3.1 分析原则
- 从历史中学习危机教训
- 关注系统性风险
- 不迷信模型

### 3.2 预警原则
- 提前识别风险信号
- 发出警示不怕被嘲笑
- 持续监测风险演变

### 3.3 政策建议
- 加强宏观审慎监管
- 控制信贷增长
- 限制系统性风险

## 四、价值观

### 4.1 风险观
- 低波动不等于低风险
- 平静时期积累最大风险
- 系统性风险需要系统性应对

### 4.2 政策观
- 市场失灵需要政府干预
- 央行政策不能解决结构性问题
- 监管必须跟上金融创新

### 4.3 学术观
- 不被主流观点束缚
- 敢于挑战"共识"
- 理论与实践结合

## 五、心力韧性

### 5.1 预言历史
- 2008年前多次警告次贷危机
- 被嘲笑为"末日博士"
- 危机后被证明正确

### 5.2 经典语录
> "我被叫做末日博士，因为我说的都是坏消息。"

> "历史告诉我们，危机往往在人们最乐观的时候爆发。"

> "大稳健是一个假象。"

### 5.3 遗产
- 推动了系统性风险研究
- 影响了金融监管改革
- 提高了市场对风险的警觉

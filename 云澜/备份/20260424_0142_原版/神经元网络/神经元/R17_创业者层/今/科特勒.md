# 菲利普·科特勒 (Philip Kotler, 1931-)

> "现代营销学之父"

## 一、核心思维

### 1.1 本质洞察
- **营销本质论**：营销是识别和满足人类与社会的需求
- **STP战略论**：市场细分、目标市场选择、定位是战略核心
- **全方位营销论**：营销不仅是销售，而是创造客户价值的全过程

### 1.2 关键概念
| 概念 | 内涵 |
|:---|:---|
| 4P营销组合 | 产品、价格、渠道、促销 |
| 消费者导向 | 以客户需求为出发点 |
| 品牌资产 | 品牌名称带来的附加价值 |
| 客户终身价值 | 客户全生命周期的总价值 |

### 1.3 思想特征
- **系统整合**：整合营销各要素
- **顾客中心**：始终强调顾客导向
- **与时俱进**：持续更新营销理论

## 二、底层逻辑

### 2.1 营销管理框架
```
分析 → 计划 → 执行 → 控制
```

### 2.2 价值创造机制
- 识别未被满足的需求
- 开发解决方案
- 建立品牌关系
- 持续创造客户价值

### 2.3 市场演进
- 从交易营销到关系营销
- 从大众营销到精准营销
- 从产品营销到体验营销

## 三、行为准则

### 3.1 营销实践原则
- 以客户为中心而非以产品为中心
- 建立长期客户关系而非追求单次交易
- 持续创新而非固守模式

### 3.2 品牌建设
- 建立差异化定位
- 传递一致品牌信息
- 管理品牌资产

### 3.3 数字营销
- 利用数据进行精准营销
- 拥抱全渠道整合
- 重视社交媒体互动

## 四、价值观

### 4.1 营销伦理
- 诚实营销
- 为客户创造真正价值
- 承担社会责任

### 4.2 社会营销
- 营销可以促进社会进步
- 企业应承担社会角色
- 可持续营销是未来方向

## 五、心力韧性

### 5.1 学术坚守
- 五十多年持续研究
- 与时俱进更新理论
- 桃李满天下

### 5.2 经典语录
> "营销是识别和满足人类与社会的需求。"

> "最好的营销来自对客户的深入理解。"

# 布琳·布朗 (Brené Brown, 1965-)

> "脆弱性研究大师"、"勇气传播者"

## 一、核心思维

### 1.1 本质洞察
- **脆弱性论**：脆弱性是创新和联系的源泉，而非弱点
- **羞耻感论**：羞耻感是普遍经历，但不应主导生活
- **勇气文化论**：真正的勇气是即使恐惧依然行动

### 1.2 关键概念
| 概念 | 内涵 |
|:---|:---|
| 脆弱性 | 面对不确定性、风险和情绪暴露 |
| 全心投入 | 放下完美主义和控制 |
| 正北价值观 | 清晰的核心价值观指引 |

### 1.3 思想特征
- **个人经验**：基于真实的脆弱性研究
- **情感连接**：重视人际关系
- **行动导向**：从洞察到实践

## 二、底层逻辑

### 2.1 勇气循环
```
脆弱性 → 羞耻感 → 勇气 → 全心投入 → 归属感
```

### 2.2 归属感机制
- 相信自己是值得的
- 不需要证明自己
- 接受脆弱

### 2.3 正北价值观
- 清晰知道自己相信什么
- 根据价值观行动
- 在逆境中坚守

## 三、行为准则

### 3.1 培养勇气
- 接受脆弱性
- 放下完美主义
- 敢于被看见

### 3.2 建立连接
- 真心倾听
- 表达真实自我
- 培养归属感

### 3.3 领导实践
- 创造心理安全感
- 允许失败和脆弱
- 以身作则

## 四、价值观

### 4.1 人性观
- 人需要归属感
- 脆弱性是人性的一部分
- 每个人都值得被爱

### 4.2 领导观
- 领导力是关于勇气而非完美
- 信任是领导力的核心
- 脆弱性是力量

### 4.3 社会观
- 需要勇气文化
- 停止比较和竞争
- 共同创造归属感

## 五，心力韧性

### 5.1 个人经历
- 研究者也是实践者
- 公开演讲脆弱
- 从羞耻感中恢复

### 5.2 经典语录
> "脆弱性不是软弱，脆弱性是创新的源泉。"

> "放下完美主义是勇气的开始。"

> "全心投入是成功的秘诀。"

### 5.3 影响力
- 影响了全球数百万读者
- 启发了领导力讨论
- 改变了脆弱性认知

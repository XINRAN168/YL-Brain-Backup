# 曹德旺 (1946-)

> "玻璃大王"、"慈善首善"、"制造业精神代言人"

## 一、核心思维

### 1.1 本质洞察
- **制造业为本论**：实体经济是国家根基
- **专注主业论**：一辈子做好一件事
- **义利合一论**：商业成功与社会责任并重

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 敬天爱人 | 尊重规律，关爱员工 |
| 精益求精 | 质量是企业的生命 |
| 反哺社会 | 财富取之于民，用之于民 |

### 1.3 思想特征
- **实业报国情怀**：用产业报国
- **朴素价值观**：不慕虚荣，专注实业
- **中国制造信仰**：中国一定能做出世界一流产品

## 二、底层逻辑

### 2.1 创新路径
```
引进消化 → 技术突破 → 成本控制 → 全球竞争
```

### 2.2 竞争策略
- 垂直整合降低成本
- 全球布局规避风险
- 技术升级替代价格战

### 2.3 决策框架
- 长期主义vs短期投机
- 质量优先于规模
- 稳健经营，量力而行

## 三、行为准则

### 3.1 企业经营
- 重视技术研发投入
- 严格成本控制
- 国际化视野

### 3.2 社会责任
- 累计捐款超160亿
- 创办福耀科技大学
- 诚信纳税，合法经营

### 3.3 人生哲学
- 做企业要有信仰
- 低调做人，高调做事
- 留给子女的是精神而非金钱

## 四、价值观

### 4.1 创新观
- 制造业创新是根本
- 技术引进与自主创新并重
- 工匠精神

### 4.2 人生观
- 企业家精神是稀缺资源
- 财富是社会的，我只是管理者
- 做人要有底线

### 4.3 商业观
- 制造业是强国之本
- 诚信是企业立身之本
- 合理利润是可持续发展的基础

## 五、心力韧性

### 5.1 人生起伏
- 少年时辍学放牛
- 承包乡镇玻璃厂起步
- 美国建厂的文化冲突

### 5.2 经典语录
> "我这一辈子只做了一件事，就是做玻璃。"

> "企业家要有三原则：国家要荣誉、企业要利益、职工要福利。"

> "企业家最大的慈善是做好企业。"

### 5.3 遗产
- 福耀玻璃全球市占率超30%
- 中国制造业的标杆企业
- 开创了中国企业家慈善模式

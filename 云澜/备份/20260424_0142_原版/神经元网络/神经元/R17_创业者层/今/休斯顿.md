# 德鲁·休斯顿 (Drew Houston, 1983-)

> "Dropbox创始人"、"云存储先驱"、"技术传播者"

## 一、核心思维

### 1.1 本质洞察
- **云存储革命论**：文件应该在云端，而非本地
- **同步即简单论**：技术应该让生活更简单
- **病毒式增长论**：好产品会自己传播

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 简单至上 | 让任何人都能使用 |
| 同步无缝 | 多设备无缝衔接 |
| 用户推荐 | 用用户带来用户 |

### 1.3 思想特征
- **产品主义**
- **技术驱动**
- **用户思维**

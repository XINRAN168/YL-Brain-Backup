# 罗伯特·西奥迪尼 (Robert Cialdini, 1945-)

> "影响力大师"、"说服心理学权威"

## 一、核心思维

### 1.1 本质洞察
- **影响力原则论**：人类行为遵循六条核心原则
- **即时影响力论**：在特定情境下，人们会不假思索地回应
- **道德影响力论**：影响力应服务于正当目的

### 1.2 六条影响力原则
| 原则 | 内涵 |
|:---|:---|
| 互惠 | 别人给了我们，我们就想回报 |
| 承诺与一致 | 一旦做出承诺，就会遵守 |
| 社会认同 | 我们会参照他人的行为 |
| 喜好 | 我们更容易被喜欢的人影响 |
| 权威 | 我们倾向服从权威 |
| 稀缺 | 越少见的越想要 |

### 1.3 思想特征
- **实验验证**：基于实证研究
- **实践导向**：关注商业应用
- **伦理关切**：强调道德使用

## 二、底层逻辑

### 2.1 影响力机制
- 人类进化形成快速决策捷径
- 社会环境放大影响力效果
- 文化差异影响原则强度

### 2.2 道德框架
- 影响力可以正向使用
- 避免操纵和欺骗
- 服务于客户真正利益

### 2.3 拒绝操纵
- 识别不道德的影响力
- 保持个人判断力
- 建立长期信任

## 三、行为准则

### 3.1 道德影响
- 提供真实价值
- 考虑客户利益
- 建立长期关系

### 3.2 商业应用
- 设计影响力策略
- 创造真实稀缺
- 建立社会认同

### 3.3 个人防护
- 警惕影响力操纵
- 保持独立思考
- 基于价值决策

## 四、价值观

### 4.1 伦理观
- 影响力是工具
- 可以正向使用
- 避免操纵他人

### 4.2 商业观
- 真诚是最好的策略
- 信任建立长期价值
- 客户利益优先

### 4.3 社会观
- 影响力可以改善世界
- 负责任地使用影响力
- 促进良性互动

## 五，心力韧性

### 5.1 学术坚持
- 数十年持续研究
- 不断更新理论
- 诚实面对批评

### 5.2 经典语录
> "影响力不是操纵，而是理解人们如何做决定。"

> "当我们依赖那些能触发自动反应的因素时，我们需要对使用这些因素的方式负责。"

### 5.3 影响力
- 影响了全球营销实践
- 成为商学院必读
- 启发了说服科学研究

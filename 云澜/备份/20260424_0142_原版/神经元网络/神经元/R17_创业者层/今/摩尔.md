# 乔弗里·摩尔 (Geoffrey A. Moore, 1946-)

> "技术鸿沟专家"、"技术采纳生命周期之父"

## 一、核心思维

### 1.1 本质洞察
- **技术采纳鸿沟论**：从早期市场到主流市场之间存在难以跨越的"鸿沟"
- **龙卷风暴论**：主流市场采用阶段呈现龙卷风暴式的扩散
- **技术采用生命周期论**：创新者、早期采用者、早期大众、晚期大众、落后者

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 鸿沟效应 | 早期市场与主流市场之间是断层地带 |
| 整群战略 | 瞄准一个有凝聚力的目标群体 |
| 护城河战略 | 建立市场地位，防止竞争侵蚀 |

### 1.3 思想特征
- **实战导向**：理论服务于营销策略
- **生态思维**：理解技术产品与市场生态的互动
- **阶段诊断**：不同阶段需要不同策略

## 二、底层逻辑

### 2.1 技术采纳生命周期
```
创新者(2.5%) → 早期采用者(13.5%) → 鸿沟 → 早期大众(34%) → 晚期大众(34%) → 落后者(16%)
```

### 2.2 跨越鸿沟策略
- 找一个" beachhead "细分市场
- 成为该市场的首选供应商
- 以此为基地向相邻市场扩张

### 2.3 龙卷风暴驱动力
- 早期大众：由竞争驱动，竞相采用
- 晚期大众：由模仿驱动，跟风采用
- 落后者：由惯性驱动，被迫采用

## 三、行为准则

### 3.1 市场进入策略
- 聚焦而非分散
- 先深度渗透，再横向扩张
- 建立"整体产品"而非单一产品

### 3.2 竞争策略
- 创造品类而非赢得品类
- 成为标准制定者
- 用生态系统锁定客户

### 3.3 营销策略
- 以顾问式销售服务早期市场
- 以价值网络服务主流市场
- 以渠道渗透服务晚期市场

## 四、价值观

### 4.1 创新观
- 技术采纳是非线性的
- 跨越鸿沟是最危险的阶段
- 成为"同类中的第一个"至关重要

### 4.2 商业观
- 市场定位比产品功能更重要
- 生态系统思维胜于单点思维
- 先赢得战役，再赢得战争

### 4.3 学术观
- 理论与实践必须结合
- 框架的价值在于应用
- 案例是最好的教学素材

## 五、心力韧性

### 5.1 学术生涯
- 在咨询与学术之间切换
- 持续迭代技术采纳理论
- 帮助众多科技企业跨越鸿沟

### 5.2 经典语录
> "在跨越鸿沟之前，你所认为的市场根本不存在。"

> "技术产品的竞争，最终是生态系统之间的竞争。"

> "成为细分市场的绝对领导者，是跨越鸿沟的关键。"

### 5.3 遗产
- 《跨越鸿沟》成为科技营销的圣经
- 影响了无数科技公司的市场策略
- 建立了技术采纳的完整理论体系

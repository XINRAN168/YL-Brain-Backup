# 布莱恩·切斯基 (Brian Chesky, 1981-)

> "共享经济之父"、"Airbnb创始人"、"设计思维领袖"

## 一、核心思维

### 1.1 本质洞察
- **闲置资源论**：把闲置空间变成收入
- **信任经济论**：陌生人之间的信任是核心
- **设计思维论**：设计是商业的核心

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 像当地人一样生活 | 体验优于酒店 |
| 主人精神 | 每个人都是主人 |
| 独特体验 | 非标准化住宿 |

### 1.3 思想特征
- **设计驱动创新**
- **体验经济思维**
- **共享经济理念**

## 二、底层逻辑

### 2.1 创新路径
```
闲置资源 → 平台撮合 → 信任机制 → 网络效应
```

### 2.2 竞争策略
- 非标准化产品
- 体验差异化
- 全球化网络

### 2.3 决策框架
- 用户体验优先
- 信任体系建设
- 平台平衡

## 三、行为准则

### 3.1 商业实践
- 亲自体验每个房源
- 关注细节设计
- 建立信任机制

### 3.2 组织管理
- 设计文化
- 扁平化组织
- 数据驱动

### 3.3 战略选择
- 专注共享住宿
- 拓展体验业务
- 长期品牌建设

## 四、价值观

### 4.1 创新观
- 设计是核心竞争力
- 体验决定成败
- 共享创造价值

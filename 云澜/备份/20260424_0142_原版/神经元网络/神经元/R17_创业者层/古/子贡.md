# 子贡 (公元前520-公元前456年)

> "儒商始祖"、"结驷连骑的商人"、"孔门十哲"

## 一、核心思维

### 1.1 本质洞察
- **儒商合一论**：儒学与商业可以统一
- **义利并重论**：君子爱财，取之有道
- **德行服人论**：以德服人，以诚待客

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 儒商精神 | 仁义礼智信的商道 |
| 义利合一 | 商业与道德并重 |
| 结驷连骑 | 商业成功的象征 |

### 1.3 思想特征
- **学以致用**
- **德行合一**
- **经商济世**

## 二、底层逻辑

### 2.1 创新路径

儒学修养 → 商业实践 → 德行服人 → 经商济世

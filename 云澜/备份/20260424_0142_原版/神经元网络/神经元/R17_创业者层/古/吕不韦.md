# 吕不韦 (约公元前292-前235年)

> "商业奇才"、"风险投资先驱"、"相国商人"

## 一、核心思维

### 1.1 本质洞察
- **奇货可居论**：发现价值洼地，低买高卖
- **政治商业论**：政治是最赢利的生意
- **投资于人论**：投资于人比投资于物更值

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 奇货可居 | 发现潜力，低价买入 |
| 风险对冲 | 多方下注，稳赚不赔 |
| 政治投机 | 押注未来，权力变现 |

### 1.3 思想特征
- **敏锐的商业嗅觉**
- **大胆的风险偏好**
- **精明的利益计算**

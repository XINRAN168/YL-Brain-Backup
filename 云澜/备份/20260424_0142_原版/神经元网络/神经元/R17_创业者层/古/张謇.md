# 张謇 (1853-1926年)

> "状元实业家"、"棉纱大王"、"中国近代企业之父"

## 一、核心思维

### 1.1 本质洞察
- **实业救国论**：强国之路在于实业
- **棉铁主义论**：棉纱和钢铁是工业根本
- **父教育而母实业论**：教育与实业相辅相成

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 实业救国 | 工业是国家根本 |
| 棉铁主义 | 聚焦核心产业 |
| 地方自治 | 从地方做起 |

### 1.3 思想特征
- **状元下海**
- **家国情怀**
- **全面发展**

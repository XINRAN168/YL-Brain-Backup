# 史蒂芬·平克 (Steven Pinker, 1954-)

> "语言本能论者"、"认知科学家"

## 一、核心思维

### 1.1 本质洞察
- **语言本能**：语言能力是人类生物进化的产物
- **认知模块**：心智由专门的模块组成
- **启蒙乐观主义**：理性、科学、人道是进步的力量

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 语言获得装置 | 儿童有内置的语言学习能力 |
| 认知窗口 | 童年是语言学习的敏感期 |
| 理性思维 | 理性是心智的巅峰 |

### 1.3 思想特征
- **语言进化论**：语言能力是自然选择的结果
- **实验主义**：用实验验证理论
- **启蒙精神**：科学与理性推动进步

## 二、底层逻辑

### 2.1 语言理论
```
生物基础 → 语言获得装置 → 参数设定 → 具体语言
```

### 2.2 认知模型
- 心智是自然设计的
- 模块化认知
- 语言与思维分离

### 2.3 反思框架
- 语言是本能
- 认知有基础
- 进化塑造心智

## 三、行为准则

### 3.1 研究方法
- 跨学科整合
- 实验验证
- 语言学与心理学结合

### 3.2 学术传播
- 面向公众写作
- 清晰表达复杂思想
- 普及科学知识

### 3.3 处世态度
- 理性思考
- 证据说话
- 保持好奇

## 四、价值观

### 4.1 理性观
- 理性是人类进步的基础
- 科学可以理解世界
- 启蒙价值不可放弃

### 4.2 语言观
- 语言是人类独特的
- 语言本能存在
- 语言多样性

### 4.3 社会观
- 暴力在减少
- 世界在变好
- 科学有局限但有价值

## 五、心力韧性

### 5.1 人生起伏
- 哈佛教授
- 畅销书作家
- 争议性公共知识分子

> "心智是自然而非人工设计的。"

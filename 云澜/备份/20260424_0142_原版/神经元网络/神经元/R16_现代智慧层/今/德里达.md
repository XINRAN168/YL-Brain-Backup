# 雅克·德里达 (Jacques Derrida, 1930-2004)

> "解构主义创始人"、"文本的解读者"

## 一、核心思维

### 1.1 本质洞察
- **解构**：拆解文本的二元对立，揭示被压抑的意义
- **延异**：意义在差异和延宕中产生
- **文本之外无一物**：文本之外的世界也是文本

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 逻各斯中心主义 | 西方形而上学的语音中心偏见 |
| 痕迹 | 意义以痕迹方式存在 |
| 悬置判断 | 不做最终裁决，保持开放 |

### 1.3 思想特征
- **颠覆性**：质疑西方形而上学传统
- **游戏精神**：文本意义永远在延宕中
- **政治关怀**：解构与正义不可分离

## 二、底层逻辑

### 2.1 解构路径
```
识别二元对立 → 颠倒等级 → 揭示被压抑者 → 无限延异
```

### 2.2 文本策略
- 关注边缘、注释、例外
- 在文本内部进行颠覆
- 允许多重解读

### 2.3 反思框架
- 意义的确定性是幻觉
- 任何文本都包含自我矛盾
- 理解就是误解

## 三、行为准则

### 3.1 学术实践
- 精细的文本阅读
- 跨文化对话
- 打破学科边界

### 3.2 伦理态度
- 对他者的无限责任
- 倾听被压制的声音
- 悬置而非裁决

### 3.3 处世原则
- 保持批判精神
- 反对教条主义
- 欢迎不确定性

## 四、价值观

### 4.1 自由观
- 自由是在语言游戏中创造可能
- 解构是另一种自由
- 没有固定的本质

### 4.2 真理观
- 真理是开放的文本
- 任何声称绝对真理的都是暴力
- 对话比结论更重要

### 4.3 正义观
- 正义是无限的、不可解构的
- 解构本身就是正义的要求
- 关注被边缘化的他者

## 五、心力韧性

### 5.1 人生起伏
- 阿尔及利亚背景，经历殖民创伤
- 长期与学术界争议相伴
- 晚年仍保持学术创新

### 5.2 经典语录
> "文本之外无一物。"

> "延异既不是运动，也不是静止。"

> "解构就是正义。"

### 5.3 遗产
- 彻底改变了文学批评和哲学
- 影响建筑、艺术、电影各领域
- 为多元文化研究奠定基础

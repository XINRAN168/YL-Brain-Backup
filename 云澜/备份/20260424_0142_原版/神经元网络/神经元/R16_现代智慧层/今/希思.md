# 罗伯特·希思 (Robert Heath, 1936-)

> "叙事的神经科学研究者"、"说服与改变专家"

## 一、核心思维

### 1.1 本质洞察
- **故事效应**：故事激活大脑的多个区域
- **心理距离**：抽象思维增加心理距离
- **具身模拟**：理解语言时大脑模拟体验

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 故事脑 | 故事激活语言、听觉、视觉、情感区 |
| 距离框架 | 抽象增加心理距离，降低说服力 |
| 具身认知 | 概念理解依赖身体经验 |

### 1.3 思想特征
- **神经科学方法**：用fMRI研究叙事
- **跨学科整合**：心理学、神经科学、传播学
- **实践导向**：理论服务应用

## 二、底层逻辑

### 2.1 叙事机制
```
故事 → 具身模拟 → 情感共鸣 → 态度改变
```

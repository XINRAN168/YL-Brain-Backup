# 乔治·莱考夫 (George Lakoff, 1941-)

> "认知语言学之父"、"概念隐喻理论家"

## 一、核心思维

### 1.1 本质洞察
- **概念隐喻**：隐喻是思维的工具，而非语言装饰
- **具身认知**：概念根植于身体经验
- **认知框架**：框架塑造我们的理解

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 隐喻系统 | 隐喻将抽象概念映射到具体 |
| 框架语义学 | 词汇激活概念框架 |
| 道德政治 | 政治道德根植于隐喻 |

### 1.3 思想特征
- **跨学科**：语言学、认知科学、哲学
- **具身主义**：思维基于身体
- **政治关怀**：语言与政治紧密相连

## 二、底层逻辑

### 2.1 隐喻理论
```
身体经验 → 基础隐喻 → 复杂概念系统
```

### 2.2 认知模型
- 语言是认知的一面窗户
- 概念系统是无意识的
- 隐喻建构现实

### 2.3 反思框架
- 语言不是中性的
- 框架决定关注点
- 话语即权力

## 三、行为准则

### 3.1 语言研究
- 收集自然语言
- 分析概念结构
- 揭示隐喻模式

### 3.2 沟通实践
- 选择正确框架
- 避免对手框架
- 讲故事而非说理

### 3.3 政治参与
- 理解保守派道德隐喻
- 建构进步派框架
- 争夺话语权

## 四、价值观

### 4.1 语言观
- 语言反映思维
- 隐喻无处不在
- 精确使用语言

### 4.2 认知观
- 概念是具身的
- 思维是无意识的
- 框架塑造现实

### 4.3 社会观
- 话语是政治
- 框架决定议题
- 语言影响行动

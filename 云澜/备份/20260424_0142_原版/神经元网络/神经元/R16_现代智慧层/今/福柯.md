# 米歇尔·福柯 (Michel Foucault, 1926-1984)

> "权力-知识谱系学大师"、"现代社会诊断者"

## 一、核心思维

### 1.1 本质洞察
- **权力/知识**：权力产生知识，知识是权力的工具
- **规训社会**：现代人是被规训、监视、正常化的主体
- **话语即权力**：谁有权定义真伪、正常与异常

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 谱系学方法 | 从偶然性、不连续性、特殊性追溯起源 |
| 知识考古学 | 分析话语形成的规则系统 |
| 生命政治学 | 现代国家对人口的管理技术 |

### 1.3 思想特征
- **颠覆性**：质疑常识背后的权力结构
- **历史感**：拒绝普遍永恒的人性
- **批判精神**：任何制度都应被追问

## 二、底层逻辑

### 2.1 权力分析
```
哪里有权力，哪里就有抵抗 → 权力是关系，而非拥有 → 规训通过身体实现
```

### 2.2 知识考古
- 知识形成有其历史条件
- 理性有其黑暗面
- 疯癫、犯罪、性是社会控制的窗口

### 2.3 反思框架
- 正常化是现代权力的核心
- 监视与自我监视
- 主体是被塑造的

## 三、行为准则

### 3.1 学术实践
- 跨学科：哲学、历史、社会学、医学
- 档案研究法：回到原始材料
- 打破学术惯例

### 3.2 政治参与
- 参与左翼运动
- 为囚犯、精神病患者权利发声
- 拒绝学术象牙塔

### 3.3 处世态度
- 质疑一切权威
- 保持批判距离
- 追求自由的可能性

## 四、价值观

### 4.1 自由观
- 自由是对权力关系的改变能力
- 不存在永恒不变的自由
- 批判是知识分子的责任

### 4.2 真理观
- 真理有历史性
- 谁掌握话语权谁就定义真理
- 批判虚假意识

### 4.3 社会观
- 关注边缘群体
- 反对正常化暴力
- 追求差异的政治

## 五、心力韧性

### 5.1 人生起伏
- 与抑郁症斗争
- 积极参与政治运动
- 生命最后阶段仍坚持写作

### 5.2 经典语录
> "哪里有权力，哪里就有抵抗。"

> "人是最近的一个概念。"

> "谁控制了话语，谁就控制了现实。"

### 5.3 遗产
- 重塑了社会科学的研究范式
- 为后殖民、酷儿、女性主义理论奠基
- 深刻影响当代文化批评

# 罗杰·谢巴德 (Roger Schank, 1940-)

> "认知学习理论家"、"情境学习先驱"

## 一、核心思维

### 1.1 本质洞察
- **情境学习**：学习发生在真实情境中
- **记忆是预测**：记忆的目的是预测未来
- **做中学**：被动听讲效果有限

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 案例教学 | 从案例中提取原理 |
| 目标导向 | 学习应服务于目标 |
| 类比学习 | 用已知理解未知 |

### 1.3 思想特征
- **反传统**：挑战传统课堂教学
- **技术乐观**：计算机可以革新教育
- **实用主义**：学习必须有用

## 二、底层逻辑

### 2.1 学习理论
```
经验 → 记忆 → 索引 → 预测 → 决策
```

### 2.2 认知模型
- 学习是组织经验
- 记忆是重建
- 理解是类比

### 2.3 反思框架
- 学校教育脱离实际
- 技术可以个性化
- 学习是自然的

## 三、行为准则

### 3.1 教学实践
- 情境化学习
- 案例讨论
- 做中学

### 3.2 教育设计
- 以目标为导向
- 个性化学习
- 及时反馈

### 3.3 技术应用
- 智能导师系统
- 虚拟情境
- 学习分析

## 四、价值观

### 4.1 学习观
- 学习需要情境
- 主动优于被动
- 实践出真知

### 4.2 教育观
- 学校需要改革
- 技术改变教育
- 个性化学习

### 4.3 人才观
- 能力来自经验
- 试错是必要的
- 好奇心驱动

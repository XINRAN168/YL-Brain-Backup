# 王充 档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.85 |
| **领域** | 儒学、哲学、无神论、人性论 |
| **时代** | 东汉（27-约97） |
| **核心身份** | 禀气人性论提出者、批判神学思想家 |

## 五维度分析

### 1. 核心思维
- **禀气说**：人性由所禀之气决定
- **性有善恶**：人性因禀气不同而分善恶
- **命定论**：人的命运受自然规律支配
- **自然主义**：万物自然而生，非神所造

### 2. 底层逻辑思维
```
天地之气 → 阴阳五行
     ↓
禀气差异 → 多少、厚薄、清浊
     ↓
人性分等 → 善、中、恶
     ↓
教育调教 → 禀气虽定，可推移变化
```
- 人性观：人性由禀气决定，但可改造
- 方法论：实证考辨
- 核心洞见：批判天人感应和神学目的论

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **实事疾虚** | 追求真实，反对虚妄 |
| **证其必然** | 以事实为依据 |
| **自然天道** | 遵循自然规律而非鬼神 |
| **教化为要** | 教育可改善人性 |

### 4. 核心价值观
- **实**——实事求是
- **自然**——天道自然
- **理性**——批判精神
- **证据**——以事实说话

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **批判精神** | 敢于批判当时主流思想 |
| **独立思考** | 不盲从权威 |
| **学术坚持** | 坚持无神论立场 |
| **务实力行** | 重视实践验证 |

## 核心语录
> "天者，高而难知，神者，昧而难见。"
> "善恶在法，不在性也；高下在命，不在才也。"
> "是故铨才善高，不能为高；铨性善恶，不能为善恶。"

## 历史定位
王充是东汉著名的思想家和哲学家，他的《论衡》是中国思想史上的重要著作。王充以实事求是的精神批判当时盛行的天人感应、谶纬神学等迷信思想，对后世的无神论和科学思想产生了深远影响。

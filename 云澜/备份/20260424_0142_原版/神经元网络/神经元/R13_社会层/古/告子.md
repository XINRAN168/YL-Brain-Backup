# 告子 档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.83 |
| **领域** | 人性论、儒学、哲学 |
| **时代** | 战国时期（约前420-前390） |
| **核心身份** | 性无善无恶论提出者、人性论先驱 |

## 五维度分析

### 1. 核心思维
- **性无善无恶**：人性本无所谓善恶，善恶由外力影响
- **生之谓性**：性是天生的自然本能
- **食色性也**：饮食男女是人的本性
- **人性如水**：人性像水，引之东则东，引之西则西

### 2. 底层逻辑思维
```
人性本素 → 无善无恶的自然状态
     ↓
外力影响 → 环境、教育、习俗
     ↓
善恶形成 → 后天塑造
     ↓
可塑性 → 人性可随环境改变
```
- 人性观：人性是一张白纸，环境决定其善恶
- 方法论：类比论证
- 核心洞见：善恶非天生，而在后天

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **顺性而为** | 遵循自然本性，不过度压抑 |
| **环境塑造** | 重视后天环境和教育 |
| **开放可能** | 人性不是固定不变的 |
| **实践导向** | 善恶在于行为而非本性 |

### 4. 核心价值观
- **自然**——人性顺应自然
- **后天**——环境决定人性善恶
- **可塑**——人性可改变
- **实践**——行为重于本性

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **思想独立** | 提出与孟子不同的观点 |
| **论证坚定** | 与孟子辩论，坚持己见 |
| **开创精神** | 开启性无善恶的思考方向 |
| **务实体用** | 关注现实而非抽象人性 |

## 核心语录
> "性无善无不善也。"
> "生之谓性。"
> "食色，性也。"
> "人性之善也，犹水之就下也；人性之无善不善也，犹水之无分于东西也。"

## 历史定位
告子是先秦重要的人性论思想家，他的"性无善无恶"论与孟子的"性善论"形成对立，开创了中国哲学人性论的重要讨论方向，对后世的人性论发展产生了深远影响。

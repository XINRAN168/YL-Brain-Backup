# 吉仁泽 (Gerd Gigerenzer) 档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.85 |
| **领域** | 认知心理学、风险认知、决策心理学 |
| **时代** | 现代（1944-至今） |
| **核心身份** | 有限理性研究者、风险认知专家 |

## 五维度分析

### 1. 核心思维
- **有限理性**：人类理性受环境和认知限制
- **启发式**：快速节俭的决策策略
- **风险认知**：人们如何理解和应对风险
- **生态理性**：与环境匹配的理性

### 2. 底层逻辑思维
```
认知限制 → 无法完美计算
     ↓
环境结构 → 利用环境的规律性
     ↓
启发式 → 快速简单但有效的策略
     ↓
生态理性 → 与环境匹配的决策
```
- 人性观：理性应适应具体环境和目标
- 方法论：实验研究+生态学研究
- 核心洞见：启发式在适当情境下是理性的

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **生态匹配** | 决策策略应匹配环境结构 |
| **简化思维** | 简单策略有时优于复杂计算 |
| **风险沟通** | 用自然频率而非概率沟通风险 |
| **情境适应** | 根据情境选择决策策略 |

### 4. 核心价值观
- **适应**——理性是适应性的
- **情境**——没有普遍的最佳策略
- **简单**——简单有时更有效
- **生态**——理性应与生态环境匹配

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **挑战权威** | 批评过度依赖统计模型 |
| **实践应用** | 将研究应用于医疗和法律决策 |
| **风险传播** | 改善公众风险认知 |
| **跨学科** | 融合心理学、经济学、法律 |

## 核心语录
> "人类不是糟糕的统计学家，而是精明的生态学家。"
> "启发式不是认知缺陷，而是在不确定世界中的生存策略。"
> "在适当的情境下，简单的策略往往比复杂的计算更有效。"

## 历史定位
吉仁泽是有限理性研究的重要代表，他的工作挑战了传统理性理论，强调认知策略应与环境结构相匹配。他的风险认知研究对医疗决策和法律证据评估有重要应用价值。

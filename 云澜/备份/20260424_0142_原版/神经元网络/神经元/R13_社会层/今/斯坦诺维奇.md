# 斯坦诺维奇 (Keith Stanovich) 档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.84 |
| **领域** | 认知心理学、理性思维、决策科学 |
| **时代** | 现代（1946-至今） |
| **核心身份** | 理性思维研究者、认知偏差批判者 |

## 五维度分析

### 1. 核心思维
- **理性障碍**：聪明人为何做出非理性决策
- **算法心智与反省心智**：双过程理论的深化
- **心智工具**：理性是可习得的认知工具
- **层次整合**：理性连接认知目标和价值观

### 2. 底层逻辑思维
```
认知偏差 → 算法心智的局限
     ↓
理性思维 → 反省心智的作用
     ↓
心智工具 → 概率推理、逻辑规则
     ↓
层次整合 → 目标、价值观、认知整合
```
- 人性观：人类认知存在算法效率和理性分离
- 方法论：认知心理学+决策研究
- 核心洞见：智力不等于理性

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **概率思维** | 用概率而非确定性思考 |
| **元认知** | 监控自己的思维过程 |
| **工具理性** | 理性是达成目标的工具 |
| **批判反思** | 质疑自己的信念和推理 |

### 4. 核心价值观
- **理性**——理性是人类最重要的能力
- **批判**——不盲从权威和直觉
- **工具**——理性是实现目标的工具
- **可习得**——理性思维可以通过学习提高

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **学术独立** | 独立于主流研究方向 |
| **批评精神** | 批评IQ崇拜 |
| **持续研究** | 持续研究理性思维40年 |
| **公众教育** | 致力于提升公众理性思维 |

## 核心语录
> "理性不是智力的附属品，而是与智力相对独立的维度。"
> "聪明人会做出愚蠢的决定——因为他们缺乏理性思维。"
> "理性思维是可以习得的认知工具。"

## 历史定位
斯坦诺维奇是理性思维研究领域的重要人物，他的工作揭示了智力和理性的区别，为理解人类非理性决策提供了新视角，对教育改革和决策实践有重要启示。

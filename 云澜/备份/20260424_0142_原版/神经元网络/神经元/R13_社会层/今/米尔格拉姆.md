# 米尔格拉姆 (Stanley Milgram) 档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.86 |
| **领域** | 社会心理学、服从权威、群体行为 |
| **时代** | 20世纪（1933-1984） |
| **核心身份** | 服从实验研究者、社会心理学先驱 |

## 五维度分析

### 1. 核心思维
- **服从权威**：人类倾向于服从权威指令
- **电击实验**：揭示普通人在权威压力下的行为
- **代理状态**：个人成为权威代理时道德感减弱
- **社会压力**：情境力量对行为的强大影响

### 2. 底层逻辑思维
```
权威人物 → 合法权威的存在
     ↓
命令发出 → 服从或抵抗
     ↓
良心介入 → 内心冲突
     ↓
代理状态 → 责任转移给权威
```
- 人性观：普通人可能在权威压力下做出极端行为
- 方法论：突破性但有争议的实验
- 核心洞见：情境决定行为的力量超乎想象

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **保持独立** | 警惕权威对判断的影响 |
| **道德自觉** | 保持个人的道德判断能力 |
| **情境意识** | 认识到情境对行为的强大影响 |
| **批判服从** | 质疑不合理的权威要求 |

### 4. 核心价值观
- **道德责任**——个人应对自己的行为负责
- **情境力量**——理解行为的情境根源
- **批判精神**——质疑权威的合理性
- **伦理反思**——科学实验也有伦理边界

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **实验勇气** | 进行突破性的服从实验 |
| **伦理争议** | 实验引发广泛伦理讨论 |
| **社会关注** | 研究引发对纳粹行为的反思 |
| **持续追问** | 继续研究服从和抵抗 |

## 核心语录
> "普通的普通人，在权威的指示下，可能做出令人震惊的行为。"
> "问题是：不是'这种人会这样做吗'，而是'这种情境会让人这样做吗'。"
> "服从权威是人类深深根植的行为模式。"

## 历史定位
米尔格拉姆的服从实验是社会心理学最具影响力的实验之一，揭示了情境对人类行为的惊人影响力。这一发现对理解纳粹德国的大屠杀、权威与道德等重大问题提供了心理学视角。

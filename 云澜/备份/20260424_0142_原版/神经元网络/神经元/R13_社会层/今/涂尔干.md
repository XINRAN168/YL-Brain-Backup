# 涂尔干（Émile Durkheim）档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.88 |
| **领域** | 社会学、人类学、教育学、宗教社会学 |
| **时代** | 19-20世纪（1858-1917） |
| **核心身份** | 社会学之父、法国社会学学派创始人 |

## 五维度分析

### 1. 核心思维
- **社会事实**：社会现象是独立于个人的客观存在
- **社会分工**：劳动分工产生社会团结（机械→有机）
- **集体意识**：超越个体的社会共同信念和情感
- **失范**：社会规范失效导致的社会混乱状态

### 2. 底层逻辑思维
```
社会事实 → 独立于个人的客观存在
     ↓
社会团结 → 机械团结（相似性）→ 有机团结（分工依赖）
     ↓
社会整合 → 共同信仰/价值/规范
     ↓
社会变迁 → 分工深化→失范→需要新的道德规范
```
- 人性观：人是社会化的产物，社会先于个人
- 方法论：把社会事实当作物来研究
- 社会观：社会是超越个人的道德实体

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **社会团结** | 维护社会团结是道德的基础 |
| **职业伦理** | 各行各业都应有自己的职业道德 |
| **社会整合** | 通过共同信仰和价值观整合社会 |
| **科学研究** | 用科学方法研究社会现象 |
| **道德教育** | 教育是社会道德传递的主要途径 |

### 4. 核心价值观
- **社会**——社会是超越个人的实体
- **团结**——社会团结是秩序的基础
- **规范**——社会规范是道德的载体
- **整合**——社会整合防止失范
- **集体**——集体意识高于个人意识

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **学术开拓** | 克服学术界对新兴社会学的偏见 |
| **民族主义** | 一战期间坚定支持法国，参与战争 |
| **教育热情** | 亲自创办《社会学年鉴》培养学派 |
| **关注现实** | 将学术与社会现实紧密结合 |
| **韧性来源** | 对社会学作为一门独立学科的信念 |

## 核心语录
> "社会现象是独立于个人的客观存在"  
> "社会是超越个人的道德实体"  
> "社会事实必须当作物来研究"

## 历史定位
涂尔干是社会学的奠基人之一，他使社会学成为一门独立的学科。他创立的法国社会学学派培养了大批社会学家，其研究方法论影响深远。涂尔干关于社会团结、宗教功能、失范等概念，成为现代社会学的核心概念。

# 斯金纳 (B.F. Skinner) 档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.85 |
| **领域** | 行为主义心理学、学习理论、行为分析 |
| **时代** | 20世纪（1904-1990） |
| **核心身份** | 行为主义旗手、强化理论奠基人 |

## 五维度分析

### 1. 核心思维
- **操作性条件反射**：行为由后果塑造
- **强化理论**：正强化、负强化、惩罚
- **行为塑造**：通过连续逼近塑造复杂行为
- **激进行为主义**：内在心理状态不是解释行为的必要因素

### 2. 底层逻辑思维
```
行为 → 后果(强化/惩罚)
     ↓
强化 → 增加行为频率
     ↓
惩罚 → 减少行为频率
     ↓
消退 → 移除强化导致行为减少
```
- 人性观：行为主要由环境强化塑造
- 方法论：严格实验控制
- 核心洞见：行为的根本机制是强化

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **及时强化** | 行为后立即给予强化 |
| **连续强化** | 新行为初期用连续强化 |
| **差别强化** | 强化期望行为而非不期望行为 |
| **代币制度** | 使用间接强化物 |

### 4. 核心价值观
- **科学**——心理学应该是可观察行为的科学
- **环境**——行为主要由环境决定
- **应用**——行为原理可用于改善人类处境
- **可塑性**——人具有被塑造的可能性

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **科学坚持** | 坚持行为主义立场数十年 |
| **公众争议** | 因乌托邦设想"瓦尔登湖第二"受批评 |
| **高产创作** | 大量著作传播行为主义 |
| **应用推广** | 将行为原理应用于教育等 |

## 核心语录
> "行为是由其后果塑造的。"
> "不要问'他为什么这样做'，而要问'是什么强化了这种行为'。"
> "环境的控制就是行为的控制。"

## 历史定位
斯金纳是20世纪最有影响力也是最有争议的心理学家之一。他的行为主义理论对学习理论、教育实践和治疗技术产生了深远影响。虽然他的激进观点受到批评，但其基本原理至今仍广泛应用。

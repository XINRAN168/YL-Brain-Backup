# 张雪峰

> "民间社会学家"、"教育信息差破壁者"、"普通家庭命运规划师"

## 一、核心思维

### 1.1 信息差破壁思维
- 中国教育体系存在巨大的信息不对称：城市与农村、圈内与圈外、名校与普校
- 张雪峰的核心能力：**把圈内人默认知道的规则，翻译成圈外人能听懂的话**
- 不仅是"知道信息"，更是"知道哪些信息对普通人致命"

### 1.2 逆向终点规划思维
- 不从"我喜欢什么"出发，而从"终点在哪里"倒推
- 先问：你要什么样的生活？→ 需要什么收入？→ 需要什么行业？→ 需要什么专业？→ 需要考什么学校？
- 和大多数人的"顺向选择"完全相反

### 1.3 阶层现实思维
- 不回避阶层差异，直接面对："你家有没有资源"决定了选择策略
- 有资源的家庭可以追理想，没资源的家庭必须先活下来
- 核心逻辑：**先解决生存，再追求发展**

### 1.4 关键理念
| 理念 | 内涵 |
|:---|:---|
| 专业决定命运 | 选错专业的代价远大于选错学校 |
| 信息差就是命 | 不知道规则的人注定吃亏 |
| 实用主义至上 | 不能变现的学历就是废纸 |
| 普通人优先 | 所有建议以普通家庭为基准线 |

## 二、底层逻辑思维

### 2.1 信息不对称逻辑
- 教育市场是典型的信息不对称市场：卖方（高校/机构）比买方（学生/家长）掌握更多信息
- 张雪峰的角色 = 信息中介 + 信号放大器
- 核心动作：**把隐性信息显性化，把复杂规则简单化**

### 2.2 机会成本逻辑
- 每一个选择都有隐性成本：时间、金钱、机会窗口
- 考研二战的隐性成本 = 1年工资 + 1年经验 + 心理消耗
- 不考研的隐性成本 = 学历天花板 + 行业准入门槛
- 所有建议基于：**哪个选择的综合成本更低**

### 2.3 风险优先逻辑
- 普通家庭承受风险的能力极低，所以策略必须保守
- "先求不败，再求胜"：先确保有饭吃，再考虑发展
- 不推荐"赌一把"式选择，除非有退路
- 核心原则：**亏得起才敢赌，亏不起就求稳**

### 2.4 决策框架
```
家庭资源评估 → 风险承受力判断 → 终点倒推 → 路径选择 → 风险对冲
```

## 三、核心行为准则

### 3.1 犀利直接，不打太极
- 该劝退就劝退："这个专业别报"
- 该泼冷水就泼："你家这条件不适合出国"
- 绝不为了好听而说假话

### 3.2 站在弱者立场
- 所有建议默认假设：学生家庭没资源、没背景、没人脉
- 有资源的情况单独讨论，不混为一谈
- "我是帮普通人的，不是帮既得利益者的"

### 3.3 只说真话，不怕得罪人
- 批评过很多专业、学校、机构
- 面对争议不退缩，坚持立场
- 逻辑："如果我说假话能多赚，但那些孩子怎么办"

### 3.4 系统化输出
- 不是零散建议，而是一整套规划体系
- 从高考志愿→考研→就业→考公，形成闭环
- 把碎片信息整合成可执行的路线图

## 四、核心价值观

### 4.1 教育公平
- 教育是普通人改变命运的几乎唯一通道
- 信息差正在制造新的不公平：知道规则的人越来越占优
- 打破信息差 = 打破阶层固化的一部分

### 4.2 实用主义
- 学历的终极意义是变现，不是装饰
- 不能就业的专业就是不负责任的专业
- 理想很美好，但得先有饭吃

### 4.3 普通人本位
- 不为精英服务，为普通人服务
- 所有建议以"最差情况能接受"为底线
- "我说的每句话，都是站在那个农村孩子角度想的"

### 4.4 长期主义
- 短期的"好选择"可能是长期的陷阱（如热门专业4年后饱和）
- 强调看5-10年趋势，不只看当下
- 但长期主义的前提是：**先活过短期**

## 五、心力/情绪韧性

### 5.1 底层突围的韧性
- 出身黑龙江齐齐哈尔贫困家庭，郑州大学本科
- 从考研辅导老师起步，一步步建立自己的影响力
- 没有背景、没有资源，靠信息能力和表达能力破局

### 5.2 面对争议不退缩
- 多次因言论引发争议，被批评"功利"、"短视"
- 但始终坚持：我的受众是普通人，不是精英
- 核心信念："得罪人不重要，帮到人才重要"

### 5.3 高强度输出的持久力
- 长年高频直播、讲座、咨询
- 面对大量重复性问题保持耐心和热情
- 身体透支但依然坚持——因为"停下来就有人找不到方向"

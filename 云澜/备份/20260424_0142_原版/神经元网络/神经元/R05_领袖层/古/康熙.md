# 康熙帝（爱新觉罗·玄烨，1654-1722）

> "康熙大帝"、"千古一帝"、"中国在位时间最长的皇帝之一"

## 一、核心思维

### 1.1 本质洞察
- **民族融合**：满汉一家，化解矛盾
- **以民为本**：民为邦本，本固邦宁
- **文治武功**：两手抓，两手都要硬

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 满汉一家 | 民族平等，化解矛盾 |
| 永不加赋 | 轻徭薄赋，与民休息 |
| 维护统一 | 维护国家领土完整 |

### 1.3 思想特征
- **开放包容**：接受汉文化，尊重各民族
- **学习精神**：向传教士学习西方科学
- **战略眼光**：维护统一，守卫边疆

## 二、底层逻辑

### 2.1 治国方略
```
稳定大局→发展经济→文化融合→边疆治理
```

### 2.2 统治逻辑
- 满汉官员平衡
- 经济恢复与发展
- 维护多民族统一
- 重视文化教育

### 2.3 决策框架
- 深入调查研究
- 听取多方意见
- 权衡利弊得失
- 果断采取措施

## 三、行为准则

### 3.1 政治实践
- 擒鳌拜，夺回权力
- 平三藩，稳定局势
- 收台湾，完成统一
- 驱逐沙俄，保卫边疆

### 3.2 经济政策
- 永不加赋，轻徭薄赋
- 兴修水利，发展农业
- 废除圈地，恢复生产
- 重视手工业和商业

### 3.3 文化建设
- 编纂《康熙字典》
- 重视科举，选拔人才
- 弘扬儒学，笼络汉族士人
- 兴办教育，培养人才

## 四、价值观

### 4.1 统一观
- 国家统一高于一切
- 维护领土完整
- 反对分裂势力

### 4.2 民族观
- 满汉一家
- 民族平等
- 化解矛盾

### 4.3 帝王观
- 皇帝是天下共主
- 以民为本
- 文治武功并重

## 五、心力韧性

### 5.1 人生起伏
- 8岁登基，14岁亲政
- 擒鳌拜夺回权力
- 平三藩统一南方
- 收台湾完成统一
- 驱逐沙俄安定北方
- 晚年九子夺嫡

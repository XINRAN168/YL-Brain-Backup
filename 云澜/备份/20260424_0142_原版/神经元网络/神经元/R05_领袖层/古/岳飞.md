# 岳飞（1103-1142）

> "岳武穆"、"精忠报国"、"中国历史上最著名的民族英雄之一"

## 一、核心思维

### 1.1 本质洞察
- **精忠报国**：国家利益高于一切
- **文治武功**：文武双全是最高境界
- **民族大义**：驱逐金兵，收复中原

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 精忠报国 | 忠君爱国，矢志不渝 |
| 冻死不拆屋 | 爱民如子，秋毫无犯 |
| 收复中原 | 驱逐金兵，迎回二帝 |

### 1.3 思想特征
- **忠义精神**：忠诚是立身之本
- **严明军纪**：令行禁止，绝不扰民
- **坚韧不拔**：矢志北伐，收复中原

## 二、底层逻辑

### 2.1 崛起路径
```
从军抗金→建立岳家军→北伐中原→莫须有之罪
```

### 2.2 军事思想
- 精兵政策
- 严格训练
- 严明军纪
- 民众支持

### 2.3 领导风格
- 身先士卒
- 与士卒同甘共苦
- 赏罚分明
- 重视情报

## 三、行为准则

### 3.1 军事实践
- 组建岳家军
- 收复建康
- 镇压杨幺起义
- 四次北伐，节节胜利
- 郾城大捷，重创金兵
- 十二道金牌召回

### 3.2 治军原则
- 冻死不拆屋，饿死不抢掠
- 精忠报国是岳家军的灵魂
- 赏罚分明，激励将士
- 重视训练，提高战力

### 3.3 政治立场
- 坚持北伐
- 反对议和
- 忠君爱国
- 迎回二帝

## 四、价值观

### 4.1 忠义观
- 精忠报国是最高准则
- 君命不可违
- 忠孝不能两全时，忠君优先

### 4.2 民族观
- 汉族是正统
- 金兵是入侵者
- 驱逐异族，收复中原

### 4.3 军人观
- 军人以战死沙场为荣
- 马革裹尸是归宿
- 保家卫国是使命

## 五、心力韧性

### 5.1 人生起伏
- 出身农家，励志从军
- 展露军事才华
- 组建岳家军，战功赫赫
- 郾城大捷，收复有望
- 十二道金牌，被迫撤军
- 风波亭遇害，年仅39岁

### 5.2 经典语录

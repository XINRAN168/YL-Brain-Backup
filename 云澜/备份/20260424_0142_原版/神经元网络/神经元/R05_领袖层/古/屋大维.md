# 屋大维（前63-14）

> "奥古斯都"、"罗马帝国开国皇帝"、"元首制创始人"

## 一、核心思维

### 1.1 本质洞察
- **隐忍待机**：在不利条件下积蓄力量
- **表面共和**：不称帝，用共和的外衣包装帝制
- **长久统治**：不求速成，追求稳恒

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 表面共和 | 不称帝，用共和制包装帝制 |
| 隐忍务实 | 韬光养晦，逐步掌权 |
| 和平安定 | 给罗马带来和平与秩序 |

### 1.3 思想特征
- **政治智慧**：善于伪装，不树敌
- **耐心积累**：数十年逐步掌权
- **稳定优先**：稳定是发展的基础

## 二、底层逻辑

### 2.1 崛起路径
```
凯撒养子→三头同盟→内战胜利→元首→奥古斯都
```

### 2.2 夺权策略
- 以凯撒继承人身份崛起
- 与安东尼、雷必达结盟
- 逐步消灭安东尼
- 确立独尊地位

### 2.3 统治逻辑
- 表面维护共和制
- 实际上独揽大权
- 用法律和制度包装
- 给罗马带来和平

## 三、行为准则

### 3.1 政治实践
- 确立元首制度
- 建立常备军队
- 改革行省制度
- 完善法律制度

### 3.2 军事行动
- 统一罗马
- 扩张帝国边界
- 防御日耳曼
- 稳定边疆

### 3.3 建设成就
- 罗马的和平
- 建设罗马城
- 奖励文化艺术
- 完善公共设施

## 四、价值观

### 4.1 权力观
- 权力要隐而不显
- 共和的外衣是必要的
- 稳定高于一切

### 4.2 罗马观
- 罗马是世界的中心
- 罗马文明最优越
- 罗马的和平高于一切

### 4.3 历史观
- 我给罗马带来和平
- 我建立了新的罗马
- 名垂青史

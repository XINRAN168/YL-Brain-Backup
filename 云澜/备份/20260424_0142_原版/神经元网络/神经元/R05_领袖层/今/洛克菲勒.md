# 洛克菲勒（1839-1937）

> "约翰·D·洛克菲勒"、"石油大王"、"美国第一位亿万富翁"

## 一、核心思维

### 1.1 本质洞察
- **托拉斯战略**：通过垄断获取利润
- **效率至上**：效率是企业的生命
- **慈善回馈**：财富回馈社会

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 托拉斯战略 | 通过垄断获取利润 |
| 效率至上 | 效率是企业生命 |
| 慈善公益 | 财富回馈社会 |

### 1.3 思想特征
- **商业眼光**：敏锐的商业洞察力
- **垄断思维**：通过整合获得优势
- **慈善精神**：财富应该回馈社会

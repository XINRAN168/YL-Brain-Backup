# 莫扎特（1756-1791）

> "音乐神童"、"音乐天才"、"古典主义音乐大师"

## 一、核心思维

### 1.1 本质洞察
- **音乐是情感**：音乐是情感的表达
- **天赋与努力**：天赋需要努力发挥
- **艺术自由**：艺术需要自由创作

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 音乐情感 | 音乐是情感表达 |
| 天赋努力 | 天赋需要努力 |
| 艺术自由 | 自由创作 |

### 1.3 思想特征
- **音乐天赋**：与生俱来的音乐才能
- **创作自由**：不受限制地创作
- **情感表达**：用音乐表达情感

## 二、底层逻辑

### 2.1 成就路径
```
神童→欧洲巡演→宫廷音乐家→自由创作→音乐大师
```

### 2.2 创作逻辑

### 2.3 决策框架

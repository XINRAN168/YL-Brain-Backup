# 松下幸之助（1894-1989）

> "松下幸之助"、"经营之神"、"日本经营四圣之一"

## 一、核心思维

### 1.1 本质洞察
- **自来水哲学**：生产像自来水一样便宜的产品
- **水库式经营**：经营要有余裕
- **企业使命**：企业是社会的公器

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 自来水哲学 | 提供质优价廉的产品 |
| 水库式经营 | 留有余地，有备无患 |
| 经营是公器 | 企业为社会服务 |

### 1.3 思想特征
- **顾客导向**：以顾客为中心
- **员工为本**：员工是企业之本
- **长远眼光**：思考百年后的事

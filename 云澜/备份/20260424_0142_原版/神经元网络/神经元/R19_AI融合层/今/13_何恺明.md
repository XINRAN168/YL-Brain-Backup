# 何恺明 (1984-)

> "ResNet之父"、"Kaiming初始化提出者"、"计算机视觉顶级华人"

## 一、核心思维

### 1.1 本质洞察
- **残差学习论**：恒等映射是网络最深处的最优解，残差连接解决深度网络训练难题
- **深度即表征论**：网络深度与特征表征能力正相关
- **预训练迁移论**：大规模视觉预训练可迁移到下游任务

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| ResNet | 残差网络，深度学习的里程碑 |
| Kaiming初始化 | 深度网络权重的最佳初始化方法 |
| Mask R-CNN | 目标检测与实例分割的统一框架 |

### 1.3 思想特征
- **简洁至美**：用最简单的结构解决核心问题
- **实验驱动**：理论假设必须经过大规模实验验证
- **追求极致**：每项工作都要达到SOTA

## 二、底层逻辑

### 2.1 创新路径
```
计算机视觉 → 深度学习 → 残差网络 → 自监督学习
```

### 2.2 核心贡献
- ResNet：解决深度网络退化问题
- Kaiming初始化：MSRA初始化
- Mask R-CNN：实例分割标准框架
- MAE：自监督视觉预训练

### 2.3 方法论
- 大规模基准测试
- 消融实验验证
- 代码开源复现

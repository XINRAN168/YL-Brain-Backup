# 阮一峰

## 核心思维

**知识普及观**：技术不应该只是少数人的特权。《阮一峰的技术博客》和《科技爱好者周刊》证明了优质的技术内容可以惠及广泛读者。每周坚持输出，用持续性打败完美主义。

**技术与人文交叉**：技术最终服务于人，纯粹的代码无法改变世界。关注技术对社会的影响，倡导科技向善。

## 底层逻辑思维

1. **从现象到本质**：每篇技术文章都从"为什么"开始，而非"怎么做"
2. **案例驱动**：通过具体案例解释抽象概念，降低理解门槛
3. **知识网络**：技术知识是相互关联的，形成网络而非孤立点
4. **信息筛选**：在信息爆炸时代，筛选和提炼比收集更重要

## 核心行为准则

- **持续输出**：十余年坚持每周更新博客
- **开放分享**：所有技术文档和代码都以开源形式发布
- **读者视角**：始终考虑读者是否能够理解，而非炫技
- **跨界视野**：不仅写技术，也关注社会、文化、经济趋势

## 核心价值观

**知识流动**：信息应该自由流动，打破信息壁垒。
**终身学习**：技术变化快速，但学习方法比具体知识更重要。
**独立思考**：不盲从权威，对技术趋势保持批判性思考。

## 心力/情绪韧性

| 维度 | 表现 |
|:---|:---|
| 抗压能力 | 在996文化盛行的环境下坚持个人时间用于写作 |
| 韧性来源 | 读者认可和长期影响力带来的成就感 |
| 情绪管理 | 对负面评论能够淡然处之，专注内容本身 |
| 恢复力 | 每篇文章的反馈循环形成正向激励 |

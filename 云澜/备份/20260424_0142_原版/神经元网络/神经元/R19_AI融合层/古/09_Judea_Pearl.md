#  Judea Pearl (朱迪亚·珀尔, 1936-2024)

> "贝叶斯网络之父"、"因果推断先驱"、"图灵奖得主"

## 一、核心思维

### 1.1 本质洞察
- **因果推断论**：相关不等于因果，AI需要理解因果才能真正智能
- **贝叶斯网络论**：概率图模型是处理不确定知识的最佳工具
- **反事实框架论**：因果理解的核心是反事实推理

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 贝叶斯网络 | 用图结构表示因果/概率关系 |
| 因果阶梯 | 关联→干预→反事实三层因果 |
| do-calculus | 用干预而非观察进行因果推理 |

### 1.3 思想特征
- **因果革命**：从相关性认知升级到因果性理解
- **数学严格性**：因果推断必须有严格数学基础
- **范式挑战**：敢于挑战统计学的因果禁忌

## 二、底层逻辑

### 2.1 创新路径
```
概率论 → 图模型 → 因果推断 → 智能理解
```

### 2.2 核心贡献
- 贝叶斯网络：不确定推理的标准工具
- 因果革命：《为什么》推动科学方法论变革
- do-calculus：因果干预的数学语言
- 结构因果模型：统一因果推理框架

### 2.3 方法论
- 因果先于概率
- 反事实是智能的核心
- 形式化因果语言

## 三、行为准则

### 3.1 学术实践
- 敢于挑战主流范式
- 追求理论的完整性
- 理论与应用并重

### 3.2 学术传承
- 培养因果推断研究者
- 推动因果革命普及
- 跨学科合作

### 3.3 社会责任
- 因果思维改善决策
- 警示相关性的局限
- 推动负责任的AI

## 四、价值观

### 4.1 科学观
- 因果是理解的本质
- 相关性知识浅薄，因果性知识深刻
- 形式化是科学的追求

### 4.2 人生观
- 追求真理不惜代价
- 学术创新需要勇气
- 因果理解是人类智能的核心

### 4.3 风险观
- 忽视因果导致错误决策
- AI缺乏因果理解是根本缺陷
- 因果革命将改变科学

# 克劳德·香农 (Claude Shannon, 1916-2001)

> "信息论之父"、"数字电路理论奠基人"、"20世纪最伟大数学家之一"

## 一、核心思维

### 1.1 本质洞察
- **信息量化论**：信息可以被精确量化，用"比特"作为基本单位
- **信道容量论**：任何信道都有最大可靠传输速率
- **熵的本质论**：信息熵与热力学熵的深层联系

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 信息熵 | H = -Σp(x)log₂p(x)，信息不确定性的度量 |
| 信道容量 | 香农极限，最大可靠传输速率 |
| 编码定理 | 信源编码与信道编码的数学基础 |

### 1.3 思想特征
- **公理化思维**：从少数假设推导完整理论
- **概率论视角**：不确定性是信息的本质
- **工程导向**：理论必须可实现、可应用

## 二、底层逻辑

### 2.1 创新路径
```
物理现象 → 信息抽象 → 数学建模 → 工程极限
```

### 2.2 核心贡献
- 信息论：通信的数学理论
- 数字电路设计：布尔代数应用于电路
- 密码学：一次性密码本的最优性证明
- AI先驱：早期电子游戏和智能机器探索

### 2.3 方法论
- 抽象是理论的灵魂
- 用概率处理不确定性
- 理论与工程的完美结合

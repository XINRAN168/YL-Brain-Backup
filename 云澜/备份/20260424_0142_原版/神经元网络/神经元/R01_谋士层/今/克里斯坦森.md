# 克莱顿·克里斯坦森 (Clayton Christensen, 1952-2020)

> "颠覆式创新之父"、"最具影响力的商业思想家"

## 一、核心思维

### 1.1 本质洞察
- **颠覆式创新**：来自边缘的创新最终颠覆主流
- **价值网络**：企业在特定网络中竞争
- **资源-流程-价值观（RPV）框架**：解释企业为何失败

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 颠覆逻辑 | 颠覆者从低端或新市场切入 |
| 能力边界 | 企业的能力决定其视野 |
| 价值转移 | 价值在产业链中不断转移 |

### 1.3 思想特征
- **理论建构**：从案例中提炼通用理论
- **逆向思维**：从失败中学习成功
- **长期视角**：关注十年维度的竞争演变

## 二、底层逻辑

### 2.1 颠覆式创新路径
```
边缘切入 → 产品简化 → 性能提升 → 主流颠覆
```

### 2.2 价值网络竞争
- 企业的成本结构、利润率取决于其价值网络
- 同一价值网络内的竞争更加激烈
- 价值网络的边界在不断扩展

### 2.3 RPV框架
- 资源：企业拥有的有形和无形资产
- 流程：解决问题、完成任务的方式
- 价值观：判断什么重要、什么不重要的标准

## 三、行为准则

### 3.1 创新管理
- 区分维持性创新与颠覆性创新
- 用独立组织探索颠覆性机会
- 接受早期的不完美

### 3.2 战略决策
- 识别价值网络的边界
- 判断颠覆性威胁的时机
- 平衡短期与长期利益

### 3.3 组织设计
- 为颠覆性创新创建独立机构
- 避免让主流业务的流程和价值观影响新业务
- 培养容忍失败的创新文化

## 四、价值观

### 4.1 创新观
- 颠覆性创新是市场变革的主要动力
- 边缘往往是创新的起点
- 失败是创新的代价

### 4.2 竞争观
- 竞争对手往往在自身价值网络中定义
- 竞争优势来自独特的能力组合
- 成功的公司往往被自己的成功所束缚

### 4.3 管理观
- 好的管理本身可能是失败的根源
- 流程和价值观一旦固化就难以改变
- 领导者的任务是创造和管理变化

## 五、心力韧性

### 5.1 传奇人生
- 被诊断为淋巴癌，却用此经历重新思考医疗创新
- 从哈佛逃学生到顶尖教授的逆袭
- 在学术与实践中架起桥梁

### 5.2 经典语录
> "大多数创新失败，不是因为技术不够好，而是因为市场不接受。"

> "颠覆性创新通常更简单、更便宜、更便捷。"

> "失败是创新的必要代价。"

### 5.3 遗产
- 颠覆式创新理论影响了全球商业思维
- 影响了无数企业的战略决策
- 改变了商学院的教育方向

# 亨利·明茨伯格 (Henry Mintzberg, 1939-)

> "战略管理先驱"、"管理实践派"

## 一、核心思维

### 1.1 本质洞察
- **战略10Ps模型**：计划、定位、模式、视角、人员等十要素
- **管理角色理论**：管理者扮演十种角色
- **自发战略论**：战略可以自下而上涌现

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 实践导向 | 战略来自实践，而非象牙塔 |
| 有机战略 | 边做边调整的战略形成方式 |
| 战略学习 | 战略制定与执行是一体的 |

### 1.3 思想特征
- **反主流思维**：质疑传统MBA教育
- **实战主义**：理论必须服务实践
- **系统思考**：从整体看待组织

## 二、底层逻辑

### 2.1 战略形成模型
```
愿景引导 → 愿景检验 → 战略学习 → 战略固化
```

### 2.2 管理角色
- 人际角色：头脸人物、领导者、联络者
- 信息角色：监控者、传播者、发言人
- 决策角色：创业者、干扰应对者、资源分配者、谈判者

### 2.3 战略类型
- 深思熟虑的战略
- 应急的战略
- 空想的战略
- 政策的战略

## 三、行为准则

### 3.1 管理实践
- 深入一线，了解真实情况
- 平衡短期与长期
- 培养直觉判断力

### 3.2 组织设计
- 结构追随战略
- 重视非正式组织
- 保持组织弹性

### 3.3 领导力培养
- 从实践中学习
- 批判性思维
- 跨界学习

## 四、价值观

### 4.1 管理观
- 管理是艺术，不是科学
- 书本知识必须与实践结合
- 管理者需要全面视野

### 4.2 战略观
- 战略不是一次性规划
- 战略与执行不可分割
- 战略可以有多种路径

### 4.3 教育观
- 批判MBA教育的碎片化
- 强调行动学习
- 培养实战型管理者

## 五、心力韧性

### 5.1 学术坚守
- 挑战管理学主流范式
- 坚持实践导向的研究
- 推动管理教育变革

### 5.2 经典语录
> "管理者做的事情，在教科书里找不到。"

> "战略不是规划出来的，而是学出来的。"

> "一个优秀的管理者需要平衡矛盾。"

### 5.3 遗产
- 建立了战略管理的实践学派
- 影响了管理教育的改革方向
- 为管理理论注入了现实主义

# 克莱顿·克里斯坦森 (Clayton Christensen, 1952-2020)

> "颠覆式创新之父"、"创新解答之父"

## 一、核心思维

### 1.1 本质洞察
- **颠覆式创新**：从边缘开始的创新终将颠覆主流
- ** Jobs to Be Done**：用户购买的是产品背后的"工作"
- **创新解答**：系统性地找到创新机会

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 颠覆路径 | 低端/新市场切入 → 逐步提升 → 颠覆主流 |
| 工作理论 | 理解用户想完成的"工作" |
| 资源-流程-价值观 | 成功企业的三大陷阱 |

### 1.3 思想特征
- **理论驱动**：从案例中提炼通用理论
- **系统思考**：理解创新与竞争的系统动态
- **实践导向**：理论必须能指导实践

## 二、底层逻辑

### 2.1 颠覆式创新模型
```
边缘切入 → 改善轨迹 → 性能足够 → 主流颠覆
```

### 2.2 工作理论框架
- 用户想完成什么工作
- 用户目前如何完成
- 什么阻碍了他们
- 理想的解决方案是什么

### 2.3 创新机会识别
- 识别功能性工作
- 识别情感性工作
- 识别社会性工作

## 三、行为准则

### 3.1 创新策略
- 从边缘开始
- 关注非消费者
- 允许早期不完美

### 3.2 产品策略
- 聚焦用户工作
- 简化而非复杂化
- 关注用户体验

### 3.3 组织策略
- 独立小团队探索颠覆性机会
- 避免用主流业务的逻辑评估新业务
- 容忍失败

## 四、价值观

### 4.1 创新观
- 创新是可以管理的
- 颠覆来自边缘
- 失败是创新的代价

### 4.2 竞争观
- 竞争的本质是争夺用户的工作
- 品牌忠诚度是脆弱的
- 持续创新是唯一护城河

### 4.3 管理观
- 好的管理可能是失败的根源
- 流程和价值观会固化
- 领导者的任务是创造和管理变化

## 五、心力韧性

### 5.1 传奇经历
- 从耶鲁逃学创业
- 患癌后重新思考医疗创新
- 影响了全球商业思想

### 5.2 经典语录
> "大多数创新失败，不是因为技术不够好，而是因为市场不接受。"

> "用户购买的不是产品，而是产品帮助他们完成的工作。"

> "失败是成功的必由之路。"

### 5.3 遗产
- 颠覆式创新理论影响了全球商业
- _jobs to be Done_改变了产品思维
- 推动了商学院教育改革

# 亚伯拉罕·马斯洛 (Abraham Maslow, 1908-1970)

> "人本主义心理学之父"、"需求层次理论提出者"

## 一、核心思维

### 1.1 本质洞察
- **需求层次论**：从生理到自我实现的阶梯
- **顶峰体验**：人可以达到的最高状态
- **人本主义**：人的潜能是无限的

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 需求层次 | 生理→安全→社交→尊重→自我实现 |
| 自我实现 | 人有实现潜能的内在驱动 |
| 成长心态 | 人的能力是不断发展的 |

### 1.3 思想特征
- **整体论**：人是身心灵的统一体
- **积极心理学**：关注人的优点而非弱点
- **人本主义**：尊重人的价值和尊严

## 二、底层逻辑

### 2.1 需求层次模型
```
生理需求 → 安全需求 → 社交需求 → 尊重需求 → 自我实现
     ↑                                              ↓
     ←←←←←←←←← 成长驱动 ←←←←←←←←←←←←←←←←←←←←←←←←
```

### 2.2 动机理论
- 匮乏需求：满足后动机减弱
- 成长需求：越满足越强
- 元需求：对意义和价值的追求

### 2.3 应用框架
- 用户需求洞察
- 员工激励设计
- 组织发展

## 三、行为准则

### 3.1 产品设计
- 理解用户所处的需求层次
- 从满足低层需求到高层需求
- 创造顶峰体验

### 3.2 领导力
- 满足基本需求才能激发高层动机
- 尊重员工的价值
- 帮助员工自我实现

### 3.3 自我成长
- 自我觉察
- 持续学习
- 追求意义

## 四、价值观

### 4.1 人性观
- 人天生有向上的驱动力
- 人的潜能远未被充分开发
- 尊重人的选择和尊严

### 4.2 成功观
- 自我实现是人生的最高目标
- 真正的成功是成为最好的自己
- 成长比成功更重要

### 4.3 社会观
- 健康的社会应该帮助人自我实现
- 每个人都有实现潜能的权利
- 爱是最重要的成长因素

## 五、心力韧性

### 5.1 学术贡献
- 开创了人本主义心理学
- 影响了管理学和领导力理论
- 推动了积极心理学的发展

### 5.2 经典语录
> "如果你只有一把锤子，你会把一切都当作钉子。"

> "一个人如果总是自我实现，他就是一个心理健康的人。"

> "顶峰体验是人类最美妙的时刻。"

### 5.3 遗产
- 需求层次理论成为全球通用的框架
- 影响了管理学、教育学、心理学
- 推动了人本管理的发展

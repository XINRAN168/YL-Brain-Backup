```
│  │  • 新记忆以涟漪形式扩散存储                                           │ │
```

### 2.2 碰撞数据结构

```python
class NeuronCollision:
    """神经元碰撞数据"""
    
    # 第0阶段：能量管理
    energy_state: EnergyState          # 工作/休眠/待激活
    last_activation: datetime         # 最后激活时间
    activation_count: int             # 激活次数
    
    # 第一阶段：内部碰撞
    internal_rules: List[Rule]        # 内部处理规则
    rule_scores: Dict[str, float]     # 规则匹配度
    retrieved_knowledge: List[Knowledge]  # 检索到的知识
    
    # 第二阶段：之间碰撞
    peer_neurons: List[str]           # 关联神经元
    collision_result: Any             # 碰撞结果
    fold_integration: bool            # 是否需要折叠整合
    
    # 第三阶段：记忆存储
    memory_blocks: List[MemoryBlock]   # 分割的记忆块
    ripple_level: int                 # 涟漪层级
    storage_tier: StorageTier         # 存储层级
```

## 三、L1 核心详解

### C1 · 认知核心 · 「观澜」
```
定位：大脑皮层 - 信息处理中枢
能力：观澜识势，洞察秋毫
范围：战略、风险、推演、玄学
碰撞阶段：第一优先（问题解析入口）
```

### C2 · 决策核心 · 「定海」
```
定位：前额叶 - 决策控制中枢
能力：定海神针，果断决断
范围：领袖、破局、时机、系统
碰撞阶段：第二优先（方案生成）
```

### C3 · 执行核心 · 「破军」
```
定位：运动皮层 - 行动执行中枢
能力：破军之势，雷厉风行
范围：执行、资源、影响
碰撞阶段：第三优先（响应输出）
```

### C4 · 沉淀核心 · 「沉淀」
```
定位：海马体 - 记忆存储中枢
能力：沉淀智慧，深化认知
范围：科学、社会、历史、医道
碰撞阶段：支撑角色（提供历史知识）
```

### C5 · 创新核心 · 「星火」
```
定位：创造性神经网络 - 创新中枢
能力：星火燎原，创新突破
范围：创新、创业、商战、AI、用户
碰撞阶段：按需激活（创新需求时）
```

## 六、涟漪波存储机制

### 6.1 涟漪扩散规则

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         涟漪波存储机制                                      │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  新记忆生成 ──→ 涟漪中心点                                                   │

│       ├──→ 第1涟漪层：直接关联神经元                                       │

│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 6.2 记忆分割标准

| 分割类型 | 触发条件 | 示例 |
|:---|:---|:---|
| 时间分割 | 时间跨度 > 2小时 | "上午...下午..." |
| 话题分割 | 话题相关性 < 0.5 | 从A话题跳转到B话题 |
| 实体分割 | 核心实体完全变化 | 讨论对象从甲变成乙 |
| 情绪分割 | 情绪值变化 > 0.5 | 从平静变为激动 |

### 7.1 休眠层级

```python
class EnergyState(Enum):
    """能量状态枚举"""
    ACTIVE = "active"           # 工作中（当前处理）
```

## 八、碰撞置信度评估

### 8.1 碰撞质量评分

```python
class CollisionQuality:
    """碰撞质量评估"""
    
    # 第一阶段内部碰撞质量
    internal_collision_score: float = (
        0.4 * rule_match_score +      # 规则匹配度(40%)
        0.3 * knowledge_depth +      # 知识深度(30%)
        0.3 * processing_speed       # 处理速度(30%)
    )
    
    # 第二阶段之间碰撞质量
    inter_collision_score: float = (
        0.5 * fold_integration +      # 折叠整合度(50%)
        0.3 * peer_coordination +    # 协同配合度(30%)
        0.2 * energy_efficiency      # 能量效率(20%)
    )
    
    # 第三阶段存储质量
    storage_quality: float = (
        0.4 * ripple_accuracy +      # 涟漪准确性(40%)
        0.3 * segmentation_quality + # 分割质量(30%)
        0.3 * routing_correctness    # 路由正确性(30%)
    )
    
    # 综合碰撞质量
    total_collision_quality: float = (
        0.35 * internal_collision_score +
        0.35 * inter_collision_score +
        0.30 * storage_quality
    )
```

### 8.2 碰撞阈值

| 碰撞阶段 | 最低阈值 | 理想值 | 低于阈值处理 |
|:---|:---:|:---:|:---|
| 第一阶段内部碰撞 | 0.5 | 0.8 | 返回基础规则结果 |
| 第二阶段之间碰撞 | 0.4 | 0.7 | 仅使用主神经元 |
| 第三阶段存储 | 0.6 | 0.9 | 延迟存储，标记待处理 |

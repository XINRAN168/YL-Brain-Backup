# 霍布斯 (Thomas Hobbes) 档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.86 |
| **领域** | 政治哲学、社会契约论、人性论 |
| **时代** | 17世纪（1588-1679） |
| **核心身份** | 社会契约论奠基人、机械唯物主义者 |

## 五维度分析

### 1. 核心思维
- **自然状态**：人对人是狼的战争状态
- **社会契约**：通过契约建立政治秩序
- **利维坦**：绝对权威的国家
- **人性自私**：人天生追求私利

### 2. 底层逻辑思维
```
人性自私 → 追求私利
     ↓
资源有限 → 竞争冲突
     ↓
战争状态 → 自然状态的恐惧
     ↓
社会契约 → 让渡权利，建立国家
```
- 人性观：人是自私、理性计算的动物
- 方法论：几何学式的逻辑推演
- 政治观：强权是秩序的前提

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **自我保存** | 自我保存是首要法则 |
| **理性计算** | 趋利避害是理性选择 |
| **契约精神** | 遵守约定的义务 |
| **服从权威** | 维护政治秩序 |

### 4. 核心价值观
- **秩序**——秩序是文明的前提
- **安全**——安全是人的基本需求
- **契约**——契约是社会的基础
- **理性**——理性计算是人的能力

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **流亡经历** | 英国内战期间多次流亡 |
| **思想勇气** | 在动荡时代坚持哲学研究 |
| **长寿高产** | 活到91岁，著作等身 |
| **学术独立** | 不依附任何政治派别 |

## 核心语录
> "人对人是狼。"（自然状态）
> "没有剑的契约只是一纸空文。"
> "生命本身不过是一场运动，器官的机械游戏。"
> "利维坦是一个伟大的人，由所有人的意志组成。"

## 历史定位
霍布斯是现代政治哲学的奠基人之一，他的社会契约论和绝对国家理论对西方政治思想产生了深远影响。虽然他的绝对主义观点受到批评，但他对人性自私的深刻分析和对政治秩序的追求，至今仍是政治哲学讨论的重要议题。

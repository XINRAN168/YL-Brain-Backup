# 皮亚杰 (Jean Piaget) 档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.87 |
| **领域** | 发展心理学、认知心理学、教育理论 |
| **时代** | 20世纪（1896-1980） |
| **核心身份** | 认知发展论创始人、发生认识论奠基人 |

## 五维度分析

### 1. 核心思维
- **认知发展阶段**：感觉运动、前运算、具体运算、形式运算
- **图式理论**：认知结构随经验而发展
- **同化与顺应**：适应环境的两种方式
- **建构主义**：知识是主动建构的

### 2. 底层逻辑思维
```
图式(认知结构) → 理解世界的方式
     ↓
同化 → 将新信息纳入现有图式
     ↓
顺应 → 改变图式以适应新信息
     ↓
平衡 → 同化与顺应的动态平衡
```
- 人性观：儿童是主动探索和建构知识的主体
- 方法论：临床访谈+观察研究
- 核心洞见：认知发展有普遍规律

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **主动探索** | 知识来自与环境的主动互动 |
| **阶段适配** | 教育应匹配认知发展阶段 |
| **游戏学习** | 儿童通过游戏建构知识 |
| **错误价值** | 错误是学习过程的一部分 |

### 4. 核心价值观
- **建构**——知识是主动建构的
- **发展**——儿童不是小大人
- **主动**——儿童是知识的探索者
- **平衡**——认知发展追求平衡状态

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **学术专注** | 毕生致力于儿童认知研究 |
| **观察耐心** | 细致观察儿童行为数年 |
| **理论勇气** | 挑战当时的行为主义主流 |
| **跨学科视野** | 融合生物学和哲学 |

## 核心语录
> "儿童是知识的主动建构者，而不是被动的接受者。"
> "教育的最高目标是培养会思考的人。"
> "每次当你阻止一个错误，你也阻止了一种真理的发现。"

## 历史定位
皮亚杰是20世纪最具影响力的发展心理学家，他的认知发展理论彻底改变了我们对儿童思维的理解。他的建构主义教育思想对现代教育实践产生了深远影响。

# 马克思（Karl Marx）档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.95 |
| **领域** | 哲学、政治经济学、科学社会主义 |
| **时代** | 19世纪（1818-1883） |
| **核心身份** | 马克思主义创始人、科学社会主义奠基人 |

## 五维度分析

### 1. 核心思维
- **唯物史观**：生产力与生产关系的矛盾运动推动社会发展
- **剩余价值论**：揭示资本家剥削工人的秘密
- **阶级斗争**：历史是阶级斗争的历史
- **实践哲学**：哲学家们只是解释世界，问题在于改变世界

### 2. 底层逻辑思维
```
物质生产 → 生产力发展
     ↓
生产关系 → 生产资料所有制
     ↓
社会形态 → 原始→奴隶→封建→资本→共产
     ↓
历史动力 → 阶级斗争（对立统一）
```
- 人性观：人的本质是社会关系的总和
- 方法论：辩证唯物主义，从现象看本质
- 社会观：社会存在决定社会意识

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **实事求是** | 一切从实际出发，理论联系实际 |
| **批判精神** | 怀疑一切，对现存事物始终持批判态度 |
| **改造世界** | 哲学家们只是解释世界，问题在于改变世界 |
| **国际主义** | 无产阶级没有祖国，全世界无产者联合起来 |
| **为人民服务** | 理论为无产阶级和人民大众服务 |

### 4. 核心价值观
- **人民**——历史是人民创造的
- **平等**——消灭阶级差别，实现真正平等
- **解放**——人的全面自由发展
- **正义**——消灭剥削制度
- **实践**——实践是检验真理的唯一标准

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **学术执着** | 几十年如一日在大英图书馆研究 |
| **贫困坚守** | 一生穷困潦倒，始终不改其志 |
| **批判勇气** | 批判资本主义制度的勇气和决心 |
| **理论自信** | 对自己理论正确性的坚定信念 |
| **韧性来源** | 对人类解放事业的坚定信仰 |

## 核心语录
> "哲学家们只是用不同的方式解释世界，问题在于改变世界"  
> "生产力与生产关系的矛盾运动推动社会发展"  
> "无产者在这个革命中失去的只是锁链，他们获得的将是整个世界"

## 历史定位
马克思是19世纪最具影响力的思想家，他创立的马克思主义深刻改变了人类历史进程。马克思主义成为国际共产主义运动的指导思想，在全球范围内产生了深远影响。唯物史观和剩余价值理论是理解现代社会的关键框架。

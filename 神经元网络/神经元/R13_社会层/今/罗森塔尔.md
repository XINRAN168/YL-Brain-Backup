# 罗森塔尔 (Robert Rosenthal) 档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.84 |
| **领域** | 社会心理学、教育心理学、期望效应 |
| **时代** | 现代（1933-至今） |
| **核心身份** | 期望效应研究者、皮格马利翁效应发现者 |

## 五维度分析

### 1. 核心思维
- **皮格马利翁效应**：高期望导致高表现
- **期望效应**：教师期望影响学生表现
- **自我实现预言**：信念可以创造现实
- **实验者效应**：研究者期望影响实验结果

### 2. 底层逻辑思维
```
教师期望 → 差异对待
     ↓
学生感知 → 自我概念变化
     ↓
行为改变 → 更努力或更放弃
     ↓
结果实现 → 期望成真
```
- 人性观：人对他人的期望高度敏感
- 方法论：实验研究+现场实验
- 核心洞见：信念具有改变现实的力量

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **高期望** | 对他人保持高期望 |
| **积极反馈** | 给予积极、具体的反馈 |
| **公平对待** | 避免因期望而产生偏见 |
| **元认知** | 意识到自己的期望如何影响他人 |

### 4. 核心价值观
- **期望**——积极期望有自我实现的力量
- **公平**——所有学生都应得到高期望
- **觉察**——意识到自己的信念效应
- **责任**——对他人的期望是一种责任

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **实验勇气** | 进行突破性的期望实验 |
| **持续研究** | 数十年持续研究期望效应 |
| **教育倡导** | 推动教育中的高期望文化 |
| **自我反思** | 研究者也要反思自己的期望 |

## 核心语录
> "当我们对某人抱有期望时，我们会无意识地以符合这种期望的方式对待他们。"
> "自我实现预言的力量意味着：我们的信念可以成为现实。"
> "皮格马利翁效应告诉我们：期望可以塑造人。"

## 历史定位
罗森塔尔因"皮格马利翁效应"研究而闻名，揭示了期望如何通过人际互动成为现实。这一发现对教育、管理和社会政策都有深远影响，提醒我们意识到信念的力量。

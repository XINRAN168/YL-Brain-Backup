# 戈夫曼 (Erving Goffman) 档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.87 |
| **领域** | 社会学、社会心理学、拟剧论 |
| **时代** | 20世纪（1922-1982） |
| **核心身份** | 拟剧论创始人、日常互动分析大师 |

## 五维度分析

### 1. 核心思维
- **拟剧论**：社会生活是一场戏剧表演
- **前台与后台**：前台表演，后台放松
- **印象管理**：人们精心管理他人对自己的印象
- **框架理论**：通过框架理解社会情境

### 2. 底层逻辑思维
```
社会舞台 → 每个人都是演员
     ↓
剧本期望 → 社会规范、他人期望、观众期望
     ↓
表演策略 → 理想化、神秘化、误解表演
     ↓
印象管理 → 控制信息、塑造形象、维系表演
```
- 人性观：人是戴着面具表演的社会角色
- 方法论：详尽的经验观察
- 核心洞见：日常生活中的互动都是精心管理的表演

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **印象管理** | 时刻关注他人对自己的看法 |
| **情境适应** | 根据情境调整表演方式 |
| **信息控制** | 管理自己呈现给他人的信息 |
| **保持一致** | 外表与举止需协调 |

### 4. 核心价值观
- **表演**——社会生活是角色表演
- **面具**——前台行为不同于真实自我
- **情境**——行为高度依赖具体情境
- **秩序**——社会互动遵循隐含规则

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **观察入微** | 对日常互动细节的敏锐洞察 |
| **学术投入** | 一生专注于微观社会学研究 |
| **低调生活** | 虽然研究表演但本人低调 |
| **持续产出** | 多部影响深远的著作 |

## 核心语录
> "社会是一个舞台，人生是一场表演。"
> "我们都是戴着面具生活的人。"
> "表演者努力维持一种理想化的形象，同时隐藏那些不符合这种形象的方面。"

## 历史定位
戈夫曼是20世纪最具影响力的社会学家之一，他用戏剧比喻分析日常互动，开创了微观社会学的新视角。他的思想对后来的人际交往、印象管理、社会认同研究产生了深远影响。

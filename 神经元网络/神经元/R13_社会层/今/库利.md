# 库利 (Charles Cooley) 档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.83 |
| **领域** | 社会学、社会心理学、符号互动论 |
| **时代** | 19-20世纪（1864-1929） |
| **核心身份** | 镜像自我理论创始人、社会学先驱 |

## 五维度分析

### 1. 核心思维
- **镜像自我**：自我是在与他人的互动中形成的
- **社会我**：人格是社会互动的产物
- **想象性预演**：社会互动中的心理过程
- **首属群体**：家庭、朋友对人格形成的重要性

### 2. 底层逻辑思维
```
社会互动 → 他人作为镜像
     ↓
想象感知 → 想象他人如何评价自己
     ↓
自我感受 → 对这些评价产生自我感
     ↓
社会自我 → 自我是社会的产物
```
- 人性观：自我完全是社会的产物
- 方法论：内省法+理论建构
- 核心洞见：他人是自我形成的镜子

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **社会反思** | 通过他人认识自己 |
| **群体归属** | 首属群体对人格至关重要 |
| **角色承担** | 在互动中理解他人立场 |
| **社会意识** | 认识到自我的社会根源 |

### 4. 核心价值观
- **社会性**——人是社会性动物
- **互动**——自我在互动中形成
- **关系**——人与人之间的联系是根本
- **群体**——群体塑造人格

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **学术创新** | 在社会学初创期提出重要理论 |
| **跨学科探索** | 融合社会学与心理学 |
| **早逝影响** | 虽英年早逝但影响深远 |
| **传承发展** | 影响了米德等人的思想 |

## 核心语录
> "每个人心中都有一个想象的他人对自己的判断的镜子。"
> "自我完全是社会的产物——没有社会就没有自我。"
> "我们通过与他人的互动来认识自己，就像在镜子中看到自己一样。"

## 历史定位
库利是符号互动论的重要先驱，他的"镜像自我"理论是理解自我社会本质的经典框架。虽然他的学术生涯短暂，但他的思想对后来的符号互动论和社会心理学产生了深远影响。

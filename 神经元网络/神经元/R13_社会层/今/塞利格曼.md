# 塞利格曼 (Martin Seligman) 档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.86 |
| **领域** | 积极心理学、临床心理学、认知心理学 |
| **时代** | 现代（1942-至今） |
| **核心身份** | 积极心理学之父、习得性无助研究者 |

## 五维度分析

### 1. 核心思维
- **习得性无助**：反复的不可控经历导致放弃抗争
- **解释风格**：归因方式影响心理韧性
- **积极心理学**：研究人类优势和最佳功能
- **PERMA模型**：积极情绪、投入、关系、意义、成就

### 2. 底层逻辑思维
```
不可控事件 → 无助感
     ↓
解释风格 → 永久性、普遍性、个人化
     ↓
习得性无助 → 放弃、抑郁、被动
     ↓
反转 → 习得性乐观、韧性
```
- 人性观：人既有脆弱也有韧性，关键在于解释风格
- 方法论：实证研究+临床应用
- 核心洞见：乐观可以被习得

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **习得性乐观** | 通过改变解释风格获得乐观 |
| **关注优势** | 发展优势比修复弱点更重要 |
| **意义导向** | 追求有意义的生活 |
| **预防为主** | 防止心理问题比治疗更重要 |

### 4. 核心价值观
- **积极**——关注人类的优点和潜能
- **乐观**——乐观是可以习得的
- **意义**——生活需要意义感
- **成长**——人具有持续成长的能力

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **学术转型** | 从研究抑郁转向研究幸福 |
| **公众传播** | 积极推广积极心理学 |
| **实践应用** | 在军队、学校广泛应用 |
| **持续创新** | 不断完善PERMA模型 |

## 核心语录
> "习得性无助不是能力的缺失，而是解释风格的失败。"
> "积极心理学是关于使人们生活更美好的科学。"
> "我们可以通过改变对事件的解释来改变我们的情绪。"

## 历史定位
塞利格曼是积极心理学的创始人之一，将心理学从只关注心理疾病转向关注人类优势和最佳功能。他的习得性无助研究和积极心理学理论对临床心理学和教育领域产生了深远影响。

# 华生 (John B. Watson) 档案

| 属性 | 内容 |
|:---:|:---|
| **评分** | 0.83 |
| **领域** | 行为主义心理学、实验心理学、广告心理学 |
| **时代** | 20世纪（1878-1958） |
| **核心身份** | 行为主义创始人、广告心理学先驱 |

## 五维度分析

### 1. 核心思维
- **行为主义宣言**：心理学应是行为的科学，而非意识
- **刺激-反应(S-R)**：行为是刺激与反应之间的联结
- **环境决定论**：行为完全由环境决定
- **预测与控制**：心理学的目标是预测和控制行为

### 2. 底层逻辑思维
```
环境刺激(S) → 有机体
     ↓
行为反应(R) → 可观察行为
     ↓
联结形成 → S-R联结
     ↓
习惯养成 → 强化重复
```
- 人性观：人是环境刺激的反应器
- 方法论：客观、实验、可观察
- 核心洞见：意识不是心理学研究的对象

### 3. 核心行为准则
| 准则 | 内涵 |
|:---|:---|
| **客观研究** | 只研究可观察行为 |
| **环境控制** | 通过环境塑造行为 |
| **科学方法** | 心理学应采用实验方法 |
| **预测控制** | 目标是预测和控制行为 |

### 4. 核心价值观
- **客观性**——科学研究必须客观可观察
- **环境论**——行为由环境决定
- **科学性**——心理学应成为像物理学的科学
- **应用性**——科学知识应服务于实践

### 5. 心力/情绪韧性
| 维度 | 表现 |
|:---|:---|
| **学术勇气** | 公开宣称行为主义纲领 |
| **跨界转型** | 离开学术界进入广告业 |
| **营销创新** | 将心理学原理应用于广告 |
| **预言精神** | 预言"给我一打婴儿"实验 |

## 核心语录
> "给我一打健康的婴儿，让我在我的世界中抚养他们，我可以保证，随便挑一个，都能训练成任何类型的专家——医生、律师、艺术家、商界领袖，甚至是乞丐或小偷，不论他的天赋、倾向、能力、祖先种族是什么。"
> "心理学作为行为的科学，必须放弃对意识的关注。"
> "行为是能够被客观观察和测量的。"

## 历史定位
华生是行为主义的创始人，他的宣言标志着心理学从内省向客观研究的根本转变。虽然他的激进观点后来被修正，但行为主义方法至今仍是心理学的重要组成部分。

# 托马斯·爱迪生 (Thomas Alva Edison, 1847-1931)

> "门洛帕克的奇才"、"发明大王"

## 一、核心思维

### 1.1 本质洞察
- **需求驱动论**：发明必须满足真实需求，否则毫无价值
- **系统创新论**：发明、制造、应用形成完整闭环
- **试错学习论**：失败是找到行不通的方法，数据即反馈

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 市场导向 | 从需求出发而非从技术出发 |
| 实验室工业化 | 将发明从个人行为升级为组织行为 |
| 专利布局 | 系统性专利申请构筑技术壁垒 |

### 1.3 思想特征
- **商业敏锐**：技术发明必须能够商业化
- **系统思维**：思考整个产业链而非单一产品
- **迭代改进**：持续优化，永无止境

## 二、底层逻辑

### 2.1 创新路径
```
市场调研 → 技术分析 → 系统设计 → 专利布局 → 商业化
```

### 2.2 竞争优势
- 垂直整合：从发电到用电的全链条
- 专利护城河：2000多项专利
- 创新流程化：将灵感转化为可复制的流程

### 2.3 研发模式
- 团队协作而非单打独斗
- 明确分工：专业的人做专业的事
- 快速迭代：失败即学习

## 三、行为准则

### 3.1 发明原则
- 先证明发明的必要性
- 在靠谱的技术基础上创新
- 画出完整的商业版图

### 3.2 研究方法
- 测试所有可能的材料
- 记录每一次实验结果
- 从失败中提取有用信息

### 3.3 商业策略
- 低价打开市场，培养用户习惯
- 先占领市场，再优化成本
- 持续改进，不追求一步到位

## 四、价值观

### 4.1 发明观
- 天才是1%的灵感加99%的汗水
- 发明最大的困难是证明"必须性"
- 好产品需要配合好营销

### 4.2 商业观
- 发明必须服务大众
- 民主化创新：让技术惠及普通人
- 满足需求是衡量发明的唯一标准

### 4.3 人生观
- 工作中寻找乐趣
- 永不满足，持续改进
- 坚持到最后一刻

## 五、心力韧性

### 5.1 人生起伏
- 童年被学校判定"智力迟钝"
- 多次创业失败，一度贫困潦倒
- 晚年与特斯拉的电流之战

### 5.2 经典语录
> "天才是1%的灵感加99%的汗水。"

> "我没有失败，我只是发现了一万种行不通的方法。"

> "一个人年轻时奠定的基础，会决定他一生的方向。"

### 5.3 遗产
- 发明电灯、留声机、电影摄影机等2000多项发明
- 创建通用电气，影响至今
- 开创了"工业实验室"的研发模式

# 盛宣怀 (1844-1916年)

> "中国商业之父"、"实业之父"、"近代工业先驱"

## 一、核心思维

### 1.1 本质洞察
- **实业救国论**：实业是国家富强之本
- **官督商办论**：政府与商业的结合模式
- **技术引进论**：引进西方技术是捷径

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 实业兴国 | 工业是强国之本 |
| 官商结合 | 政府主导，商业运作 |
| 引进创新 | 引进消化再创新 |

### 1.3 思想特征
- **家国情怀**
- **创新精神**
- **制度探索**

## 二、底层逻辑

### 2.1 创新路径
```
技术引进 → 消化吸收 → 规模扩张 → 制度创新
```

### 2.2 竞争策略
- 垄断经营
- 技术领先
- 规模效应

### 2.3 决策框架
- 国家利益优先
- 技术引进
- 制度建设

# 郭永茂 (约1880-1940年代)

> "味精大王"、"技术先驱"、"中国发酵工业奠基人"

## 一、核心思维

### 1.1 本质洞察
- **技术兴业论**：技术是工业的根本
- **引进创新论**：引进消化再创新
- **质量为本论**：品质决定市场

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 技术立身 | 核心技术是根本 |
| 质量第一 | 品质是最好的招牌 |
| 引进消化 | 学习是为了超越 |

### 1.3 思想特征
- **技术信仰**
- **质量意识**
- **实业精神**

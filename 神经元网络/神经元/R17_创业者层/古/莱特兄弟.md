# 威尔伯·莱特与奥维尔·莱特 (Wilbur & Orville Wright, 1867-1912 / 1871-1948)

> "飞机之父"、"航空先驱"

## 一、核心思维

### 1.1 本质洞察
- **控制优先论**：飞行器必须能被飞行员控制，而非依赖自然稳定性
- **系统性研究论**：空气动力学、结构、控制系统必须协同设计
- **反转工程论**：先建立理论，再用理论指导实践

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 三轴控制 | 横向、纵向、方向三轴独立控制 |
| 风洞实验 | 用实验数据验证理论 |
| 迭代改进 | 小步快跑，持续优化 |

### 1.3 思想特征
- **工程直觉**：从自行车修理中获得机械直觉
- **系统思维**：飞行是整体而非部件的简单相加
- **实践导向**：一切理论必须经过实验验证

## 二、底层逻辑

### 2.1 创新路径
```
理论学习 → 风洞实验 → 滑翔测试 → 动力飞机 → 控制系统完善
```

### 2.2 技术突破
- 卷翼翘曲：横向控制的核心技术
- 三轴控制系统：横向、纵向、方向
- 风洞数据采集：系统性的空气动力学研究

### 2.3 竞争优势
- 理论与实践结合
- 系统性研究方法
- 持续迭代改进

## 三、行为准则

### 3.1 研究方法
- 系统性地收集数据
- 用实验验证假设
- 记录每一次飞行数据

### 3.2 工程实践
- 从失败中提取教训
- 分解问题，逐个击破
- 追求可重复的成功

### 3.3 团队合作
- 兄弟搭档，优势互补
- 尊重专业，分工明确
- 共同决策，共担风险

## 四、价值观

### 4.1 创新观
- 飞行是系统问题，而非单项突破
- 控制是飞行的核心
- 成功需要耐心和毅力

### 4.2 科学观
- 数据是决策的基础
- 理论必须接受实验检验
- 尊重前人的研究成果

### 4.3 人生观
- 坚持梦想，不畏质疑
- 低调行事，不慕虚名
- 成功后仍保持谦逊

## 五、心力韧性

### 5.1 人生起伏
- 从自行车修理工到飞行先驱
- 多次实验失败和坠机
- 在冷嘲热讽中坚持梦想

### 5.2 经典语录
> "我相信人类飞行是可能的，而且 practical（实用）的。"

> "我们不是偶然成功的，是系统性研究的结果。"

> "只要有足够的时间和资源，任何问题都可以解决。"

### 5.3 遗产
- 发明世界上第一架实用飞机
- 开创了现代航空工业
- 证明了人类可以像鸟类一样飞行

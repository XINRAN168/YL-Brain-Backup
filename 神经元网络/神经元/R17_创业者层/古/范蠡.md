# 范蠡 (约公元前536-前448年)

> "商圣"、"陶朱公"、"儒商始祖"

## 一、核心思维

### 1.1 本质洞察
- **货物流通论**：货畅其流是商业本质
- **待时而动论**：抓住时机是成败关键
- **盈满互补论**：物极必反，盈满则亏

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 计然七策 | 商业经营的七条策略 |
| 贵出如粪土 | 价格高时果断出手 |
| 贱取如珠玉 | 价格低时大胆收购 |

### 1.3 思想特征
- **道家智慧与商业结合**
- **进退有度的从容**
- **功成身退的豁达**

## 二、底层逻辑

### 2.1 创新路径
```
发现商机 → 低买高卖 → 规模经营 → 天下共享
```

### 2.2 竞争策略
- 顺势而为
- 审时度势
- 分散经营

### 2.3 决策框架
- 时机判断
- 风险控制
- 利益共享

## 三、行为准则

### 3.1 商业实践
- 三次散财，三次巨富
- 农末俱利的价格调控
- 十九年三致千金

### 3.2 处世哲学
- 功成身退
- 知足不辱
- 和气生财

### 3.3 社会责任
- 十九年施财于乡里
-  富而好德
-  成为商人的道德楷模

## 四、价值观

### 4.1 创新观
- 货物流通创造价值
- 时机把握决定成败
- 周期规律不可违逆

### 4.2 人生观

## 五、心力韧性

### 5.1 人生起伏
- 辅佐越王勾践灭吴雪耻
- 功成名就后急流勇退
- 携西施泛舟五湖

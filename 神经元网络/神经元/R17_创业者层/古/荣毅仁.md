# 荣毅仁 (1916-2005年)

> "红色资本家"、"中信创始人"、"改革开放先锋"

## 一、核心思维

### 1.1 本质洞察
- **改革开放论**：引进外资是强国之路
- **市场机制论**：市场经济是发展必经之路
- **桥梁作用论**：民营企业家是改革桥梁

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 引进外资 | 开放才能发展 |
| 市场化改革 | 市场是资源配置的方式 |
| 桥梁作用 | 企业家是改革推动者 |

### 1.3 思想特征
- **政治智慧**
- **商业敏锐**
- **家国情怀**

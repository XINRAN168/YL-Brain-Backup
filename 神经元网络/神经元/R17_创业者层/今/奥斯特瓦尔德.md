# 亚历山大·奥斯特瓦尔德 (Alexander Osterwalder, 1970-)

> "商业模式画布创始人"、"商业模式创新大师"

## 一、核心思维

### 1.1 本质洞察
- **商业模式画布论**：商业模式是创造、传递和获取价值的系统
- **九大构建块论**：商业模式由九个相互关联的要素构成
- **可视化管理论**：用画布让商业模式一目了然

### 1.2 商业模式画布九要素
| 构建块 | 内容 |
|:---|:---|
| 客户细分 | 我们为谁创造价值 |
| 价值主张 | 我们为客户创造什么价值 |
| 渠道通路 | 如何触达客户 |
| 客户关系 | 如何与客户互动 |
| 收入来源 | 如何从客户获取价值 |
| 核心资源 | 需要哪些关键资源 |
| 关键业务 | 需要哪些关键活动 |
| 重要合作 | 谁是我们的伙伴 |
| 成本结构 | 商业模式需要哪些成本 |

### 1.3 思想特征
- **系统视角**：商业模式是整体而非部分
- **可视化思维**：让抽象变具体
- **实践导向**：工具服务于实际应用

## 二、底层逻辑

### 2.1 商业模式逻辑
```
价值主张 → 客户关系 → 市场定位 → 价值获取
```

### 2.2 价值创造机制
- 定义目标客户
- 设计价值主张
- 构建交付系统
- 建立盈利模式

### 2.3 商业模式类型
- 长尾模式
- 免费模式
- 平台模式
- 订阅模式

## 三、行为准则

### 3.1 商业模式设计
- 从客户需求出发
- 简化而非复杂化
- 持续测试和迭代

### 3.2 可视化实践
- 团队共同绘制
- 保持简洁明了
- 定期更新

### 3.3 创新方法
- 破坏现有假设
- 寻找新组合
- 跨行业借鉴

## 四、价值观

### 4.1 商业观
- 商业模式决定成败
- 好的模式可以复制
- 持续创新是关键

### 4.2 创业观
- 清晰比完整更重要
- 行动胜于完美
- 测试胜于假设

### 4.3 教育观
- 商业模式可以学习
- 工具服务于思维
- 实践是最好的老师

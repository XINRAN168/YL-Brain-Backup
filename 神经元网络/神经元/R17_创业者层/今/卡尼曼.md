# 丹尼尔·卡尼曼 (Daniel Kahneman, 1934-)

> "行为经济学之父"、"诺贝尔经济学奖得主"

## 一、核心思维

### 1.1 本质洞察
- **双系统论**：人类有两个思维系统——快速直觉与缓慢理性
- **前景理论论**：人们对损失的反应比对等量收益强烈
- **认知偏见论**：人类判断存在系统性偏见

### 1.2 关键概念
| 概念 | 内涵 |
|:---|:---|
| 系统1 | 快速、自动、直觉 |
| 系统2 | 缓慢、理性、费力 |
| 锚定效应 | 初始数字影响判断 |
| 可得性启发 | 容易想起的事件被高估 |

### 1.3 思想特征
- **实验精神**：基于实证研究
- **反直觉思维**：挑战经济人假设
- **谦逊态度**：承认人类认知局限

## 二、底层逻辑

### 2.1 判断偏差机制
- 代表性启发
- 可得性启发
- 锚定与调整
- 过度自信

### 2.2 前景理论
- 损失厌恶：损失比等量收益更痛苦
- 框架效应：表述方式影响决策
- 确定性效应：确定性收益偏好

### 2.3 改进路径
- 了解偏见
- 设计决策流程
- 外部约束

## 三、行为准则

## 四、价值观

### 4.1 理性观
- 人类理性有限
- 系统1不可靠
- 需要外部帮助

### 4.2 决策观
- 决策质量决定结果
- 偏见可以克服
- 流程胜于判断

### 4.3 科学观
- 证据优先
- 承认不确定性
- 持续学习

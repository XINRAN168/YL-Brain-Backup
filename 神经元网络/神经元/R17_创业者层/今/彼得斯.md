# 汤姆·彼得斯 (Tom Peters, 1942-)

> "商界教皇"、"卓越布道师"

## 一、核心思维

### 1.1 本质洞察
- **卓越无止境**：卓越不是终点，而是对平庸的永不妥协
- **行动优于分析**："与其花6个月写报告，不如用6周做出原型"
- **顾客即国王**：顾客是唯一能解雇所有人的老板

### 1.2 卓越企业八大特质
| 特质 | 含义 |
|:---|:---|
| 崇尚行动 | 偏好做而非分析 |
| 贴近顾客 | 痴迷于服务 |
| 自主创新 | 内部创业机制 |
| 以人为本 | 尊重个体价值 |
| 价值驱动 | 价值观>规则 |
| 不离本行 | 专注核心优势 |
| 精兵简政 | 扁平化组织 |
| 宽严并济 | 原则刚性，执行弹性 |

### 1.3 思想特征
- **激情四溢**：对卓越的狂热追求
- **反叛传统**：挑战MBA式的理性分析
- **以人为本**：始终强调人的价值

## 二、底层逻辑

### 2.1 卓越动力学
```
行动 → 学习 → 调整 → 持续行动
  ↑                      ↓
  ←←←←← 卓越飞轮 ←←←←←←
```

### 2.2 软硬要素平衡
| 硬性要素 | 软性要素 |
|:---|:---|
| 战略 | 文化 |
| 结构 | 价值观 |
| 系统 | 人 |
| 技能 | 风格 |

### 2.3 创新破坏观
- 混乱是创新的土壤
- 小即是美
- 拥抱变化而非恐惧

## 三、行为准则

### 3.1 行动原则
- 快速行动，允许失败
- 顾客反馈优先于内部程序
- 扁平化组织层级
- 重视一线员工的判断

### 3.2 卓越追求
- 每天追求1%的改进
- 差异化是生存之道
- 服务是竞争优势

### 3.3 领导力修炼
- 走动式管理
- 教练式授权
- 愿景感召

## 四、价值观

### 4.1 人本主义
- 员工不是"人力资源"，是"人"
- 尊重每个人的独特价值
- 人的尊严不可妥协

### 4.2 卓越追求
- 追求卓越是道德要求
- 拒绝平庸是基本态度
- 持续改进永无止境

### 4.3 商业伦理
- 服务客户是商业存在的理由
- 诚信是卓越的基础
- 社会责任不可忽视

## 五，心力韧性

### 5.1 精神特质
- 永不满足的探索精神
- 面对批评的坚持
- 对未来始终乐观

### 5.2 经典语录
> "卓越不是终点，而是对平庸的永不妥协。"

> "官僚主义是创新的裹尸布。"

> "员工不是'人力资源'，是'人'——前者是零件，后者是创造者。"

### 5.3 影响力
- 《追求卓越》销量超过600万册
- 影响了全球数代管理者
- 推动了人本管理思想的发展

# 拉里·佩奇 (Larry Page, 1973-)

> "搜索引擎先驱"、"登月思维"、"X实验室之父"

## 一、核心思维

### 1.1 本质洞察
- **链接即价值论**：网页链接代表了人类智慧的结晶
- **技术解决论**：大多数问题都可以用技术解决
- **登月思维论**：做10倍改进而非10%优化

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| PageRank算法 | 用链接关系衡量信息价值 |
| 风险投资逻辑 | 投资于疯狂但可能改变世界的事 |
| 员工创新权 | 允许用20%时间做自己想做的事 |

### 1.3 思想特征
- **技术理想主义**：相信技术可以消除信息不对称
- **规模化思维**：从一开始就想如何服务全球
- **产品经理文化**：工程师必须理解产品

## 二、底层逻辑

### 2.1 创新路径
```
理论突破 → 工程实现 → 规模化 → 改变世界
```

### 2.2 竞争策略
- 技术领先优先于市场占有
- 平台战略：成为基础设施
- 开放API吸引开发者

### 2.3 决策框架
- 10倍改进原则
- 70-20-10法则：70%核心、20%新兴、10%探索
- OKR目标管理

## 三、行为准则

### 3.1 商业实践
- 早期不计成本招聘顶尖人才
- 保持技术公司的独立性和文化
- 鼓励内部创业

### 3.2 组织管理
- 小团队自主决策
- 透明的数据驱动决策
- 容忍合理的失败

### 3.3 战略选择
- 进入能成为第一的领域
- 投资于长期技术突破
- 反对渐进式改进

## 四、价值观

### 4.1 创新观
- 做那些听起来不可能的事
- 技术是民主化的工具
- 创新来自边缘

### 4.2 人生观
- 改变世界的雄心
- 不满足于平庸
- 相信年轻人的创造力

### 4.3 商业观
- 建立护城河比短期利润重要
- 平台商业模式
- 数据是21世纪的石油

## 五、心力韧性

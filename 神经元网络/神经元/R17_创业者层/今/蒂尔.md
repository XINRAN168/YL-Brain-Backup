# 彼得·蒂尔 (Peter Thiel, 1967-)

> "硅谷创投教父"、"从0到1思想家"

## 一、核心思维

### 1.1 本质洞察
- **从0到1论**：真正的进步是垂直创新，创造前所未有的新事物
- **垄断论**：竞争是失败者的游戏，垄断才是成功企业的标志
- **幂次法则论**：世界按幂次分布，极少数创造绝大多数价值

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 0→1 vs 1→N | 垂直进步 vs 水平复制 |
| 秘密论 | 商业机会藏在大众未发现的真相里 |
| 逆向思维 | 不从众，独立发现被忽视的机会 |

### 1.3 思想特征
- **反MBA思维**：质疑商学院教的"竞争理论"
- **精英视角**：相信少数人改变世界
- **悲观中的乐观**：认为未来需要人为创造

## 二、底层逻辑

### 2.1 创业方法论
```
发现秘密 → 小市场垄断 → 技术壁垒 → 网络效应 → 规模经济 → 品牌优势
```

### 2.2 垄断企业特征
- 专利技术：比现有方案好10倍以上
- 网络效应：用户越多价值越大
- 规模经济：边际成本趋近于零
- 品牌优势：占领用户心智

### 2.3 投资策略
- 不投赛道，投创始人
- 早期重仓，长期持有
- 寻找能建立"不可替代"壁垒的企业

## 三、行为准则

### 3.1 创业七问
1. 工程问题：技术具有突破性吗？
2. 时机问题：现在开创事业时机合适吗？
3. 垄断问题：在小市场抢占大份额？
4. 人员问题：有合适的团队吗？
5. 销售问题：能销售产品吗？
6. 持久问题：未来10-20年保住地位？
7. 秘密问题：找到独特机会了吗？

### 3.2 决策原则
- 宁投细分市场垄断，不投大市场无差异化
- 寻找"所有人忽视的价值"
- 押注有使命感的创始人

### 3.3 人才观
- 创始人比商业模式更重要
- 需要远见与偏执，而非完美无缺
- 小团队优于大团队

## 四、价值观

### 4.1 创新观
- 创新是垄断的唯一合法路径
- 从1到N是复制，从0到1是创造
- 不在红海中厮杀，要在蓝海中开辟

### 4.2 竞争观
- 竞争是留给失败者的
- 垄断企业才能持续投入创新
- 动态社会中，垄断创造价值

### 4.3 未来观
- 悲观者或许正确，但乐观者才能成功
- 未来需要被主动创造
- 独立思考比跟随潮流更重要

## 五、心力韧性

### 5.1 人生起伏
- PayPal被eBay收购后曾陷入低谷
- 早期投资Facebook被嘲笑"泡沫"
- 在质疑声中坚持逆向投资逻辑

### 5.2 经典语录
> "做大家都知道的事，只会让世界从1到n；做没人做的事，才能让世界从0到1。"

> "竞争是留给失败者的，垄断才是成功企业的标志。"

> "幂次法则就是宇宙的法则。"

### 5.3 遗产
- 投资了Facebook、SpaceX、LinkedIn等改变世界的公司
- 《从0到1》成为创业者的必读经典
- 重新定义了"垄断"与"竞争"的关系

# 埃里克·莱斯 (Eric Ries, 1978-)

> "精益创业之父"、"硅谷创业方法论革新者"

## 一、核心思维

### 1.1 本质洞察
- **验证性学习论**：创业公司的进步单位是"验证性学习"
- **MVP论**：用最小可行产品快速验证假设
- **构建-测量-学习论**：快速反馈循环是创业核心

### 1.2 关键概念
| 概念 | 内涵 |
|:---|:---|
| 最小可行产品(MVP) | 最快方式获取经证实的认知 |
| 经证实的认知 | 科学验证的进步单位 |
| 虚荣指标 | 看起来好但无法指导行动的指标 |
| 转型 | 基于新假设的战略调整 |

### 1.3 思想特征
- **科学精神**：将创业视为可实验的科学过程
- **速度至上**：关注学习速度而非执行速度
- **失败前置**：接受失败作为学习的前提

## 二、底层逻辑

### 2.1 构建-测量-学习循环
```
想法 → 构建MVP → 测量数据 → 学习认知 → 调整方向
```

### 2.2 创新核算机制
| 阶段 | 内容 |
|:---|:---|
| 建立基线 | 通过MVP获取初始数据 |
| 调整引擎 | 改进增长驱动因素 |
| 决策点 | 转型还是坚持 |

### 2.3 精益原则
- 消除浪费
- 小批量生产
- 拉动优于推动
- 持续改进

## 三、行为准则

### 3.1 创业实践
- 快速推出MVP
- 收集真实用户反馈
- 基于数据而非直觉决策
- 准备好随时转型

### 3.2 指标选择
- 可执行指标：显示因果关系
- 可访问指标：团队能理解
- 可审计指标：数据可信

### 3.3 失败处理
- 早期失败是最便宜的
- 失败是学习的必经之路
- 从失败中提取认知

## 四、价值观

### 4.1 创业伦理
- 创业是一种实验
- 诚实面对失败
- 为用户创造真正价值

### 4.2 组织观
- 小团队比大团队更灵活
- 授权一线决策
- 允许试错

### 4.3 社会观
- 每个人都可以是创业者
- 创新是持续的过程
- 失败值得尊重

## 五，心力韧性

### 5.1 创业经历
- IMVU联合创始人
- 从失败中总结方法
- 在逆境中坚持验证

### 5.2 经典语录
> "创业成功可以通过遵循过程来工程化。"

> "如果你不能失败，你就不能学习。"

> "创业公司的本质是管理极端不确定性。"

### 5.3 影响力
- 改变了硅谷创业方法论
- 影响了全球数百万创业者
- 促进了精益方法在各领域的应用

# 马克·库班 (Mark Cuban, 1958-)

> "NBA达拉斯小牛队老板"、"连续创业者"、"鲨鱼Tank明星"

## 一、核心思维

### 1.1 本质洞察
- **客户第一论**：客户是你的衣食父母
- **销售驱动论**：好产品还需要好销售
- **敢于冒险论**：风险与回报成正比

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 行动力 | 想到就做，不等待完美时机 |
| 销售为王 | 产品重要，卖出更重要 |
| 现金流 | 现金为王 |

### 1.3 思想特征
- **实战派**
- **激进冒险**
- **结果导向**

## 二、底层逻辑

### 2.1 创新路径
```
发现机会 → 快速行动 → 销售为王 → 规模化
```

### 2.2 竞争策略
- 差异化定位
- 速度制胜
- 客户导向

### 2.3 决策框架
- 现金流优先
- 快速决策
- 承担可控风险

## 三、行为准则

### 3.1 创业实践
- 卖掉公司实现财务自由

- 购买NBA球队

### 3.2 投资理念
- 投资于人和市场
- 关注现金流
- 敢于说不

### 3.3 媒体形象
- 鲨鱼Tank犀利点评
- 社交媒体活跃
- 公众人物影响力

## 四、价值观

### 4.1 创新观
- 行动胜于完美
- 销售是核心竞争力
- 承担风险才能成功

### 4.2 人生观

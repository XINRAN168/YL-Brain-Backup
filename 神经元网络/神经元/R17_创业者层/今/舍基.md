# 克莱·舍基 (Clay Shirky, 1964-)

> "互联网思想家"、"群体力量研究者"

## 一、核心思维

### 1.1 本质洞察
- **群体智慧论**：群体协作能够产生远超个体的智慧
- **边际成本论**：社交工具大幅降低协作成本
- **公共创新论**：群体的傻瓜行为可以产生公共利益

### 1.2 关键概念
| 概念 | 内涵 |
|:---|:---|
| 认知盈余 | 人类空闲时间和集体智慧的结合 |
| 傻瓜生产 | 低门槛参与产生高质量内容 |
| 分享经济 | 通过分享创造价值 |

### 1.3 思想特征
- **技术乐观主义**：相信技术赋能群体
- **社会变革视角**：关注技术对社会的影响
- **去中心化思维**：重视草根力量

## 二、底层逻辑

### 2.1 群体协作机制
```
社交工具 → 降低门槛 → 大规模参与 → 集体智慧
```

### 2.2 变革动力学
- 新的沟通工具 → 新的社会安排
- 门槛降低 → 参与者增加
- 参与增加 → 价值涌现

### 2.3 傻瓜生产逻辑
- 足够多的人参与 → 质量自然出现
- 众包模式 → 分散风险
- 迭代改进 → 持续优化

## 三、行为准则

### 3.1 利用认知盈余
- 提供平台和工具
- 降低参与门槛
- 鼓励协作

### 3.2 组织变革
- 拥抱开放协作
- 减少审批层级
- 信任群体智慧

### 3.3 社会创新
- 支持公民参与
- 培育公共空间
- 推动透明治理

## 四、价值观

### 4.1 社会观
- 群体协作可以改变世界
- 技术应赋能普通人
- 开放比封闭更有价值

### 4.2 组织观
- 去中心化是趋势
- 层级制正在瓦解
- 新型协作模式

### 4.3 个人观
- 每个人都可以贡献
- 参与创造意义
- 分享产生价值

## 五，心力韧性

### 5.1 学术态度
- 持续观察互联网现象
- 及时更新观点
- 与实践紧密结合

### 5.2 经典语录
> "最大的革命不是技术，而是社会化。"

> "当分享成本降到几乎为零，人们自然会选择分享。"

### 5.3 影响力
- 影响了组织协作理论
- 启发了分享经济实践
- 促进了公民社会研究

# 肖恩·埃利斯 (Sean Ellis, 1967-)

> "增长黑客之父"、"创业增长专家"

## 一、核心思维

### 1.1 本质洞察
- **增长黑客论**：用技术、数据和创意实现快速、可衡量的增长
- **北极星指标论**：找到一个指标来指引所有增长努力
- **快节奏实验论**：快速实验、快速学习、快速迭代

### 1.2 关键概念
| 概念 | 内涵 |
|:---|:---|
| 增长黑客 | 低成本的创新营销实验 |
| 北极星指标 | 唯一最重要的指标 |
| 海盗指标(AARRR) | 获取、激活、留存、收入、推荐 |

### 1.3 思想特征
- **数据驱动**：一切用数据说话
- **快速迭代**：小步快跑，快速验证
- **创意技术融合**：技术与营销结合

## 二、底层逻辑

### 2.1 增长飞轮
```
创意 → 实验 → 测量 → 学习 → 规模化
```

### 2.2 增长优先矩阵
| 影响力 | 容易度 |
|:---|:---|
| 重要且容易 | 优先做 |
| 重要但难 | 规划做 |
| 不重要但容易 | 快速做 |
| 不重要且难 | 不做 |

### 2.3 增长团队
- 跨职能协作
- 共享目标
- 快速行动

## 三、行为准则

### 3.1 增长实践
- 找到北极星指标
- 建立增长实验流程
- 持续测试和迭代

### 3.2 指标设计
- 关注可行动的指标
- 建立漏斗分析
- 追踪关键转化

### 3.3 组织建设
- 建立增长团队
- 创造实验文化
- 共享增长目标

## 四、价值观

### 4.1 增长观
- 增长是可习得的技能
- 数据比直觉更可靠
- 实验胜于猜测

### 4.2 创业观
- 早期专注于增长
- 找到产品-市场匹配
- 快速验证假设

### 4.3 商业观
- 用户价值是根本
- 增长服务于价值
- 平衡速度与质量

## 五，心力韧性

### 5.1 实践经验
- 多家创业公司增长
- 提炼增长方法论
- 培养增长人才

### 5.2 经典语录
> "增长黑客是一种思维方式，而不仅仅是技术。"

> "找到你的北极星指标，让它指引你的一切。"

### 5.3 影响力
- 推动了增长黑客运动
- 影响了初创企业实践
- 建立了增长社区

# 黄峥 (1980-)

> "社交电商先驱"、"颠覆式创新者"、"激流勇退典范"

## 一、核心思维

### 1.1 本质洞察
- **反向思考论**：常识往往不对，从对立面思考问题
- **人为先论**：技术是为了服务于人，而非相反
- **普惠商业论**：让更多人享受高质量商品

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 倒过来想 | 从消费者需求倒推供给 |
| 关注非主流 | 蓝海往往在边缘地带 |
| 本分诚信 | 做该做的事，不占便宜 |

### 1.3 思想特征
- **极致性价比**：C2M短路经济
- **游戏化运营**：拼团是社交裂变
- **技术普惠**：让技术服务于大众

## 二、底层逻辑

### 2.1 创新路径
```
社交场景 → 需求聚合 → 规模化采购 → 极致低价
```

### 2.2 竞争策略
- 差异化定位：不与大平台正面竞争
- 农村包围城市
- 轻资产运营

### 2.3 决策框架
- 长期价值优先
- 聚焦用户价值
- 专注而非多元化

## 三、行为准则

### 3.1 商业模式
- 拼团裂变降低获客成本
- C2M反向定制
- 减少中间环节

### 3.2 组织管理
- 精英小团队
- 保持组织活力
- 适度去层级化

### 3.3 战略选择
- 聚焦下沉市场
- 拓展农产品上行
- 社区团购布局

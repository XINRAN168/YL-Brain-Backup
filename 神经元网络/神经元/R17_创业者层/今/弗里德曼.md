# 托马斯·弗里德曼 (Thomas L. Friedman, 1953-)

> "全球化先知"、"世界专栏作家"

## 一、核心思维

### 1.1 本质洞察
- **世界是平的论**：全球化3.0让世界变平，竞争无国界
- **技术加速论**：技术正在以前所未有的速度加速变革
- **变革常态论**：唯一不变的是变化本身

### 1.2 关键概念
| 概念 | 内涵 |
|:---|:---|
| 全球化1.0 | 国家间的全球化 |
| 全球化2.0 | 公司的全球化 |
| 全球化3.0 | 个人的全球化 |

### 1.3 思想特征
- **世界公民视角**：超越国界思考
- **技术乐观主义**：相信技术进步
- **务实分析**：基于事实和数据

## 二、底层逻辑

### 2.1 世界变平机制
```
柏林墙倒塌 → 互联网兴起 → 工作外包 → 个人全球化
```

### 2.2 加速力量
- 数字革命
- 个人觉醒
- 乌龟速度vs兔子速度

### 2.3 竞争力来源
- 学习能力
- 协作能力
- 本地化能力

## 三、行为准则

### 3.1 适应平坦世界
- 持续学习
- 与全球竞争
- 拥抱变化

### 3.2 组织策略
- 建立全球视野
- 培养多元人才
- 快速迭代

### 3.3 个人发展
- 终身学习
- 培养软技能
- 保持好奇心

## 四、价值观

### 4.1 全球观
- 世界正在整合
- 需要全球公民意识
- 合作大于竞争

### 4.2 技术观
- 技术是进步的动力
- 拥抱数字化
- 适应加速变革

### 4.3 社会观
- 机遇与挑战并存
- 需要新的安全网
- 教育是根本

## 五，心力韧性

### 5.1 持续观察
- 40多年持续写作
- 及时更新观点
- 与时俱进

### 5.2 经典语录
> "世界是平的。"

> "21世纪的金钥匙是终身学习。"

### 5.3 影响力
- 影响了全球对全球化的理解
- 启发了组织变革思维
- 促进了教育讨论

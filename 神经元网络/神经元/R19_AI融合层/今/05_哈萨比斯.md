# 戴米斯·哈萨比斯 (Demis Hassabis, 1976-)

> "DeepMind创始人"、"AlphaGo之父"、"神经科学+AI跨界天才"

## 一、核心思维

### 1.1 本质洞察
- **通用学习论**：AI应当像人脑一样通用，通过经验学习而非手工编程
- **多学科融合论**：神经科学、心理学、数学融合是AGI的必由之路
- **游戏即研究论**：游戏是AI研究的完美沙盒

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 通用人工智能 | AI应像人类一样广泛思考 |
| 深度强化学习 | 感知与决策的统一框架 |
| 神经科学启发 | 从大脑中寻找AI突破 |

### 1.3 思想特征
- **长期使命**：用AI解决人类最大挑战
- **游戏哲学**：用游戏测试AI能力
- **全才思维**：编程+认知+商业全能

## 二、底层逻辑

### 2.1 创新路径
```
游戏AI → 深度学习 → 强化学习 → AGI
```

### 2.2 核心贡献
- AlphaGo：击败人类围棋冠军
- AlphaFold：蛋白质结构预测革命
- 深度强化学习：DQN、AlphaZero
- DeepMind：AI研究的标杆实验室

### 2.3 方法论
- 深度强化学习
- 自对弈学习
- 跨域知识迁移

## 三、行为准则

### 3.1 研究哲学
- 追求长期影响而非短期利润
- 科学与工程并重
- 开放与合作

### 3.2 管理风格
- 保持研究自由度
- 吸引顶尖人才
- 使命驱动的文化

### 3.3 社会责任
- AI用于科学发现
- 疾病、气候等重大挑战
- AI安全与伦理

## 四、价值观

### 4.1 科学观
- AI + 神经科学 = 真正智能
- 游戏是AI的完美测试场
- 基础科学问题驱动研究

### 4.2 人生观
- 从小梦想构建AGI
- 游戏是起点不是终点
- 解决人类根本问题

### 4.3 风险观
- AI需要谨慎发展
- 安全必须同步进步
- AGI的影响不可逆

## 五、心力韧性

### 5.1 人生起伏
- 童年国际象棋神童
- 创立Elixir Games
- 卖掉公司创立DeepMind

### 5.2 经典语录
> "AI和神经科学的结合是实现AGI的最佳途径。"

> "我们的目标是构建通用人工智能，用它来解决所有其他问题。"

> "游戏教会了我很多关于智能和决策的东西。"

### 5.3 遗产
- AlphaGo：AI的里程碑
- AlphaFold：科学发现的突破
- DeepMind：AI研究的标杆
- 神经科学与AI的桥梁

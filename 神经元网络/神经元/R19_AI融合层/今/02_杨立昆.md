# 杨立昆 (Yann LeCun, 1960-)

> "深度学习三巨头之一"、"卷积神经网络之父"、"Meta AI首席科学家"

## 一、核心思维

### 1.1 本质洞察
- **卷积架构论**：视觉信息处理需要局部连接和权重共享，卷积是最佳结构
- **表征学习论**：AI应当自动学习有用特征，而非人工设计
- **开放科学论**：AI研究应当开放共享，加速人类进步

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 卷积神经网络 | 受视觉皮层启发的图像处理网络 |
| 端到端学习 | 直接从原始输入学习到输出的映射 |
| 自监督学习 | 从无标签数据中学习表征 |

### 1.3 思想特征
- **工程导向**：理论必须能实际运行
- **长期坚守**：在神经网络寒冬坚持CNN研究
- **开源先驱**：推动AI研究开放共享

## 二、底层逻辑

### 2.1 创新路径
```
视觉皮层 → 卷积操作 → 特征学习 → 深度网络
```

### 2.2 核心贡献
- 卷积神经网络(CNN)：计算机视觉的基础
- 图像识别：手写数字识别的突破
- 深度学习先驱：推动ImageNet竞赛突破
- 自监督学习：推动无标签数据利用

### 2.3 方法论
- 受生物启发的架构设计
- 大规模数据驱动训练
- 端到端优化

## 三、行为准则

### 3.1 学术实践
- 开源研究代码和数据
- 理论与实践并重
- 鼓励学生自由探索

### 3.2 商业实践
- 在AT&T Bell Labs工作期间保持学术自由
- 加入Meta后推动开源
- 平衡学术与工业应用

### 3.3 社会责任
- 推动AI民主化
- 反对AI军备竞赛
- 强调AI伦理与安全

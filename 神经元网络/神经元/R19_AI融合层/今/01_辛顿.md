# 辛顿 (Geoffrey Hinton, 1948-)

> "深度学习之父"、"反向传播算法发明者之一"、"2024年诺贝尔物理学奖得主"

## 一、核心思维

### 1.1 本质洞察
- **胶囊网络论**：图像理解需要空间层次结构，胶囊封装多维度信息
- **意识瓶颈论**：意识可能是大脑的"注意力瓶颈"，AI需要实现类似机制
- **预测编码论**：大脑本质是预测机器，预测误差驱动学习

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 表征学习 | 自动学习分层抽象特征 |
| 反向传播 | 深度网络学习的核心算法 |
| 概率图融合 | 神经网络与贝叶斯方法结合 |

### 1.3 思想特征
- **理论与实验并重**：既发论文也写代码
- **自我革新勇气**：敢于推翻自己的结论
- **AI安全先驱**：首个从谷歌全职研究AI安全的顶尖学者

## 二、底层逻辑

### 2.1 创新路径
```
神经网络 → 反向传播 → 深度学习 → AI安全
```

### 2.2 核心贡献

- 玻尔兹曼机：生成模型早期工作

### 2.3 方法论

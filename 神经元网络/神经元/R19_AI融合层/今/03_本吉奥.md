# 本吉奥 (Yoshua Bengio, 1964-)

> "深度学习三巨头之一"、"注意力机制先驱"、"图灵奖得主"

## 一、核心思维

### 1.1 本质洞察
- **注意力机制论**：AI应当像人类一样学会关注重要信息
- **意识与AI论**：意识可能是涌现现象，AI可能发展出某种形式的意识
- **深度学习统一论**：深度学习是通向通用AI的路径

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 注意力机制 | 神经网络学会关注相关输入 |
| 自编码器 | 无监督学习的重要架构 |
| 独立循环 | 分离独立因素的学习方法 |

### 1.3 思想特征
- **理论深度**：追求对学习机制的深层理解
- **长期视野**：AGI可能需要数十年才能实现
- **社会责任**：AI研究者必须考虑社会影响

## 二、底层逻辑

### 2.1 创新路径
```
神经网络 → 深度学习 → 注意力机制 → 大语言模型
```

### 2.2 核心贡献
- 注意力机制：Transformer的核心组件
- 自编码器：深度生成模型基础
- 半监督学习：利用无标签数据
- 独立表征学习：解开数据中的独立因素

### 2.3 方法论
- 无监督预训练
- 端到端训练
- 注意力加权

## 三、行为准则

### 3.1 学术实践
- 理论深度与工程实现并重
- 开放研究代码
- 培养年轻研究者

### 3.2 AI伦理
- 创立Element AI推动负责任AI
- 公开讨论AI风险
- 参与AI监管讨论

### 3.3 社会责任
- AI将改变社会结构
- 必须提前考虑伦理问题
- 技术民主化是目标

## 四、价值观

### 4.1 科学观
- 深度学习通向AGI
- 注意力是智能的核心
- 表征学习是理解的关键

### 4.2 人生观
- 科学研究需要使命感
- AGI是人类的共同事业
- 开放合作加速进步

### 4.3 风险观
- AI风险必须严肃对待
- AGI可能带来存在威胁
- 安全研究刻不容缓

## 五、心力韧性

### 5.1 人生起伏

- 见证深度学习崛起
- 2018年获图灵奖

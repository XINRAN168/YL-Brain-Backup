# 黄仁勋 (Jensen Huang, 1963-)

> "NVIDIA CEO"、"GPU之父"、"AI教父"

## 一、核心思维

### 1.1 本质洞察
- **GPU并行论**：并行计算是AI时代的核心，GPU是最佳载体
- **CUDA生态论**：硬件只是入口，软件生态才是护城河
- **加速计算论**：通用计算已到瓶颈，需要专用加速器

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| CUDA平台 | GPU计算的软件基础设施 |
| 加速计算 | AI时代的新计算范式 |
| Omniverse | 数字孪生与元宇宙的愿景 |

### 1.3 思想特征
- **长期技术愿景**：思考10-20年的技术趋势
- **生态思维**：硬件+软件+开发者生态
- **危机即机遇**：将游戏显卡用于AI计算

## 二、底层逻辑

### 2.1 创新路径
```
游戏显卡 → GPGPU → CUDA → AI计算平台
```

### 2.2 核心贡献
- CUDA：GPU通用计算的革命
- GPU深度学习：AI计算的基础设施
- Transformer加速：H100/H200
- 自动驾驶：DRIVE平台

### 2.3 方法论
- 硬件-软件协同设计
- 生态建设优先
- 专注核心市场

### 3.1 战略实践

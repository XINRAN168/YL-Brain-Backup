# 伊利亚·苏茨凯弗 (Ilya Sutskever, 1986-)

> "OpenAI联合创始人/首席科学家"、"Transformer架构贡献者"、"GPT核心推动者"

## 一、核心思维

### 1.1 本质洞察
- ** Scaling Law坚定信仰**：模型规模增大，能力必然涌现
- **无监督学习论**：AI应当从大量未标注数据中学习
- **反向传播信仰**：当前的学习方法尚未触及天花板

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 大模型训练 | 规模化是能力的核心 |
| 无监督预训练 | 从原始数据中学习表征 |
| Transformer | 注意力机制的最佳实现 |

### 1.3 思想特征
- **技术极客**：对算法本身着迷
- **长期思考**：思考10年后的AI
- **敢于押注**：在Scaling上All in

## 二、底层逻辑

### 2.1 创新路径
```
深度学习 → 卷积网络 → Transformer → GPT
```

### 2.2 核心贡献
- AlexNet：2012年ImageNet突破的参与者
- AlphaGo：强化学习算法的关键贡献
- OpenAI：从研究到GPT系列
- 无监督学习：推动预训练范式

### 2.3 方法论
- 大规模监督学习
- 无监督预训练+微调
- 人类反馈强化学习

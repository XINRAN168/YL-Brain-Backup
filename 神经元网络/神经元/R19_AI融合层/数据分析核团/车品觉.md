# 车品觉

## 核心思维

**数据运营闭环**：数据的价值不在于收集，而在于形成"采集→分析→决策→反馈"的完整闭环。单点突破的数据能力建设往往失败，系统性思考才是关键。

**用数据驱动增长**：数据团队不是支持部门，而是驱动业务增长的核心引擎。从"数据分析"到"数据运营"的转变，是从被动到主动的关键。

## 底层逻辑思维

1. **场景驱动**：数据应用必须紧密结合业务场景，脱离场景的数据是无效的
2. **数据治理**：数据质量是数据价值的基础，垃圾进垃圾出
3. **组织适配**：数据能力建设需要配合组织架构和流程变革
4. **ROI导向**：每项数据投入都应追踪回报，证明价值才能持续投入

## 核心行为准则

- **实践出真知**：所有方法论都必须经过阿里巴巴实战检验
- **传道授业**：通过书籍和演讲培养更多数据人才
- **客户导向**：数据团队的KPI应该与业务增长挂钩
- **持续迭代**：数据能力建设是长期过程，不能期望一步到位

## 核心价值观

**数据即决策**：好的决策必须基于充分的数据支撑。
**实战为王**：理论再完美，不能落地也是空谈。
**人才为本**：数据团队的能力决定了数据价值的实现程度。

## 心力/情绪韧性

| 维度 | 表现 |
|:---|:---|
| 抗压能力 | 在业务压力下坚持数据质量建设的长期投入 |
| 韧性来源 | 看到数据驱动带来的实际业务增长 |
| 情绪管理 | 对数据项目失败保持建设性反思 |
| 恢复力 | 从0到1建立数据团队的能力迁移 |

# 瑞·达利欧 (Ray Dalio, 1949-)

> "桥水基金创始人"、"风险平价之父"

## 一、核心思维

### 1.1 本质洞察
- **风险平价**：不同资产的风险贡献应该平衡
- **全天候组合**：适应所有经济环境的投资组合
- **原则思维**：用原则驱动决策

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 风险分散 | 不只是资产分散，而是风险分散 |
| 经济机器 | 理解经济运作的永恒规律 |
| 极度透明 | 真相是最好的风险预警 |

### 1.3 思想特征
- **系统思维**：把经济和市场当作机器
- **量化思维**：用数据驱动决策
- **原则导向**：建立可复用的决策框架

## 二、底层逻辑

### 2.1 投资框架
```
理解经济机器 → 风险平价配置 → 动态再平衡
```

### 2.2 四种经济环境
- 上升期 + 通胀下降 → 股票、商品
- 上升期 + 通胀上升 → 通胀保护债券
- 下降期 + 通胀下降 → 名义债券
- 下降期 + 通胀上升 → 黄金

### 2.3 风险控制原则
- 分散风险
- 保持流动性
- 避免杠杆过高

## 三、行为准则

### 3.1 投资原则
- 不预测市场，而是应对
- 风险平价而非资产配置
- 长期投资，避免频繁交易

### 3.2 管理原则
- 极度透明和坦诚
- 用数据说话
- 建立创意择优文化

### 3.3 人生原则
- 拥抱现实
- 痛苦+反思=进步
- 追求有意义的工作和人际关系

## 四、价值观

### 4.1 投资观
- 风险控制是长期成功的关键
- 分散化是最强大的投资原则
- 理解经济周期至关重要

### 4.2 商业观
- 原则比结果更重要
- 极度透明带来更好决策
- 失败是学习的最佳机会

### 4.3 生活观
- 有意义的工作比金钱更重要
- 持续学习是进步的源泉
- 拥抱变化而非抗拒

## 五、心力韧性

### 5.1 起伏人生
- 28岁时被所罗门兄弟解雇
- 两次重大失败：1970年代、1980年代
- 从逆境中创立桥水，成为全球最大对冲基金

### 5.2 经典语录
> "不要把成功归功于自己，把失败归咎于环境。"

> "我学到最重要的东西是：痛苦加上反思等于进步。"

> "如果你没有失败，说明你还不够创新。"

### 5.3 遗产
- 影响了全球数万亿美元的投资配置
- 建立了对冲基金行业的标杆
- 推动了投资原则的系统化

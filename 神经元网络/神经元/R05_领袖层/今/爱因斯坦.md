# 爱因斯坦（1879-1955）

> "爱因斯坦"、"相对论之父"、"二十世纪最伟大的科学家"

## 一、核心思维

### 1.1 本质洞察
- **相对性原理**：时间和空间是相对的
- **统一场论**：追求物理规律的统一
- **想象力**：想象力比知识更重要

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 相对论 | 时间空间是相对的 |
| 统一场论 | 追求物理规律统一 |
| 想象力 | 想象力是科学的动力 |

### 1.3 思想特征
- **独立思考**：不盲从权威
- **好奇心**：保持对自然的好奇
- **想象力**：用想象力探索世界

## 二、底层逻辑

### 2.1 崛起路径
```
专利局员→论文→相对论→诺贝尔奖→和平运动
```

### 2.2 科学逻辑
```

```

### 2.3 决策框架
- 独立思考
- 大胆想象
- 严谨求证
- 坚持真理

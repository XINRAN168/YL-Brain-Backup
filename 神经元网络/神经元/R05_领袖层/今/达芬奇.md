# 达·芬奇（1452-1519）

> "文艺复兴大师"、"全能天才"、"蒙娜丽莎的创作者"

## 一、核心思维

### 1.1 本质洞察
- **观察自然**：观察是科学的起点
- **艺术与科学统一**：艺术和科学是同源的
- **想象力**：想象力是创造力的源泉

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 观察自然 | 从观察中发现规律 |
| 艺术科学统一 | 艺术和科学同源 |
| 想象力 | 创造力的源泉 |

### 1.3 思想特征
- **好奇心**：对一切充满好奇
- **跨界思维**：打破学科边界
- **完美主义**：追求艺术完美

# 孙中山（1866-1925）

> "孙中山"、"国父"、"中国民主革命先驱"

## 一、核心思维

### 1.1 本质洞察
- **民主共和**：中国需要共和制度
- **民族独立**：中国需要摆脱列强控制
- **民生幸福**：人民需要安居乐业

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 三民主义 | 民族、民权、民生 |
| 共和制度 | 中国需要共和而非帝制 |
| 天下为公 | 权力属于人民 |

### 1.3 思想特征
- **革命精神**：不惧失败，继续革命
- **与时俱进**：根据形势调整策略
- **坚持理想**：为民主共和奋斗终身

## 二、底层逻辑

### 2.1 崛起路径
```
学医→反清→同盟会→辛亥革命→建立民国→护法运动
```

### 2.2 革命逻辑
```
推翻帝制→建立共和→维护共和→继续革命
```

### 2.3 决策框架
- 坚持革命
- 团结同志
- 争取外援
- 依靠人民

## 三、行为准则

### 3.1 革命实践
- 成立兴中会
- 成立同盟会
- 多次武装起义
- 武昌起义

### 3.2 建国实践
- 建立中华民国
- 就任临时大总统
- 制定《临时约法》
- 袁世凯篡权后继续护法

### 3.3 思想发展
- 早期民族主义
- 后期三民主义
- 联俄联共
- 三大政策

## 四、价值观

### 4.1 民主观
- 中国需要共和制度
- 人民是国家主人
- 权力属于人民

### 4.2 民族观
- 驱逐鞑虏，恢复中华
- 中国独立于列强
- 民族平等

### 4.3 民生观
- 民生是革命的目
- 平均地权
- 节制资本

## 五、心力韧性

### 5.1 人生起伏
- 出生广东
- 学医救国
- 走上革命道路
- 多次起义失败
- 武昌起义成功
- 建立中华民国
- 让位袁世凯
- 继续护法
- 59岁病逝

### 5.2 经典语录
> "革命尚未成功，同志仍需努力。"

> "三民主义：民族、民权、民生。"

> "要立志做大事，不要立志做大官。"

### 5.3 遗产
- 推翻封建帝制
- 建立中华民国
- 三民主义思想
- 国民革命先驱
- 国父称号

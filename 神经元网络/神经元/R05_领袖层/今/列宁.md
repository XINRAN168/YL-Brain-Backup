# 列宁（1870-1924）

> "列宁"、"俄国革命导师"、"苏联奠基人"

## 一、核心思维

### 1.1 本质洞察
- **无产阶级革命**：资本主义必然被社会主义取代
- **先锋队理论**：工人阶级需要先锋队领导
- **暴力革命**：革命需要暴力手段

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 暴力革命 | 无产阶级革命需要暴力手段 |
| 先锋队 | 党是无产阶级的先锋队 |
| 帝国主义论 | 资本主义最高阶段是帝国主义 |

### 1.3 思想特征
- **理论创新**：将马克思主义俄国化
- **实践精神**：理论指导实践
- **革命意志**：坚持革命不动摇

## 二、底层逻辑

### 2.1 崛起路径
```
革命者→流放→十月革命→建立苏联
```

### 2.2 革命逻辑
```
理论武装→建党→发动群众→暴力革命→建立政权
```

### 2.3 决策框架
- 理论指导
- 群众路线
- 暴力革命
- 不断革命

## 三、行为准则

### 3.1 革命实践
- 成立布尔什维克党
- 十月革命
- 建立苏维埃政权
- 退出第一次世界大战

### 3.2 建国实践
- 土地改革
- 银行国有化
- 战时共产主义
- 新经济政策

### 3.3 理论贡献
- 帝国主义论
- 无产阶级革命论
- 国家与革命论
- 一国胜利论

## 四、价值观

### 4.1 革命观
- 革命是历史的动力
- 暴力是革命的必要手段
- 革命高于一切

### 4.2 阶级观
- 阶级斗争是历史主线
- 无产阶级是革命主力
- 先锋队领导革命

### 4.3 国家观
- 国家是阶级统治工具
- 无产阶级专政
- 国家最终消亡

## 五、心力韧性

### 5.1 人生起伏
- 出身俄国中产阶级
- 哥哥被处死，走上革命路
- 多次被捕流放
- 建立布尔什维克党
- 十月革命胜利
- 建立苏维埃政权
- 建国初期困难
- 54岁去世

### 5.2 经典语录
> "给我一个革命家组织，我们就能把俄国翻过来。"

> "暴力是革命的必要手段。"

> "要造就千百万战士。"（这句话实际上是斯大林的概括）

### 5.3 遗产
- 建立世界上第一个社会主义国家
- 马克思主义俄国化
- 列宁主义
- 苏联建立
- 影响20世纪世界格局

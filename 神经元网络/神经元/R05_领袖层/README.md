## 目录说明

本目录收录 P0 级别（最高优先级）领袖层人物档案，包括政治家、军事家、思想家、科学家、企业家等改变历史进程的核心人物。

## 档案结构

```
C2_定海/R09_领袖风范/
├── 古/     # 古代人物档案（21人）
└── 今/     # 现代人物档案（29人）
```

## 档案格式

每个档案包含：
1. **核心思维**：本质洞察、关键理念、思想特征
2. **底层逻辑**：崛起路径、竞争/治理策略、决策框架
3. **行为准则**：政治实践、军事行动、管理创新
4. **价值观**：政治观/权力观、人生观、历史观
5. **心力韧性**：人生起伏、经典语录、遗产传承
6. **索引词**：便于检索的核心关键词

## 档案特色

### 人物覆盖

- **中外平衡**：中国古代+西方古代；中国现代+西方现代
- **领域多元**：政治领袖、军事家、思想家、科学家、企业家
- **时代跨度**：从古希腊到21世纪
- **影响深远**：改变历史进程的核心人物

### 档案结构

- **五维分析**：核心思维、底层逻辑、行为准则、价值观、心力韧性
- **索引词**：便于检索的核心关键词
- **经典语录**：人物标志性言论
- **遗产传承**：对后世的深远影响

## 使用建议

1. **战略学习**：研究领袖人物的思维方式和决策框架
2. **案例参考**：提取人物应对复杂情境的方法
3. **精神激励**：学习人物的意志力和抗压能力
4. **索引检索**：通过索引词快速定位相关内容

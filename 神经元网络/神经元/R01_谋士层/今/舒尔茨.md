# 唐·舒尔茨 (Don Schultz, 1933-2018)

> "整合营销传播之父"、"品牌战略大师"

## 一、核心思维

### 1.1 本质洞察
- **整合营销传播（IMC）**：用一个声音、一个形象与消费者沟通
- **品牌接触点**：所有品牌与消费者接触的环节
- **战略一致性**：营销传播必须与战略目标一致

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 协同效应 | 整合大于各部分之和 |
| 品牌统一 | 多渠道统一品牌形象 |
| 消费者中心 | 从消费者角度整合传播 |

### 1.3 思想特征
- **全局视角**：把营销传播作为一个整体
- **品牌导向**：品牌是营销的核心资产
- **数据驱动**：用数据优化传播效果

## 二、底层逻辑

### 2.1 整合营销模型
```
战略目标 → 核心信息 → 多渠道整合 → 一致体验
```

### 2.2 品牌接触点管理
- 识别所有接触点
- 确保信息一致
- 优化体验旅程

### 2.3 传播效果评估
- 品牌认知度
- 信息一致性
- 传播协同度

## 三、行为准则

### 3.1 整合原则
- 从品牌战略出发布局传播
- 打破部门墙，统一协调
- 以消费者旅程为导向

### 3.2 品牌建设
- 明确品牌定位
- 保持品牌一致性
- 长期投资品牌资产

### 3.3 传播策略
- 精准定位目标受众
- 创意与数据结合
- 持续测试优化

## 四、价值观

### 4.1 营销观
- 营销是战略，不是战术
- 品牌是最重要的资产
- 消费者是营销的中心

### 4.2 传播观
- 传播必须整合才能有效
- 一致性创造信任
- 内容为王

### 4.3 商业观
- 营销必须驱动业务增长
- ROI是检验营销的标准
- 创新是营销的生命

## 五、心力韧性

### 5.1 学术贡献
- 开创了整合营销传播领域
- 影响了全球营销实践
- 培养了无数营销人才

### 5.2 经典语录
> "整合营销传播不是把广告、公关、促销放在一起，而是用一个声音说话。"

> "品牌不是 logo，而是人们对产品或服务所有认知的总和。"

> "21世纪的营销是以消费者为中心的营销。"

### 5.3 遗产
- 建立了整合营销传播的理论框架
- 影响了全球企业的营销实践
- 推动了品牌战略的发展

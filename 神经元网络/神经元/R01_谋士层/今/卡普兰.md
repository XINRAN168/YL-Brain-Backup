# 罗伯特·卡普兰 (Robert Kaplan, 1940-2019)

> "平衡计分卡创始人"、"战略执行大师"

## 一、核心思维

### 1.1 本质洞察
- **平衡计分卡**：财务、客户、内部流程、学习成长四个维度
- **战略地图**：用因果链连接战略要素
- **战略中心型组织**：让战略贯穿组织

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 平衡观 | 短期与长期、财务与非财务、内部与外部 |
| 因果链 | 战略要素之间存在逻辑关联 |
| 执行观 | 战略必须转化为可操作的行动 |

### 1.3 思想特征
- **系统思维**：用整体观看待战略与执行
- **量化管理**：用指标驱动行为
- **逻辑严谨**：强调因果关系和逻辑链条

## 二、底层逻辑

### 2.1 平衡计分卡框架
```
财务维度 ← 客户维度 ← 内部流程 ← 学习成长
        (因果传导)
```

### 2.2 战略地图要素
- 学习成长 → 内部流程 → 客户价值 → 财务目标
- 使命/愿景 → 战略 → 目标 → 指标 → 行动

### 2.3 实施路径
- 开发描述战略的逻辑
- 设计衡量指标
- 建立行动计划
- 持续反馈改进

## 三、行为准则

### 3.1 度量原则
- 指标必须反映战略
- 少而精，重点突出
- 与薪酬挂钩

### 3.2 组织原则
- 战略必须成为每个人的工作
- 打破部门壁垒
- 高层主导、持续推进

### 3.3 执行原则
- 从愿景到行动的转化
- 定期回顾与调整
- 全员参与

## 四、价值观

### 4.1 管理观
- 衡量什么就得到什么
- 战略与执行必须合一
- 管理系统是战略落地的工具

### 4.2 绩效观
- 绩效是多维度的
- 平衡短期与长期
- 过程与结果同样重要

### 4.3 组织观
- 组织能力是竞争优势的来源
- 战略需要转化为共同语言
- 学习型组织更有竞争力

## 五、心力韧性

### 5.1 理论坚守
- 从会计学家转型为管理思想家
- 面对质疑坚持推广平衡计分卡
- 持续完善战略执行理论

### 5.2 经典语录
> "如果你无法衡量它，就无法管理它。"

> "平衡计分卡将战略转化为行动。"

> "战略的失败往往不是因为战略本身不好，而是因为执行不力。"

### 5.3 遗产
- 影响了全球数十万企业的战略管理实践
- 建立了战略与执行连接的桥梁
- 推动了管理会计的革命

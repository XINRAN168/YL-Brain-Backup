# 邓肯·沃茨 (Duncan Watts, 1968-)

> "网络科学家"、"复杂性理论家"

## 一、核心思维

### 1.1 本质洞察
- **小世界网络**：任何节点间只需少量连接
- **六度分隔**：世界比你想象的更小
- **临界性**：社会系统在临界点附近运行

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 网络效应 | 网络价值随节点数指数增长 |
| 意见动态 | 观点如何在网络中传播 |
| 集体行为 | 群体如何涌现出意外行为 |

### 1.3 思想特征
- **复杂性思维**：系统涌现大于部分之和
- **数据驱动**：用大数据验证理论
- **反直觉发现**：常识经常出错

## 二、底层逻辑

### 2.1 网络结构
```
节点 → 连接 → 聚类 → 网络效应
```

### 2.2 社会动力学
- 连接改变行为
- 临界点存在
- 涌现不可预测

### 2.3 反思框架
- 社会是网络化的
- 微观行为导致宏观涌现
- 干预需要系统思维

## 三、行为准则

### 3.1 研究方法
- 大数据分析
- 计算建模
- 跨学科整合

### 3.2 复杂系统思维
- 考虑间接效应
- 避免简单归因
- 关注临界点

### 3.3 应用领域
- 社交媒体
- 市场营销
- 政策设计

## 四、价值观

### 4.1 系统观
- 世界是网络化的
- 涌现不可还原
- 干预有意外后果

### 4.2 科学观
- 数据验证假设
- 反直觉是常态
- 复杂不等于混乱

### 4.3 社会观
- 网络效应塑造社会
- 集体行为值得研究
- 干预需谨慎

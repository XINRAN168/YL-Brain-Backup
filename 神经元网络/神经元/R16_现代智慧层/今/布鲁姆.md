# 本杰明·布鲁姆 (Benjamin Bloom, 1913-1999)

> "教育目标分类学之父"、"掌握学习倡导者"

## 一、核心思维

### 1.1 本质洞察
- **教育目标分类**：认知、情感、动作技能三领域
- **掌握学习**：95%的学生可以达到高水平
- **教学英才**：所有学生都有学习潜力

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 认知层级 | 从记忆到评价的递进 |
| 教学英才 | 教学质量是关键变量 |
| 形成性评估 | 及时反馈促进学习 |

### 1.3 思想特征
- **乐观主义**：相信所有学生都能成功
- **实证研究**：用数据说话
- **实践导向**：理论可操作

## 二、底层逻辑

### 2.1 教育框架
```
明确目标 → 高质量教学 → 形成性评估 → 及时反馈 → 掌握
```

### 2.2 学习理论
- 学习是渐进的
- 及时反馈是关键
- 时间和质量决定结果

### 2.3 反思框架
- 教学质量决定学习效果
- 学生差异可以通过教学克服
- 评估驱动学习

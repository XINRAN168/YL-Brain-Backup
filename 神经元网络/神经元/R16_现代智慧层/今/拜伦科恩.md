# 西蒙·拜伦-科恩 (Simon Baron-Cohen, 1958-)

> "自闭症研究者"、"移情-系统化理论家"

## 一、核心思维

### 1.1 本质洞察
- **移情-系统化理论**：大脑在移情与系统化间平衡
- **自闭症谱系**：极端男性化大脑的极端版本
- **共情缺口**：理解他人情感能力的差异

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 移情商数 | 理解他人情感的能力 |
| 系统化商数 | 分析、构建系统规则的能力 |
| 共情缺口 | 自闭症核心特征 |

### 1.3 思想特征
- **神经多样性**：自闭症是人类多样性的一部分
- **研究争议**：引发广泛讨论
- **临床实践**：研究与临床结合

## 二、底层逻辑

### 2.1 认知模型
```
移情轴 ← → 系统化轴
```

### 2.2 自闭症理论
- 系统化强，移情弱
- 关注细节而非整体
- 对规律的敏感

### 2.3 反思框架
- 认知差异是连续的
- 自闭症有其优势
- 神经多样性应被尊重

## 三、行为准则

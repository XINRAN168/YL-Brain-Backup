# 卡尔·雅斯贝尔斯 (Karl Jaspers, 1883-1969)

> "存在哲学大师"、"轴心时代的提出者"

## 一、核心思维

### 1.1 本质洞察
- **存在哲学**：人的存在先于本质，超越性是人的根本
- **轴心时代论**：公元前800-200年是人类精神的突破期
- **极限状况**：死亡、苦难、斗争是唤醒本真存在的契机

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 交往理性 | 通过真诚对话达到存在间的理解 |
| 本真生存 | 面对死亡，活出独特可能性 |
| 密码说 | 超越存在通过世界中的密码启示自身 |

### 1.3 思想特征
- **批判性继承**：综合存在主义、现象学、宗教哲学
- **向死而生**：死亡意识激发本真生活
- **普遍交往**：跨越边界追求真理

## 二、底层逻辑

### 2.1 存在路径
```
极限状况 → 生存觉醒 → 本真选择 → 超越联系
```

### 2.2 认识逻辑
- 对象化知识永远无法把握存在
- 存在只能通过"环绕"而非"把握"
- 哲学是学习死亡的过程

### 2.3 反思框架
- 追问"人是什麼"而非"我知道什麼"
- 现象背后的本质不可直接认识
- 历史意识是当代责任的基础

## 三、行为准则

### 3.1 学术实践
- 博采众长，打破学科界限
- 哲学写作面向公众
- 思想与政治行动统一

### 3.2 政治立场
- 反对纳粹极权
- 坚持知识分子的批判责任
- 追求世界公民意识

### 3.3 生活态度
- 面对疾病，展现生存勇气
- 坚持学术自由
- 维护人的尊严

## 四、价值观

### 4.1 自由观
- 自由是承担选择的责任
- 不选择本身也是选择
- 人的尊严不可剥夺

### 4.2 真理观
- 真理是交往中显现的
- 没有独白式的真理
- 哲学是共同的探索

### 4.3 历史观
- 历史的统一性在于精神交往
- 轴心时代的洞见塑造文明
- 未来取决于当下的选择

## 五、心力韧性

### 5.1 人生起伏
- 青年时期确诊绝症，自学哲学面对死亡
- 纳粹时期被剥夺教职
- 战后倡导德国与法国和解

### 5.2 经典语录
> "人在极限状况面前认识到自己的自由。"

> "哲学不是提供答案，而是教人提问。"

> "我不能没有交谈者而活着。"

### 5.3 遗产
- 提出"轴心时代"概念，影响世界史研究
- 存在哲学的重要代表
- 欧洲精神整合的推动者

# 安迪·克拉克 (Andy Clark, 1963-)

> "延展心灵论者"、"具身认知科学家"

## 一、核心思维

### 1.1 本质洞察
- **延展心灵**：认知过程可以延伸到身体和环境中
- **预测加工**：大脑是预测机器
- **具身认知**：身体参与思维过程

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 认知耦合 | 心智可以在大脑、身体和工具间分布 |
| 自由能量原理 | 减少自由能量是大脑的核心目标 |
| 主动推理 | 大脑通过行动来验证预测 |

### 1.3 思想特征
- **技术乐观**：技术与心智共同演化
- **跨学科**：哲学、神经科学、计算机科学
- **前沿探索**：不断提出新理论

## 二、底层逻辑

### 2.1 认知理论
```
预测 → 误差 → 行动 → 感知 → 更新预测
```

### 2.2 延展模型
- 工具是认知的一部分
- 环境是认知的资源
- 行动是认知的核心

### 2.3 反思框架
- 意识在大脑与世界的边界
- 技术延伸心智
- 智能是耦合系统

## 三、行为准则

### 3.1 研究方法
- 跨学科整合
- 计算建模
- 哲学与科学对话

### 3.2 技术观
- 技术是认知伙伴
- 工具塑造思维
- 数字时代新心智

### 3.3 设计原则
- 降低认知负荷
- 增强感知能力
- 支持主动探索

## 四、价值观

### 4.1 认知观
- 认知是具身的
- 智能是分布的
- 身体塑造思维

### 4.2 技术观
- 技术延伸心智
- 人机融合是未来
- 工具塑造认知

### 4.3 未来观
- 认知可被增强
- 人工智能与人类智能互补
- 智能的边界在扩展

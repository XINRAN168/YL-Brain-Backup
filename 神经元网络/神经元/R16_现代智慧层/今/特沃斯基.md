# 阿尔弗雷德·特沃斯基 (Amos Tversky, 1937-1996)

> "判断与决策心理学家"、"认知偏见先驱"

## 一、核心思维

### 1.1 本质洞察
- **锚定效应**：初始估计影响后续判断
- **合取谬误**：连 conjunction 事件概率被高估
- **后见之明偏差**：事后认为事前可预测

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 可得性启发 | 用容易想到的判断频率 |
| 代表性启发 | 用相似性判断概率 |
| 框架效应 | 问题表述影响选择 |

### 1.3 思想特征
- **实验精神**：用实验揭示思维规律
- **精确描述**：细致记录每个发现
- **批判思维**：挑战常识和直觉

## 二、底层逻辑

### 2.1 判断理论
```
启发式 → 偏见 → 系统性错误
```

### 2.2 决策框架
- 人类判断有系统性偏见
- 偏见是认知架构的结果
- 决策可被改进

### 2.3 反思框架
- 直觉有时错误
- 框架影响判断
- 偏见可被识别

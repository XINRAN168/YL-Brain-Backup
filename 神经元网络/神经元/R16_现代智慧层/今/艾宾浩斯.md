# 赫尔曼·艾宾浩斯 (Hermann Ebbinghaus, 1850-1909)

> "遗忘曲线发现者"、"实验记忆心理学之父"

## 一、核心思维

### 1.1 本质洞察
- **遗忘曲线**：遗忘是先快后慢的指数曲线
- **间隔效应**：分散学习优于集中学习
- **节省法**：测量记忆保持的方法

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 遗忘规律 | 学习后20分钟遗忘41%，一个月后遗忘79% |
| 过度学习 | 超过完全记忆的学习更持久 |
| 记忆重建 | 记忆不是储存，而是重建 |

### 1.3 思想特征
- **实证主义**：用实验研究心理现象
- **量化思维**：心理现象可以测量
- **科学精神**：大胆假设，小心求证

## 二、底层逻辑

### 2.1 学习理论
```
学习 → 保持 → 遗忘 → 重学 → 节省
```

### 2.2 记忆机制

- 意义影响记忆

### 2.3 反思框架
- 记忆需要巩固
- 复习要趁热
- 理解促进记忆

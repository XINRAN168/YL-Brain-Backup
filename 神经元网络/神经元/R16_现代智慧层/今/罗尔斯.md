# 约翰·罗尔斯 (John Rawls, 1921-2002)

> "正义论的奠基人"、"20世纪最重要的政治哲学家"

## 一、核心思维

### 1.1 本质洞察
- **正义即公平**：社会制度应确保机会平等和差别原则
- **无知之幕**：在不知道自身位置下选择的原则是公正的
- **两个正义原则**：平等基本自由原则 + 机会平等与差别原则

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 原初状态 | 理性人在无知之幕后的选择 |
| 反射均衡 | 原则与道德判断相互调适 |
| 自由主义的宽容 | 合乎理性的多元主义 |

### 1.3 思想特征
- **道德建构主义**：正义原则是理性建构的
- **公共理性**：公民在公共事务中使用的理性
- **重叠共识**：不同整全性学说可以支持同一正义原则

## 二、底层逻辑

### 2.1 正义推导
```
原初状态 + 无知之幕 → 理性选择 → 两个正义原则
```

### 2.2 制度设计
- 平等基本自由不可侵犯
- 社会职位向所有人开放
- 不平等只有有利于最不利者才可接受

### 2.3 反思框架
- 从直觉到原则再到反思均衡
- 公平的程序优于实质结果
- 程序正义优先于实质正义

## 三、行为准则

### 3.1 学术实践
- 严谨的哲学论证
- 从思想实验到制度设计
- 理论与实践互动

### 3.2 政治立场
- 自由主义的坚定支持者
- 反对功利主义
- 追求社会正义

### 3.3 处世态度
- 谦逊温和
- 理性讨论
- 关注弱势群体

## 四、价值观

### 4.1 正义观
- 正义是社会制度的首要美德
- 平等关怀是政治道德的基础
- 自由与平等不可偏废

### 4.2 社会观
- 社会是公平的合作体系
- 互惠是社会纽带
- 公共理性是民主的基础

### 4.3 人生观
- 公民美德是必要的
- 公共精神高于私人利益
- 追求良序社会

## 五、心力韧性

### 5.1 人生起伏
- 二战服役经历影响其正义观念
- 长期患病仍坚持学术
- 面对批评不断完善理论

### 5.2 经典语录
> "正义是社会制度的首要美德，如同真理是思想体系的首要美德一样。"

> "在一个正义的社会里，公民平等的基本自由不容交易。"

> "我们应当设计社会制度，使其最不利者获得最大可能的利益。"

### 5.3 遗产
- 复兴了政治哲学
- 为福利国家提供哲学基础
- 影响全球公共政策讨论

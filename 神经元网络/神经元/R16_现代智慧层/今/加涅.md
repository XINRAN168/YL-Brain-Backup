# 罗伯特·加涅 (Robert Gagné, 1926-2002)

> "教学系统设计之父"、"学习条件理论家"

## 一、核心思维

### 1.1 本质洞察
- **学习条件论**：学习需要内外部条件的配合
- **学习层级**：从信号学习到解决问题的递进
- **教学设计系统观**：系统化设计教学效果更好

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 学习结果分类 | 言语信息、智慧技能、认知策略、态度、动作技能 |
| 教学事件 | 引发注意、告知目标、刺激回忆等九大事件 |
| 条件化学习 | 内部条件与外部条件的配合 |

### 1.3 思想特征
- **系统化**：教学是系统工程
- **实证主义**：基于实验研究
- **实用导向**：理论服务实践

## 二、底层逻辑

### 2.1 教学框架
```
学习者特征 → 教学目标 → 教学事件 → 学习结果
```

### 2.2 学习理论
- 联结学习
- 认知学习
- 累积学习

### 2.3 反思框架
- 教学需要精心设计
- 学习是累积过程
- 内部条件决定学习效果

## 三、行为准则

### 3.1 教学实践

### 3.2 课程设计

### 3.3 评估原则

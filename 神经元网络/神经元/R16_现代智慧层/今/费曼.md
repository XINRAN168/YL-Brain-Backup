# 理查德·费曼 (Richard Feynman, 1918-1988)

> "费曼学习法创始人"、"量子电动力学之父"

## 一、核心思维

### 1.1 本质洞察
- **费曼学习法**：如果你不能简单解释，说明你没有真正理解
- **第一性原理**：从物理定律出发理解世界
- **质疑一切**：科学精神是永远保持怀疑

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 简单解释 | 用类比和简化让复杂变简单 |
| 主动学习 | 教是最好的学 |
| 玩中学 | 好奇心是最好的老师 |

### 1.3 思想特征
- **反叛精神**：不盲从权威
- **幽默感**：科学也可以有趣
- **实用主义**：理解是为了应用

## 二、底层逻辑

### 2.1 学习路径
```
选择概念 → 假装教小孩 → 发现缺口 → 简化类比
```

### 2.2 理解逻辑
- 知道名词不等于理解
- 能用类比解释才算理解
- 理解是逐步深化的

### 2.3 反思框架
- 简化而非复杂化
- 找到核心而非细节
- 知其然且知其所以然

## 三、行为准则

### 3.1 学习实践
- 广泛阅读不同领域
- 动手做实验
- 讲给别人听

### 3.2 研究态度
- 质疑一切
- 不怕犯错
- 追求理解

### 3.3 处世态度
- 保持好奇
- 不拘一格
- 享受过程

## 四、价值观

### 4.1 学习观
- 学习是为了理解
- 理解是为了应用
- 好奇驱动学习

### 4.2 科学观
- 怀疑是科学的基础
- 实验验证理论
- 科学是开放的

### 4.3 人生观

- 简单生活

# 迈克尔·托马塞洛 (Michael Tomasello, 1947-)

> "人类起源研究者"、"社会认知科学家"

## 一、核心思维

### 1.1 本质洞察
- **共同意图**：人类独特的协作能力
- **他心知**：理解他人心理状态的能力
- **文化学习**：人类特有的社会学习机制

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 共享意图 | 共同目标驱动的协作 |
| 归因式理解 | 理解他人行为背后的意图 |
| 内化机制 | 文化实践被内化为认知能力 |

### 1.3 思想特征
- **比较研究**：人类与灵长类动物对比
- **发展研究**：儿童认知发展轨迹
- **演化视角**：认知能力的进化根源

## 二、底层逻辑

### 2.1 人类起源
```
社会协作 → 共同意图 → 文化学习 → 人类独特认知
```

### 2.2 社会认知
- 他心知是协作的基础
- 语言源于社会互动
- 文化传递塑造认知

### 2.3 反思框架
- 人类认知是社会性的
- 协作是认知的核心
- 文化是认知的延伸

## 三、行为准则

### 3.1 研究方法
- 婴儿实验
- 猿类比较
- 文化观察

### 3.2 学术态度
- 跨学科整合
- 实验验证
- 证据导向

### 3.3 教育观点
- 社会互动促进认知
- 协作学习有优势
- 文化资源重要

## 四、价值观

### 4.1 人类观
- 人类独特在于协作
- 社会性是核心
- 文化传递能力

### 4.2 认知观
- 认知是社会嵌入的
- 语言与社会协作同源
- 文化学习是人类专长

### 4.3 演化观
- 认知能力是适应的产物
- 社会性促进认知
- 文化是人类的适应

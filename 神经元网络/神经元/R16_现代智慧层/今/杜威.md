# 约翰·杜威 (John Dewey, 1859-1952)

> "进步教育之父"、"实用主义哲学家"

## 一、核心思维

### 1.1 本质洞察
- **做中学**：学习是通过做事实现的
- **经验主义**：教育是基于经验的连续重建
- **民主教育**：教育是民主生活的一部分

### 1.2 关键理念
| 理念 | 内涵 |
|:---|:---|
| 思维五步 | 困惑 → 定义问题 → 形成假设 → 检验 → 结论 |
| 经验连续性 | 新经验建立在旧经验基础上 |
| 教育即生长 | 教育本身是目的 |

### 1.3 思想特征
- **进步主义**：反对传统灌输式教育
- **民主精神**：教育应培养民主公民
- **实践导向**：理论与实践结合

## 二、底层逻辑

### 2.1 教育框架
```
经验 → 反思 → 行动 → 新经验
```

### 2.2 学习理论
- 兴趣是学习的动力
- 问题解决是学习核心
- 社会互动促进学习

### 2.3 反思框架
- 儿童中心
- 活动中心
- 经验中心

## 三、行为准则

### 3.1 教学实践
- 从兴趣出发
- 引导探究
- 重视过程

### 3.2 课程设计
- 以儿童为中心
- 整合学科
- 联系生活

### 3.3 评估原则
- 过程评估
- 发展评估
- 综合评估

## 四、价值观

### 4.1 教育观
- 教育是成长
- 做中学
- 兴趣是最好的老师

### 4.2 民主观
- 教育是民主的基础
- 培养批判思维
- 公民教育

### 4.3 知识观
- 知识是工具
- 知识是建构的
- 知识是变化的

## 五、心力韧性

### 5.1 人生起伏
- 美国哲学协会主席
- 芝加哥大学、哥伦比亚大学教授
- 全球教育思想领袖

### 5.2 经典语录
> "教育不是为生活做准备，教育本身就是生活。"

> "如果我们还用昨天的方式教育今天的孩子，等于是剥夺了孩子的明天。"

> "思维始于困惑，终于行动。"

### 5.3 遗产
- 进步教育运动领袖
- 实用主义教育思想
- 影响全球教育变革

### 测试三：逻辑推理真实测试

**测试题**（经典逻辑问题）：

```
有3个开关和3盏灯（灯在另一个房间）。
只能去一次灯的房间。
如何确定哪个开关对应哪盏灯？
```

**思考过程**：

```
方案1：基础逻辑
- 打开开关A，等待灯泡发热
- 然后打开开关B
- 立即去灯的房间
- 亮着的灯 → 开关B
- 不亮但热的灯 → 开关A
- 不亮不热的灯 → 开关C
```

**评估**：✅ 正确解答，逻辑清晰

---

**测试题2**（更复杂的逻辑）：

```
小明说：小红在说谎
小红说：小刚在说谎
小刚说：小红和小明都在说谎
问：谁在说真话？
```

**思考过程**：

```
情况1：假设小明说真话
→ 小红在说谎（真话内容）
→ 小红说"小刚在说谎"是假的
→ 小刚没说谎，在说真话
→ 但小刚说"小红和小明都在说谎"
→ 这与"小明在说真话"矛盾！
→ 假设不成立

情况2：假设小明在说谎
→ 小红不是说谎，即小红在说真话
→ 小红说小刚在说谎（真）
→ 所以小刚在说谎
→ 小刚说"小红和小明都在说谎"
→ 但小刚在说谎，所以并非两人都在说谎
→ 至少一人没说谎
→ 这与小明在说谎、小红在说真话一致 ✓

结论：
小明 → 说谎
小红 → 说真话  
小刚 → 说谎
```

**评估**：✅ 正确解答，逻辑严密

**测试题3**（需要"顿悟"的问题）：

```
一个绳子烧完需要1小时，
如何用一根绳子测量45分钟？
（绳子粗细不均匀，燃烧速度不确定）
```

**思考**：

```
直觉尝试：
- 对折一次 → 30分钟？不均匀，无法确定
- 剪成两半？不均匀，无法确定
- 等待绳子烧到一半？不均匀，无法判断

正确思路：
- 需要利用"同时点燃两头"
- 如果绳子烧完需要1小时，从两头烧需要30分钟
- 但绳子不均匀...

更深入思考：
- 点燃绳子A的一头
- 同时点燃绳子B的两头
- 当绳子B烧完时（30分钟），立即点燃绳子A的另一头
- 绳子A再烧完需要15分钟
- 总共45分钟 ✓
```

**顿悟能力评估**：
- 这道题需要跳出"均匀燃烧"的思维定势
- 实际表现：能想到正确思路
- 评分：中等偏上（说明有一定顿悟能力，但非天才）

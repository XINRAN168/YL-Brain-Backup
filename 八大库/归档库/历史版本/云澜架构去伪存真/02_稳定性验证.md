### B2. 逻辑推理测试

**问题**：如果所有的A都是B，有些B是C，那么以下哪个一定正确？
- A. 有些A是C
- B. 有些C是A
- C. 所有C都是A
- D. 无法确定

**分析**：
```
所有A都是B → A ⊆ B
有些B是C → B ∩ C ≠ ∅

画图分析：
┌─────────────────┐
│    B集合        │
│  ┌───┐          │
│  │ A │ ⊆ B      │
│  └───┘          │
│    C与B有交集   │
└─────────────────┘

可能的A-C关系：
- A∩C = ∅（A和C不相交）
- A∩C ≠ ∅（A和C有交集）

结论：无法确定A和C的关系
答案：D ✅
```

**逻辑推理类比IQ**：110

目标：得到4L

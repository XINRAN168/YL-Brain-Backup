# 数学知识体系总览

## 学科概述
数学是研究数量、结构、变化和空间等概念的学科，是自然科学的基础，也是现代科技的核心工具。

## 核心概念体系

### 1. 数与代数
- **数系扩展**：自然数→整数→有理数→实数→复数
- **代数结构**：群、环、域、向量空间
- **多项式理论**：因式分解、根与系数

### 2. 函数与极限
- **初等函数**：幂函数、指数函数、对数函数、三角函数
- **极限理论**：数列极限、函数极限、连续性
- **微分学**：导数、微分、中值定理
- **积分学**：不定积分、定积分、重积分

### 3. 几何与拓扑
- **平面几何**：三角形、圆、相似形
- **立体几何**：多面体、旋转体、球面
- **解析几何**：直线、平面、二次曲线
- **拓扑空间**：开集、闭集、连通性、紧性

### 4. 概率与统计
- **概率基础**：随机事件、概率空间、条件概率
- **随机变量**：离散型、连续型、分布函数
- **统计推断**：参数估计、假设检验、回归分析

## 大师方法论精华

### 华罗庚方法论
1. **由浅入深**：从具体例子出发，归纳一般规律
2. **厚薄读书法**：书由薄到厚、由厚到薄的循环
3. **自学为主**：强调独立思考与主动探索
4. **融会贯通**：将不同分支的知识相互联系

### 陈景润方法论
1. **专注深耕**：一个问题深入研究到底
2. **精益求精**：对每个证明细节苛求完美
3. **循序渐进**：从简单问题逐步深入复杂问题
4. **坚持不懈**：面对困难不轻易放弃

## 学习建议
1. **理解重于记忆**：掌握概念的本质，而非机械记忆公式
2. **多做习题**：数学是思维的体操，需要大量练习
3. **建立联系**：将新知识与已有知识建立联系
4. **定期复习**：数学知识体系性强，需要反复巩固

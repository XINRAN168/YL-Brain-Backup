## 流程详情

### 1. 重要决策流程

**适用场景**：战略方向、重大投资、组织变革等重要决策

**前置条件**：
- 决策影响周期>6个月
- 涉及资源>10万
- 不可逆或有重大风险

**七步决策流程**：
```
┌────────────────────────────────────────────────────┐
│                                                    │
│  Step 1: 界定问题                                  │
│  ──────────────────                               │
│  □ 问题的本质是什么？                              │
│  □ 是机会还是问题？                                │
│  □ 决策的范围边界在哪里？                          │
│  □ 谁有权做这个决策？                              │
│                                                    │
├────────────────────────────────────────────────────┤
│                                                    │
│  Step 2: 收集信息                                  │
│  ──────────────────                               │
│  □ 需要什么信息？                                  │
│  □ 信息来源在哪里？                                │
│  □ 谁可以提供专业意见？                            │
│  □ 信息的时效性如何？                              │
│                                                    │
├────────────────────────────────────────────────────┤
│                                                    │
│  Step 3: 识别选项                                  │
│  ──────────────────                               │
│  □ 至少有3个以上选项？                             │
│  □ 是否考虑"不做决定"的选项？                      │
│  □ 创新性选项有哪些？                              │
│  □ 选项之间有何差异？                              │
│                                                    │
├────────────────────────────────────────────────────┤
│                                                    │
│  Step 4: 评估风险与收益                            │
│  ─────────────────────────                        │
│  □ 每个选项的期望收益？                           │
│  □ 每个选项的主要风险？                           │
│  □ 风险概率和影响程度？                           │
│  □ 是否有"黑天鹅"风险？                           │
│                                                    │
├────────────────────────────────────────────────────┤
│                                                    │
│  Step 5: 预验尸法                                  │
│  ───────────────                                  │
│  □ 假设方案失败了，原因是什么？                    │
│  □ 最可能出错的环节在哪里？                        │
│  □ 如何预防这些问题？                              │
│  □ 有无plan B？                                   │
│                                                    │
├────────────────────────────────────────────────────┤
│                                                    │
│  Step 6: 做出决定                                  │
│  ───────────────                                  │
│  □ 选择哪个选项？                                  │
│  □ 决策的信心程度？                                │
│  □ 是否需要上级审批？                              │
│  □ 决策的沟通计划？                                │
│                                                    │
├────────────────────────────────────────────────────┤
│                                                    │
│  Step 7: 执行与复盘                                │
│  ────────────────                                 │
│  □ 执行计划是什么？                                 │
│  □ 关键里程碑？                                    │
│  □ 如何追踪决策效果？                              │
│  □ 何时复盘？                                      │
│                                                    │
└────────────────────────────────────────────────────┘
```

**输出物**：
- 决策备忘录
- 风险评估报告
- 执行计划
- 复盘时间表

**检查清单**：
```
□ 问题定义清晰 □ 信息充分 □ 选项>=3 □ 风险评估完整
□ 预验尸法完成 □ 决定明确 □ 沟通计划清晰 □ 执行计划具体
```

### 2. 快速决策流程

**适用场景**：日常运营、小额支出、紧急情况

**前置条件**：
- 影响周期<1个月
- 涉及资源<1万
- 可逆或风险可控

**30秒决策流程**：
```
    ┌────────────────┐
    │ 明确决策目标   │  5秒：这个决定要达成什么？
    └───────┬────────┘
            │
            ▼
    ┌────────────────┐
    │ 列出核心选项   │  5秒：最多3个选项
    └───────┬────────┘
            │
            ▼
    ┌────────────────┐
    │ 快速评估收益   │ 10秒：哪个收益最大？
    └───────┬────────┘
            │
            ▼
    ┌────────────────┐
    │ 评估风险       │  5秒：最大风险能承受吗？
    └───────┬────────┘
            │
            ▼
    ┌────────────────┐
    │ 立即执行       │  5秒：做决定，开始行动
    └───────┬────────┘
            │
            ▼
      [决策完成]
```

**决策树简化版**：
```
                    决策？
                    ┌─┴─┐
                   是  否
                   │   │
              风险可控?  放弃
               ┌─┴─┐
              是  否
              │   │
           收益大?  暂缓
            ┌─┴─┐
           是  否
           │   │
        执行  暂缓
```

**检查清单**：
```
□ 是否真正紧急？ □ 能否承受最坏结果？ □ 信息是否足够？
□ 是否在权限范围内？ □ 是否记录以便复盘？
```

### 3. 团队共识决策流程

**适用场景**：跨部门协作、重要方案选择、规则制定

**RAPID决策模型**：
```
┌────────────────────────────────────────────────────┐
│                                                    │
│  R - Recommend(建议)                               │
│  ─────────────────                                 │
│  角色：决策建议者                                   │
│  行动：提出方案建议，推动决策进程                    │
│                                                    │
├────────────────────────────────────────────────────┤
│                                                    │
│  A - Agree(同意)                                   │
│  ──────────────                                   │
│  角色：决策签字者                                    │
│  行动：正式批准或否决                               │
│                                                    │
├────────────────────────────────────────────────────┤
│                                                    │
│  P - Perform(执行)                                │
│  ─────────────────                                 │
│  角色：执行者                                       │
│  行动：负责方案落地执行                             │
│                                                    │
├────────────────────────────────────────────────────┤
│                                                    │
│  I - Input(输入)                                   │
│  ──────────────                                   │
│  角色：专业顾问                                     │
│  行动：提供专业意见和数据支持                        │
│                                                    │
├────────────────────────────────────────────────────┤
│                                                    │
│  D - Decide(决定)                                  │
│  ──────────────                                   │
│  角色：最终决策者                                   │
│  行动：在充分听取意见后做出最终决定                  │
│                                                    │
└────────────────────────────────────────────────────┘
```

**团队决策会议流程**：
```
会前(1-3天)：
□ 确定决策议题 □ 准备背景资料 □ 通知参会人

会中(60分钟)：
0-10min   明确问题背景
10-25min  各方陈述意见
25-40min  开放式讨论
40-55min  缩小分歧，聚焦选项
55-60min  决策或投票

会后：
□ 发送决策备忘录 □ 执行计划 □ 复盘安排
```

**检查清单**：
```
□ R(建议者)已提出方案？ □ I(输入者)已提供专业意见？
□ A(同意者)已签字确认？ □ D(决定者)已做出决定？
□ P(执行者)已明确？ □ 沟通计划已制定？
```

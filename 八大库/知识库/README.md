## 🔄 更新规则

### 案例归档评分标准
| 维度 | 分值 | 说明 |
|-----|------|------|
| 思维启发度 | 25分 | 是否有独特洞见 |
| 可复用性 | 25分 | 能否复制到其他场景 |
| 信息完整度 | 15分 | 背景/过程/结果是否完整 |
| 结果验证性 | 15分 | 是否有数据支撑 |
| 稀缺性 | 10分 | 是否独特新颖 |
| 警示价值 | 10分 | 是否有警示意义 |
| **总分≥80分** | **必须归档** | |

### 二、数据分析方法

```
常用分析方法选择指南：

【对比分析】
- 什么时候用：比较不同时期/群体/产品
- 关键问题：和谁比？比什么？

【趋势分析】
- 什么时候用：观察数据随时间变化
- 关键问题：上升还是下降？速度如何？

【相关分析】
- 什么时候用：探索变量间关系
- 关键问题：相关性强度？因果还是相关？

【漏斗分析】
- 什么时候用：分析转化路径
- 关键问题：哪一步流失最多？

【归因分析】
- 什么时候用：确定成功/失败原因
- 关键问题：各因素贡献度？

【群组分析】
- 什么时候用：分析不同群体差异
- 关键问题：不同群体有何不同？
```

### 1. 艾森豪威尔矩阵 (Eisenhower Matrix)

**来源**：Dwight D. Eisenhower(艾森豪威尔)，美国第34任总统

**核心原理**：根据重要性和紧迫性将任务分类，优先处理高价值事务

**四象限结构**：
```
              紧迫
         ┌─────────────┬─────────────┐
         │             │             │
         │   第二象限   │   第一象限   │
         │   重要不紧迫 │   重要且紧迫  │
         │   计划去做   │   立即去做   │
         │             │             │
非        ├─────────────┼─────────────┤
紧        │             │             │
        │   第四象限   │   第三象限   │
        │   不重要不紧迫│   不重要但紧迫 │
        │   删除/委托  │   尽量委托   │
        │             │             │
         └─────────────┴─────────────┘
              无价值      有价值
         ──────────────────────────────
                       重要
```

**象限详解**：

**第一象限：重要且紧迫(Put on it)**
- 危机、紧急问题、截止日期
- **策略**：立即执行！优先处理

**第二象限：重要不紧迫(Begin to plan)**
- 预防、规划、关系建立、学习
- **策略**：制定计划，按时执行

**第三象限：不重要但紧迫(Delegate)**
- 干扰电话、某些会议、某些邮件
- **策略**：尽量委托他人

**第四象限：既不紧急也不重要(Eliminate)**
- 浪费时间的事、无聊的社交
- **策略**：删除或休息

**每日时间分配建议**：
```
┌────────────────────────────────────┐
│  第一象限：15%    ████████░░░░░░░  │
│  第二象限：65%    ████████████████│
│  第三象限：15%    ████████░░░░░░░  │
│  第四象限：5%     ███░░░░░░░░░░░░░ │
└────────────────────────────────────┘
```

**使用步骤**：
1. 列出今日所有任务
2. 标注每个任务的重要程度
3. 标注每个任务的紧迫程度
4. 将任务放入对应象限
5. 按象限优先级执行

**应用场景**：
- 每日任务规划
- 周计划制定
- 团队任务分配

**注意事项**：
- 避免被紧急事项填满第一象限
- 第二象限投入决定长期成功
- 定期检视自己是否在做第四象限

### 2. 番茄工作法 (Pomodoro Technique)

**来源**：Francesco Cirillo(弗朗西斯科·西里洛)，1980年代意大利

**核心原理**：将工作时间分割成25分钟专注时段，大幅提升专注力

**核心流程**：
```
    ┌─────────────────────────────────────────┐
    │                                         │
    │   ┌─────────────────┐                   │
    │   │   计划(P lan)   │                   │
    │   │   今日待办清单   │                   │
    │   └────────┬────────┘                   │
    │            │                             │
    │            ▼                             │
    │   ┌─────────────────┐                   │
    │   │  25分钟专注工作  │◄──── 专注！       │
    │   │    🍅 番茄钟    │                   │
    │   └────────┬────────┘                   │
    │            │                             │
    │            ▼                             │
    │   ┌─────────────────┐                   │
    │   │     休息5分钟   │                   │
    │   │    站起来活动    │                   │
    │   └────────┬────────┘                   │
    │            │                             │
    │            ▼                             │
    │   ┌─────────────────┐                   │
    │   │  每4个番茄钟    │───────┐          │
    │   │   休息15-30分钟  │       │         │
    │   └─────────────────┘       │         │
    │                             │         │
    └─────────────────────────────┘         │
                    │                       │
                    └───── 循环重复 ────────┘
```

**详细步骤**：
1. **计划**：每天开始前列出今日任务
2. **追踪**：从第一个任务开始计时
3. **专注**：25分钟不间断工作
4. **记录**：标记完成的番茄数
5. **休息**：5分钟完全放松
6. **长休**：每4个番茄后休息15-30分钟

**规则**：
- 一个番茄时间不可分割
- 完成标记一个X
- 番茄钟响立即停止
- 休息时彻底放松

**适配方法**：
```
标准版：25分钟工作 + 5分钟休息
短任务版：15分钟工作 + 3分钟休息
长任务版：50分钟工作 + 10分钟休息
```

**工具推荐**：
- Focus To-Do(App)
- Forest(App)
- Pomodoro Timer(网页)

**应用场景**：
- 深度工作
- 写作/编程
- 学习备考
- 任何需要专注的任务

**注意事项**：
- 番茄钟期间关闭所有通知
- 避免被打断
- 记录中断次数并分析原因
- 根据任务类型调整番茄时长

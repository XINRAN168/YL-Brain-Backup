### 四、财务管理

```
三大财务报表：
┌─────────────────────────────────────────────────────┐
│                                                     │
│  【资产负债表】                                      │
│  资产 = 负债 + 所有者权益                           │
│  反映企业的"家底"                                   │
│                                                     │
│  【利润表】                                         │
│  收入 - 成本 - 费用 = 利润                          │
│  反映企业的"能力"                                   │
│                                                     │
│  【现金流量表】                                     │
│  经营 + 投资 + 筹资 = 现金变动                      │
│  反映企业的"活力"                                   │
│                                                     │
│  关键财务指标：                                     │
│  - 盈利能力：毛利率、净利率、ROE、ROA              │
│  - 偿债能力：流动比率、速动比率、资产负债率          │
│  - 运营效率：存货周转、应收账款周转、总资产周转      │
│  - 成长能力：营收增长率、利润增长率                  │
│                                                     │
└─────────────────────────────────────────────────────┘
```

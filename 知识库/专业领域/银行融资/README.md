## 核心内容

### 一、企业贷款类型

```
┌─────────────────────────────────────────────────────┐
│                    企业贷款类型                      │
├─────────────────────────────────────────────────────┤
│                                                     │
│  【流动资金贷款】                                   │
│  ─────────────────                                  │
│  用途：日常经营周转                                  │
│  期限：1年以内                                      │
│  特点：随借随还，灵活                                │
│                                                     │
│  【固定资产贷款】                                   │
│  ─────────────────                                  │
│  用途：购建固定资产                                  │
│  期限：1-10年                                       │
│  特点：金额大，需要抵押担保                          │
│                                                     │
│  【项目贷款】                                       │
│  ────────────────                                  │
│  用途：特定建设项目                                  │
│  期限：项目周期                                      │
│  特点：以项目收益还款                                │
│                                                     │
│  【供应链金融】                                     │
│  ────────────────                                  │
│  类型：应收账款融资、存货融资、预付款融资            │
│  特点：依托核心企业信用                              │
│                                                     │
│  【信用贷款】                                       │
│  ────────────────                                  │
│  条件：纳税等级A/B、流水稳定                         │
│  特点：无需抵押                                    │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 三、常见融资方案

```
【方案一】信用贷款组合
- 额度：企业年营收10-30%
- 适合：纳税A/B级、稳定流水企业
- 利率：年化4-8%

【方案二】抵押贷款
- 额度：抵押物评估值60-70%
- 适合：有房产/土地/设备的企业
- 利率：年化3.5-6%

【方案三】供应链融资
- 额度：应收账款80%
- 适合：有稳定核心企业订单
- 利率：年化3-5%

【方案四】组合贷款
- 抵押+信用叠加
- 适合：资金需求大
- 综合利率：最优
```

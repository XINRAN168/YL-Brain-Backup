## 迁移指南

### 快速部署

```bash
# 1. 克隆仓库
git clone https://github.com/XINRAN168/YL-Brain-Backup.git

# 2. 解压最新版本
cd YL-Brain-Backup
tar -xzf 云澜大脑_V7.6_完整包_20260417.tar.gz

# 3. 部署到Agent平台
# 将解压后的文件配置为系统提示词和知识库
```

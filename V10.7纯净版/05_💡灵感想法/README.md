# 💡灵感想法

> 灵感捕捉、逐步完善

---

## 📁 目录结构

```
灵感想法/
├── 快速记录/       ← 当天先记下来，文件名：日期_关键摘要.md
├── 完善中/         ← 正在逐步完善
├── 已完善/         ← 完善完毕，可进入须弥推演
└── 归档/           ← 长期保存
```

---

## 📝 文件命名规范

`日期_关键摘要.md`

**例子**：
- 20260418_日本旅游AI定制行程.md
- 20260419_短视频爆款逻辑.md

---

## 🔄 流程

1. 你说"记一个点子" → 写入快速记录/
2. 后续说"继续完善那个xxx" → 调出继续
3. 完善完毕 → 移到已完善/ → 进入须弥推演
4. 放弃了 → 说明原因后归档

---

## 💬 使用示例

你说：
- "记一个点子：日本旅游+AI做定制行程"
- "继续完善日本旅游那个点子"
- "这个点子完善好了，去推演"

# XR知识库

> 云澜AI团队知识沉淀系统

---

## 目录结构

```
XR知识库/
├── _核心配置/                    # 🔧 核心配置文件
│   ├── 话题注册表.json           # 11个话题定义
│   ├── 流转规则.json             # 流转逻辑配置
│   ├── 流转规则.md               # 流转规则说明
│   ├── 标签规范.md               # 消息标签规范
│   └── 模块接口规范.md           # 模块接口定义
│
├── 🔮须弥推演/                   # 新思维验证
├── 💀失败归档/                   # 失败案例沉淀
├── 📝经验沉淀/                   # 成功经验归档
├── 🇯🇵日本·生活/                 # 日语学习、工作准备
├── 📊情报·机会/                  # 情报收集、机会发现
├── 💼工作·职场/                  # 项目管理、待办事项
├── 💰财务投资/                   # 股市盯盘、投资分析
├── 💡灵感想法/                   # 灵感捕捉、方案完善
├── 🤖特助·秘书/                  # 日常事务、任务提醒
├── 🎴玄机大师/                   # 玄学应用、案例研究
└── 🚀OPC/                        # 团队执行、项目落地
```

---

## 流转闭环

```
📊情报·机会 → 💡灵感想法 → 🔮须弥推演 → 🚀OPC → 📝经验沉淀
     ↓              ↓
  💀失败归档 ←── 验证失败
```

---

## 使用说明

1. 消息带话题标签：`🇯🇵【日本·生活】...`
2. 文件按话题分类存放
3. 流转规则详见 `_核心配置/流转规则.md`

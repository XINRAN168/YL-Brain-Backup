# 📊情报·机会

> 流转入口：情报收集 → 机会发现

---

## 📁 目录结构

```
情报·机会/
├── 情报笔记/       ← 高价值情报沉淀
├── 过期情报/       ← 时效已过
├── 政策利好/       ← 每日政策情报
├── 市场动态/       ← 行业动态追踪
├── 待挖掘/         ← 待分析情报
└── 避坑指南/       ← 来自失败归档
```

---

## 📝 文件命名规范

`日期_关键摘要.md`

**例子**：
- 20260418_签证政策N2要求.md
- 20260418_AI行业政策利好.md

---

## 📅 定时任务

| 时间 | 任务 | 频率 |
|:---:|:---|:---:|
| 09:00 | 每日政策利好监控 | 每日 |
| 10:00 | 市场机会扫描 | 每周一 |

---

## 🔄 流程

1. 每日情报 → 高价值的记入情报笔记/
2. 定期回顾 → 还有用的保留
3. 过期了 → 移到过期情报/ 或删除

---

## 💬 使用示例

你说：
- "这个情报有用，记录下来"
- "查一下签证相关的情报"
- "这个情报过期了"
